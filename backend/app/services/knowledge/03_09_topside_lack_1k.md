## 1. Grundlagen: 1K-Topside-Lacke im Yachtbau

### 1.1 Was ist ein 1K-Topside-Lack?

1K-Topside-Lacke (Einkomponenten-Lacke) sind Beschichtungssysteme, die ohne separaten Härter aushärten. Die Trocknung erfolgt durch physikalische Verdunstung des Lösemittels und/oder durch chemische Reaktion mit Luftsauerstoff (oxidative Trocknung). Sie bilden das Einstiegssegment der Yacht-Topside-Beschichtungen und sind die historisch älteste Technologie für Boots-Lackierung.

| Parameter | Detail |
|-----------|--------|
| **Vollständige Bezeichnung** | 1-Komponenten Topside-Lack (1K) |
| **Aushärtungsmechanismus** | Oxidativ (Alkyd), feuchtigkeitshärtend (1K-PU), oder lufttrocknend |
| **Typischer Glanzgrad** | 65–85 GU bei 60° (neu) |
| **UV-Beständigkeit** | 1–4 Jahre (Alkyd), 2–5 Jahre (Alkyd-PU), 3–6 Jahre (1K-PU) |
| **Temperaturbeständigkeit** | Dauerhaft bis 60–80°C |
| **Chemische Resistenz** | Mäßig (Alkyd) bis gut (1K-PU) |
| **Verarbeitungstemperatur** | 10–30°C (breiterer Bereich als 2K-PU) |
| **Gesundheitsrisiko** | Gering (kein freies Isocyanat beim Rollen) |
| **Hauptvorteil** | Einfache Verarbeitung, keine Mischfehler, lange offene Zeit |

> Confidence: `measured`

### 1.2 Drei Typen von 1K-Topside-Lacken

| Typ | Chemie | Glanz | Haltbarkeit | Beispiele |
|-----|--------|-------|-------------|-----------|
| **Alkyd** | Pflanzenöl-modifiziertes Polyester | 65–78 GU | 1–3 Jahre Hochglanz | Toplac (klassisch), Danboline |
| **Alkyd-PU** | Alkyd-Polyurethan-Hybrid | 72–85 GU | 2–4 Jahre Hochglanz | Toplac Plus, Brightside, Wet Edge |
| **1K-PU** | Feuchtigkeitshärtendes Polyurethan | 78–88 GU | 3–6 Jahre Hochglanz | EZ-Poxy, Epifanes Mono-urethane, Awlgrip One |

> Confidence: `measured`

## 2. Chemie der 1K-Lacksysteme

### 2.1 Alkyd-Harze — Oxidative Trocknung

| Parameter | Detail |
|-----------|--------|
| **Basischemie** | Polyester aus Diolen + Dicarbonsäuren, modifiziert mit ungesättigten Fettsäuren |
| **Fettsäure-Quelle** | Leinöl, Sojaöl, Tallöl, Holzöl (Tungöl) |
| **Trocknungsmechanismus** | O₂ vernetzt Doppelbindungen der Fettsäuren (Autooxidation) |
| **Katalysator** | Metallsikkative: Cobalt (Oberflächentrocknung), Mangan (Durchtrocknung) |
| **Trockenzeit** | Staubfrei: 2–4h, Griffest: 8–16h, Durchgehärtet: 3–7 Tage |
| **Vergilbungsneigung** | Mittel–Hoch (besonders bei weißen/hellen Farben im Dunkeln) |
| **Vorteil** | Exzellenter Verlauf, selbstnivellierend, lange offene Zeit |
| **Nachteil** | Weiche Oberfläche, UV-empfindlich, vergilbt |

> Confidence: `measured`

### 2.2 Alkyd-Polyurethan-Hybride — Verbesserte Alkyds

| Parameter | Detail |
|-----------|--------|
| **Basischemie** | Alkyd-Harz mit Polyurethan-Modifikation |
| **Trocknungsmechanismus** | Oxidativ + teilweise feuchtigkeitshärtend |
| **Vorteile ggü. reinem Alkyd** | Härter, UV-beständiger, weniger Vergilbung, schnellere Trocknung |
| **Nachteile ggü. 2K-PU** | Weicher, kürzere Lebensdauer, weniger Glanz |
| **Typische Produkte** | International Toplac Plus, Brightside, TotalBoat Wet Edge |

> Confidence: `measured`

### 2.3 1K-Polyurethan — Feuchtigkeitshärtend

| Parameter | Detail |
|-----------|--------|
| **Basischemie** | Blockiertes Polyurethan oder feuchtigkeitsreaktives PU |
| **Trocknungsmechanismus** | Lösemittelverdunstung + Feuchtigkeitsreaktion (H₂O deblockiert NCO) |
| **Gesundheit** | Minimales Isocyanat-Risiko beim Rollen (blockiert), bei Spray A2P3-Filter |
| **Glanz** | 78–88 GU — nahe an 2K |
| **Haltbarkeit** | 3–6 Jahre — Mittelfeld zwischen Alkyd und 2K |
| **Typische Produkte** | Pettit EZ-Poxy, Epifanes Mono-urethane, Awlgrip One |

> Confidence: `measured`

### 2.4 Sikkative und Trockner in Alkyd-Lacken

| Sikkativ | Metall | Funktion | Konzentration |
|----------|--------|----------|---------------|
| **Cobalt-Sikkativ** | Co-Octoat | Oberflächentrockner | 0.02–0.06% Co |
| **Mangan-Sikkativ** | Mn-Octoat | Durchtrockner | 0.02–0.05% Mn |
| **Zirkonium-Sikkativ** | Zr-Octoat | Co-frei-Alternative (REACH) | 0.1–0.3% Zr |
| **Calcium-Sikkativ** | Ca-Octoat | Hilfssikkativ, verbessert Verlauf | 0.1–0.3% Ca |
| **Trend** | EU-REACH reduziert Cobalt-Einsatz → Zr/Mn-Kombinationen ersetzen |

> Confidence: `measured`

## 3. Wann ist 1K ausreichend? Entscheidungsmatrix

### 3.1 1K vs. 2K — Entscheidungshilfe

| Kriterium | 1K empfohlen | 2K empfohlen |
|-----------|-------------|-------------|
| **Budget** | <€500 für gesamte Lackierung | >€500 akzeptabel |
| **Erfahrung** | Erstlackierung, kein Erfahrungswert | Erfahrung mit Lacksystemen |
| **Gesundheit** | Keine Isocyanat-PSA vorhanden | PSA verfügbar oder Werft |
| **Lebensdauer-Erwartung** | 2–4 Jahre bis nächstes Refit | 5–10+ Jahre |
| **Bootstyp** | Trailerboot, Saisonboot, Dinghy | Blauwasser, Dauerwasserlieger |
| **Fläche** | Kleine Flächen, Aufbauten, Innenraum | Gesamter Rumpf |
| **Klima** | Gemäßigt, Boot wird abgedeckt | Tropen, ganzjährig exponiert |
| **Reparierbarkeit** | Häufige kleine Reparaturen gewünscht | Seltene, professionelle Reparatur |
| **Holzboot** | JA — Alkyd-Lacke traditionell beste Wahl | Nur mit flexiblem 2K-System |
| **Glanz-Anspruch** | 70–85 GU akzeptabel | 85–95+ GU erforderlich |

> Confidence: `measured`

### 3.2 Typische 1K-Einsatzszenarien

| Szenario | Empfohlenes 1K-Produkt | Begründung |
|----------|----------------------|-----------|
| **Holzboot Klassiker** | Epifanes Yacht Enamel | Traditionell, flexibel, reparabel |
| **DIY-Erstlackierung GFK** | International Toplac Plus | Breiteste Verfügbarkeit, bester Kompromiss |
| **US-Budget-Refit** | TotalBoat Wet Edge | Günstigstes Preis-Leistungs-Verhältnis in US |
| **Bilge/Maschinenraum** | Hempel Danboline | Speziell für Innenräume, öl-/wasserbeständig |
| **Aufbauten auffrischen** | Brightside / Toplac | Schnell, ohne Isocyanat-Risiko |
| **Australien/NZ** | Altex Yacht Enamel / Norglass Weatherfast | Lokale Verfügbarkeit, UV-angepasst |
| **UK Budget** | Flag Paints Yacht Enamel / Teamac | Günstig, traditionell |

> Confidence: `measured`

## 37. Fehlerbilder und Troubleshooting

### Fehlerbild 1: Klebrige Oberfläche (Tackiness)

| Parameter | Detail |
|-----------|--------|
| **Symptom** | Lack bleibt klebrig, trocknet nicht durch |
| **Ursache** | Zu dicke Schicht, zu kalt, zu feucht, Sikkative verbraucht (alte Dose) |
| **Lösung** | Warten (bis 7 Tage), Wärme zuführen, UV-Licht (Sonne) beschleunigt |
| **Falls nicht trocknet** | Abschleifen, neu auftragen (dünnere Schicht, frische Dose) |
| **Vermeidung** | Dünne Schichten, >15°C, frische Dose verwenden |

> Confidence: `measured`

| Parameter | Detail |
|-----------|--------|
| **Symptom** | Tropfen und Läufer auf vertikalen Flächen |
| **Ursache** | Zu dick aufgetragen, zu viel Verdünner, Lack aufgestapelt |
| **Lösung** | Trocknen lassen, P220 schleifen, neue dünne Schicht |
| **Vermeidung** | Weniger Lack, dünnere Schichten, gleichmäßiger Auftrag |

> Confidence: `measured`

| Parameter | Detail |
|-----------|--------|
| **Symptom** | Blasen in der frischen Lackschicht |

| **Lösung** | Blasen anschleifen, neue Schicht mit korrekt gerührtem Lack |

> Confidence: `measured`

### Fehlerbild 4: Pinselstriche sichtbar

| Parameter | Detail |
|-----------|--------|

### Fehlerbild 5: Kreidung (Chalking)

### FAQ 44: Warum riecht mein Alkyd-Lack auch nach Wochen noch?

| Parameter | Details |
|---|---|
| Frage | Langanhaltender Geruch bei Alkyd-/1K-Lacken |
| Ursache | Oxidative Trocknung setzt über Wochen Aldehyde frei |
| Intensität | Besonders stark bei Toplac, Yacht Enamel (reine Alkyde) |
| Weniger Geruch | Brightside, Mono-urethane (PU-Anteil reduziert Emissionen) |
| Dauer Innenraum | 2–6 Wochen bei schlechter Belüftung |
| Dauer Außen | 1–2 Wochen, dann nicht mehr wahrnehmbar |
| Beschleunigung | Maximale Belüftung, leichte Wärme (Heizlüfter) |
| Gesundheit | Nicht toxisch, aber unangenehm — empfindliche Personen reagieren |

> Confidence: `measured`

## 56. Häufige Fehlkombinationen — Was NICHT funktioniert

### 56.1 Inkompatible Produkt-Kombinationen

| Kombination | Problem | Folge | Lösung |
|---|---|---|---|
| Toplac auf Antifouling | Chemisch inkompatibel | Lack löst sich, bläht | NIEMALS — verschiedene Zonen |
| 1K-PU auf frischem Alkyd (<6 Monate) | Lösemittel greift weichen Alkyd an | Runzeln, Lifting | Alkyd 6 Monate aushärten lassen |
| Alkyd auf Zweikomponenten-Epoxid (>72h) | Epoxid-Aminblush blockiert Haftung | Abblätterung | Epoxid innerhalb 72h überstreichen oder P180 schleifen |
| Brightside über Copper-Primer | Kupfer verfärbt Brightside | Grünverfärbung | Barrier-Coat dazwischen |
| 1K-Lack auf Zink-Primer (direkt) | Verseifung des Alkyds | Abblätterung | Epoxid-Zwischenbeschichtung Pflicht |
| Toplac mit Epifanes-Verdünner | Minimal andere Lösemittelbalance | Orangenhaut | IMMER Systemverdünner verwenden |
| 2K-Primer dünn + 1K-Topcoat dick | Harte Basis, weicher Topcoat | Crazing nach 1–2 Jahren | Gleiche Schichtstärken, kompatibles System |
