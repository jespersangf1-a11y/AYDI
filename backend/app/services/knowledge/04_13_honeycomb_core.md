## 1. Grundlagen

### 1.1 Was ist Honeycomb-Kernmaterial?

Honeycomb (Wabenkern) ist ein Sandwichkern-Material bestehend aus dünnen Zellwänden in hexagonaler Anordnung. Die Wabenstruktur erzeugt ein extrem hohes Verhältnis von Steifigkeit zu Gewicht. Zwischen zwei Deckschichten (Skins) bildet der Honeycomb-Kern eine Sandwichplatte, deren Biegesteifigkeit um ein Vielfaches höher liegt als die einer monolithischen Platte gleichen Gewichts.

### 1.2 Funktionsprinzip

| Parameter | Monolithisches GFK (10mm) | Sandwich mit Honeycomb (2×2mm Skin + 10mm Kern) |
|-----------|---------------------------|--------------------------------------------------|
| Gesamtdicke | 10 mm | 14 mm |
| Gewicht (relativ) | 100 % | ~40 % |
| Biegesteifigkeit (relativ) | 100 % | ~700 % |
| Biege-Festigkeit (relativ) | 100 % | ~350 % |

**Prinzip:** Verdoppelung der Kerndicke → ~7× Biegesteifigkeit bei nahezu gleichem Gewicht.

### 1.3 Honeycomb vs. Schaumkern

| Eigenschaft | Honeycomb | Schaumkern (PVC/SAN) |
|-------------|-----------|----------------------|
| Zellstruktur | Offen (Zellen durchgängig) | Geschlossen (geschäumte Blasen) |
| Druckfestigkeit | Sehr hoch (axial) | Isotrop |
| Scherfestigkeit | Richtungsabhängig (L > W) | Weitgehend isotrop |
| Feuchteaufnahme | Offene Zellen = Risiko bei Beschädigung | Geschlossene Zellen = gering |
| Infusionsfähigkeit | Schwierig (Harz füllt Zellen) | Standard |
| Gewicht | Leichtester Kern bei gleicher Steifigkeit | Leicht bis mittel |
| Reparierbarkeit | Komplex (Kern austauschen) | Einfacher (Schleifen, Füllen) |
| Kosten | PP: günstig, Nomex: teuer | PVC: mittel, SAN: mittel |

| Typ | Material | Dichte (kg/m³) | Druckfestigkeit (MPa) | Feuchte | Brandverhalten | Kosten |
|-----|----------|----------------|------------------------|---------|----------------|--------|
| **Nomex** | Aramid-Papier + Phenolharz | 29–144 | 1,4–7,0+ | Hygroskopisch | Hervorragend (LOI ≥28%) | €18–45/m² |

### 1.7 Sandwich-Theorie Kurzfassung

Die Sandwichplatte funktioniert analog einem I-Träger:

- **Deckschichten (Skins)** = Flansche des I-Trägers → nehmen Zug-/Druckkräfte auf
- **Kern** = Steg des I-Trägers → nimmt Scherkräfte auf, hält Skins auf Abstand
- **Klebschicht** = Verbindung Flansch-Steg → überträgt Scherkräfte

**Biegesteifigkeit einer Sandwichplatte:**

```
D = (E_f × t_f × d²) / 2 + (E_f × t_f³) / 6 + (E_c × t_c³) / 12

Wobei:
E_f = E-Modul der Deckschicht
t_f = Dicke der Deckschicht
d = Abstand der Deckschicht-Mittellinien (≈ t_c + t_f)
E_c = E-Modul des Kerns
t_c = Kerndicke

Vereinfacht (dünne Skins, steifer Kern):
D ≈ (E_f × t_f × t_c²) / 2
```

**Kernversagen-Modi:**
1. **Skin-Faltenbildung (Wrinkling):** σ_wr ≈ 0,5 × (E_f × E_c × G_c)^(1/3) — Skin knickt lokal ein
2. **Kern-Schubversagen:** τ_max > τ_c_allow — Kern wird durch Querkraft zerstört
3. **Kern-Druckversagen (Core Crush):** σ_compress > σ_c_allow — punktuelle Überlastung
4. **Delamination:** Interface-Versagen zwischen Skin und Kern

## 2. Wabengeometrie & Physik

### 2.1 Hexagonale Geometrie

Die Standard-Wabenform ist das regelmäßige Sechseck. Wichtige Begriffe:

| Begriff | Definition |
|---------|------------|
| **Zellweite (Cell Size)** | Abstand zwischen zwei parallelen Zellwänden (Nennmaß) |
| **Zellwandstärke** | Dicke der einzelnen Folien/Papier-Wände |
| **L-Richtung (Ribbon)** | Richtung der Bandverbindung — höhere Scherfestigkeit |
| **W-Richtung (Transverse)** | Senkrecht zum Band — niedrigere Scherfestigkeit |
| **T-Richtung** | Kerndicke (Druckrichtung) |
| **Über-Expansion** | Zellen gestreckt → rechteckiger → höhere Flexibilität für Krümmungen |

### 2.2 Standard-Zellweiten

| Zellweite | Zoll | Typische Anwendung |
|-----------|------|-------------------|
| 1,6 mm | 1/16" | Aerospace, Radome |
| 3,2 mm | 1/8" | Marine-Standard (Nomex, Alu) |
| 4,8 mm | 3/16" | Marine/Industrie |
| 6,4 mm | 1/4" | Industrie, große Flächen |
| 8,0 mm | 5/16" | PP-Standard (Nida-Core) |
| 9,5 mm | 3/8" | Großflächen, Leichtbau |
| 12,7 mm | 1/2" | Architektur, Nicht-Struktur |
| 19,0 mm | 3/4" | Dekor, Füllung |

**Marine-Regel:** Kleinere Zellweite = höhere Druckfestigkeit, bessere Skin-Abstützung, weniger Telegraphing. Standard im Bootsbau: **3,2 mm** (Nomex/Alu), **8,0 mm** (PP).

### 2.3 Richtungsabhängige Eigenschaften

Honeycomb ist **nicht isotrop**. Die mechanischen Eigenschaften unterscheiden sich erheblich:

| Eigenschaft | L-Richtung | W-Richtung | Verhältnis |
|-------------|------------|------------|------------|
| Scherfestigkeit | Höher | Niedriger | L/W ≈ 1,5–1,8 |
| Schermodul | Höher | Niedriger | L/W ≈ 1,5–2,0 |
| Druckfestigkeit | Gleich | Gleich | 1,0 |

**Praxisrelevanz:** Die L-Richtung muss entlang der Hauptbelastung orientiert werden. Bei Deckspaneelen: L-Richtung quer zum Schiff (= Hauptbiegebelastung durch Wellengang).

**Orientierungshilfe nach Yacht-Zone:**

| Zone | Hauptbelastung | L-Richtung orientieren |
|------|---------------|----------------------|
| Deck (Mittschiff) | Querbiegung (Wellengang) | Quer zum Schiff (Backbord-Steuerbord) |
| Deck (Vorschiff) | Längs + Quer (Wellenschlag) | ±45° oder Quasi-Isotrop (2 Lagen) |
| Aufbau (Seiten) | Torsion + Biegung | Vertikal (Höhe) |
| Aufbau (Dach) | Querbiegung | Quer zum Schiff |
| Cockpit-Sohle | Punkt-/Flächenlast | Quer oder ±45° |
| Schotten | Querbelastung | Vertikal |
| Rumpf (Seiten) | Hydrostatischer Druck | Vertikal |

**Identifikation der L-Richtung:**
- Bandrichtung (Ribbon) ist auf Platte markiert (Pfeil oder Linie)
- Wenn nicht markiert: Zellen in L-Richtung haben doppelte Zellwand (zwei verklebte Folien)
- In W-Richtung: einfache Zellwand
- Doppelte Wand durch Verformungswiderstand fühlbar

### 2.4 Telegraphing (Durchzeichnung)

Wenn die Deckschicht zu dünn ist, zeichnen sich die Wabenzellen als Muster auf der Oberfläche ab. Vermeidung:

| Zellweite | Minimale Skin-Dicke (GFK) | Minimale Skin-Dicke (CFK) |
|-----------|---------------------------|---------------------------|
| 3,2 mm | ≥0,8 mm | ≥0,5 mm |
| 4,8 mm | ≥1,0 mm | ≥0,6 mm |
| 8,0 mm | ≥1,5 mm | ≥1,0 mm |

**Zusätzliche Maßnahmen gegen Telegraphing:**
- Surfacing-Film (Epoxid-Harzfilm 100–150 g/m²) zwischen Kern und äußerer Skin
- Gefülltes Epoxid (Microballons) als Fillet zwischen Zellwand und Skin
- Zusätzliche Glasfaser-Lage (160 g/m²) direkt über Kern (beide Seiten)
- Feinzelliger Nomex (1,6mm) für sichtbare Oberflächen
- Post-Cure unter kontrolliertem Druck (Telegraphing kann bei Überdruck entstehen)

**Telegraphing-Risikobewertung (AYDI):**

| Faktor | Niedrig | Mittel | Hoch |
|--------|---------|--------|------|
| Zellweite | ≤3,2 mm | 4,8 mm | ≥6,4 mm |
| Skin-Dicke (CFK) | ≥1,0 mm | 0,6–1,0 mm | <0,6 mm |
| Skin-Dicke (GFK) | ≥1,5 mm | 1,0–1,5 mm | <1,0 mm |
| Surfacing Film | Vorhanden | — | Fehlend |
| Lackierung | Dick (>500µm) | Mittel | Dünn/transparent |

## 3. Honeycomb-Typen im Überblick

### 3.1 Klassifikation nach Material

```
Honeycomb-Kernmaterialien
├── Metallisch
│   ├── Aluminium (3003, 5052, 5056)
│   ├── Edelstahl (selten, Sonderanwendungen)
│   └── Titan (Aerospace, nicht Marine)
├── Aramid (Organisch)
│   ├── Nomex (DuPont T412/T722 Papier + Phenolharz)
│   └── Kevlar (Para-Aramid, höhere Festigkeit)
├── Thermoplastisch
│   ├── Polypropylen (PP) — Standard Marine
│   ├── Polyethylenterephthalat (PET)
│   ├── Polycarbonat (PC)
│   └── PEEK (Hochtemperatur, Spezial)
└── Sonstige
    ├── Glasfaser-Honeycomb (Hexcel HRP)
    ├── Karton-Honeycomb (nicht marine)
    └── 3D-gedruckte Honeycomb-Strukturen
```

## 7. Nomex — Feuchteaufnahme & Wasserverhalten

### 7.1 Das Problem

Nomex ist **hygroskopisch** — es nimmt Feuchtigkeit aus der Umgebungsluft auf. Die offene Zellenstruktur ist das größte Risiko:

| Phase | Mechanismus | Zeitrahmen |
|-------|------------|------------|
| 1. Eindringen | Wasser durch Risse, Impact, Kanten, undichte Befestigungen | Sofort |
| 2. Ansammlung | Wasser sammelt sich in offenen Wabenzellen | Wochen–Monate |
| 3. Degradation | Feuchtigkeit schwächt Phänolharz-Beschichtung | Monate–Jahre |
| 4. Frost-Tau | Eis expandiert, Zellwände brechen | Winter-Zyklen |
| 5. Druckaufbau | Wärme → Dampfdruck → Skin-Ablösung | Sonneneinstrahlung |

### 7.2 Absorptionsraten

| Umgebung | Absorptionsrate | Festigkeitsverlust |
|----------|----------------|-------------------|
| Trocken/Niedrige Feuchte | Minimal | <5% |
| Gemäßigt (Nordeuropa) | Langsam über Monate/Jahre | 5–10% |
| Tropisch/Hochfeucht | Signifikant | 10–20% |
| Salzwasser-Spritzzone | Beschleunigt | 15–25% |
| Getaucht (nach Impact) | 3,5% nach 2.400h | >25% |

### 7.3 Schutzmaßnahmen

| Maßnahme | Wirksamkeit | Kosten |
|----------|-------------|--------|
| Barriere-Folien (Polyester, Epoxid) | 70–90% Reduktion | Niedrig–Mittel |
| Kantenversiegelung | Hoch | Niedrig |
| Vakuumversiegelter Kern | Gut (initial) | Mittel |
| Geschlossenzelliger Schaum als Alternative | Hervorragend | Höher |
| Vollständige Kernkapselung | Sehr gut | Mittel |

### 7.4 Erkennung von Wassereinbruch

| Methode | Empfindlichkeit | Ausrüstung |
|---------|-----------------|------------|
| Infrarot-Thermografie | Erkennt 0,2 ml Wasser aus 20m Entfernung | IR-Kamera |
| Ultraschall | Hoch (Wasser verändert Schalleigenschaften) | US-Prüfgerät |
| Feuchtemesser | Mittel (Oberfläche) | Handgerät |
| Gewichtszunahme | 5–10% bei Wassereinbruch | Waage |
| Klopftest | Grob (dumpf = Problem) | Hammer |
| Visuell | Weiche Stellen, Verfärbung, Delamination | Auge |

## 8. Nomex — Verarbeitung im Bootsbau

### 8.1 Harzsysteme

| Harz | Kompatibilität | Anmerkung | Marine-Einsatz |
|------|---------------|-----------|----------------|
| **Epoxid** | Hervorragend | Bevorzugt für Nomex-Verklebung | Primäre Wahl |
| **Vinylester** | Gut | Marine-Korrosionsschutz | Üblich |
| **Polyester** | Gut | Vakuuminfusion schwierig | Seltener |
| **Phänolharz** | N/A | Bereits im Kern integriert | — |

### 8.2 KRITISCHE Einschränkung: Vakuuminfusion

**Nomex ist NICHT für Vakuuminfusion geeignet!**

- Offene Zellenstruktur → Harz füllt Wabenzellen statt Skins zu binden
- Ergebnis: Massiver Gewichtszuwachs, keine Kernfunktion, Kostenexplosion
- Geschlossenzelliger Schaum (Divinycell, Corecell) ist besser für Infusion

### 8.3 Verarbeitungsmethoden (Marine)

#### Methode 1: Prepreg + Vakuumbag (Industriestandard für Regatta)

1. Prepreg-CFK-Lagen auf Form
2. Nomex-Kern positionieren
3. Klebfilm (Redux, XA120) zwischen Kern und Skins
4. Vakuumsack aufbringen
5. Ofenhärtung (120–180°C je nach Epoxid)
6. **Anwendung:** IMOCA 60, AC75, Racing-Yachten

#### Methode 2: Nasslamination + Vakuumbag

1. Manuelle Nasslamination der Skins über Nomex
2. Vakuum zum Konsolidieren und Entlüften
3. Raumtemperatur- oder Niedertemperatur-Aushärtung
4. **Anwendung:** Kleinere Projekte, Reparaturen, Custom-Boote

#### Methode 3: Dreistufiges Verfahren (Geringstes Risiko)

1. Erste Skin laminieren und separat aushärten
2. Abreißgewebe entfernen, Oberfläche vorbereiten
3. Kern mit Vakuumbag aufkleben
4. Zweite Skin separat laminieren und aushärten
5. **Vorteil:** Geringste Fehlwahrscheinlichkeit
6. **Nachteil:** Zeitaufwändig, mehrere Aushärtezyklen

#### Methode 4: Zweistufiges Verfahren (Häufigste Methode)

1. Erste Skin + Klebfilm + Kern gemeinsam auflegen
2. Co-Cure unter Vakuumbag
3. Zweite Skin in zweiter Stufe aufbringen
4. **Vorteil:** Schneller, effizient, moderates Risiko
5. **Nachteil:** Sorgfältige Vakuumdruckkontrolle erforderlich

#### Methode 5: Autoklavverfahren (Industriell, Aerospace-Standard)

1. Prepreg-Skins + Klebfilm + Kern im Werkzeug aufbauen
2. Vakuumsack versiegeln
3. Autoklav-Zyklus programmieren:
   - Aufheizrate: 1–3°C/min
   - Haltetemperatur: 120°C (250°F) oder 177°C (350°F) je nach Harzsystem
   - Haltedauer: 60–120 min (systemabhängig)
   - Druck: 0,28–0,35 MPa (40–50 psi) — **KRITISCH: Nicht über 0,35 MPa bei Nomex!**
   - Abkühlrate: ≤3°C/min (langsam, Eigenspannungen minimieren)
4. **Anwendung:** IMOCA 60, AC75, Superyacht-Strukturteile
5. **Vorteil:** Höchste Laminatqualität, reproduzierbar, minimaler Porengehalt
6. **Nachteil:** Hohe Investitionskosten (Autoklav €200k–2M+), Bauteilgröße limitiert

**Kritische Autoklavparameter für Nomex-Honeycomb:**

| Parameter | Wert | Warum kritisch |
|-----------|------|---------------|
| Max. Druck | 0,35 MPa (50 psi) | Höherer Druck → Core Crush |
| Vakuumniveau | -0,7 bis -0,85 bar | Zu hohes Vakuum → Telegraphing |
| Aufheizrate | ≤3°C/min | Zu schnell → Harz fließt vor Gelierung |
| Haltetemperatur Toleranz | ±5°C | Ungleichmäßig → lokale Überhärtung |
| Rückdruck (Back-Pressure) | Immer anwenden | Verhindert Ausgasung/Porosität |
| Trocknungsschritt | 4h bei 70°C vor Verarbeitung | Feuchte im Nomex → Blasenbildung |

### 10.6 Schadensfortschritt — Kaskadierendes Versagen

**Typischer Schadensverlauf bei unbehandeltem Honeycomb-Schaden:**

```
1. Impact (z.B. Ankertreffer) → lokale Delle (Tage)
   ↓
2. Mikrorisse in Skin → Feuchteeintritt (Wochen–Monate)
   ↓
3. Wasser in Nomex-Zellen → Kern absorbiert Feuchte (Monate)
   ↓
4. Frost-Tau-Zyklen → Zellen expandieren, Skin hebt sich (Winter)
   ↓
5. Dampfdruck bei Sonneneinstrahlung → Delamination breitet sich aus (Sommer)
   ↓
6. Großflächige Delamination → Steifigkeitsverlust (1–3 Jahre)
   ↓
7. Strukturelles Versagen unter Last (2–5 Jahre nach Impact)
```

**AYDI-Regel:** Jeder Impact auf Honeycomb-Sandwich → sofortige Inspektion + Reparatur innerhalb von 30 Tagen. Unbehandelte Schäden eskalieren exponentiell.

## 11. Nomex — Reparaturverfahren

### 11.1 Kleine Druckstelle (Delle)

**Schritt-für-Schritt:**

1. Äußere Skin über Schadensbereich entfernen (Flexscheibe, Fräser)
2. Beschädigten Kern freilegen und Ausdehnung prüfen
3. **KRITISCH:** Kern muss vollständig trocken sein
4. Beschädigte Zellen mit Epoxid + Glaskügelchen-Mischung füllen
5. ODER: Kern-Abschnitt durch Corecell/Airex-Schaum ersetzen
6. Neue GFK/CFK-Skin laminieren
7. Schleifen, Grundieren, Lackieren

### 11.2 Delamination / Wasserschaden

1. Skin(s) im betroffenen Bereich entfernen
2. Beschädigten Kern komplett herausnehmen
3. Bereich vollständig austrocknen (Tage/Wochen je nach Ausmaß)
4. Hohlraum mit Verguss-Epoxid oder strukturellem Epoxid füllen
5. Neue Skin(s) laminieren
6. Kanten versiegeln mit Potting-Compound
7. **WICHTIG:** Injektions-Reparaturen sind bei Honeycomb NICHT empfohlen — vollständige Entfernung erforderlich

### 11.3 Kernaustausch (vollständig)

1. Beide Skins über Schadensbereich entfernen
2. Alten Kern komplett herausnehmen
3. Neuen Honeycomb (Nida-Core, Nomex oder Corecell) einsetzen
4. Klebfilm oder Epoxid-Kleber für Verbindung
5. Vakuumbag oder Druckbag für Aushärtung
6. GFK/CFK-Skins aufbringen
7. Alle Kanten mit Potting-Compound versiegeln

### 11.4 Durchgehende Befestigungen (Through-Deck)

**Steve D'Antonio-Regel:**

> Kern muss bei Durchbrüchen entweder **während der Laminierung ausgespart** ODER **nachträglich entfernt und mit verdicktem Epoxid verfüllt** werden.

| Schritt | Verfahren |
|---------|-----------|
| 1 | Bohrung durch Sandwich |
| 2 | Kern um Bohrung herum entfernen (Fräser, überdimensioniert) |
| 3 | Hohlraum mit verdicktem Epoxid füllen |
| 4 | Aushärten lassen |

## 17. Aluminium — Galvanische Korrosion mit CFK (KRITISCH)

### 17.1 Das Problem

```
Elektrochemisches Potential in Seewasser:
  CFK (Kohlefaser):     +0,2 bis +0,5 V
  Aluminium-Legierung:  -1,0 bis -1,3 V
  ─────────────────────────────────────
  Potentialdifferenz:   ~1,2 V (EXTREM)
```

**Mechanismus:**
- CFK hat hohes, positives elektrochemisches Potential (Kathode)
- Aluminium hat sehr niedriges, negatives Potential (Anode)
- Leitfähige Verbindung + Elektrolyt (Seewasser) → Aluminium korrodiert bevorzugt
- Aluminium wird galvanische Opferanode; CFK bleibt geschützt
- Sauerstoff-Reduktionsreaktion kontrolliert Korrosionsrate

### 17.2 Versagen in der Praxis

| Szenario | Zeitrahmen | Festigkeitsverlust |
|----------|-----------|-------------------|
| Offene Kante + Seewasser | 2–3 Jahre | Katastrophale Delamination |
| Kleine Skin-Beschädigung + Seewasser | 3–5 Jahre | 40–60% Festigkeitsverlust |
| Vollständig versiegelt, kein Defekt | 10+ Jahre | Minimal (wenn intakt) |

### 17.3 Prävention

| Maßnahme | Wirksamkeit | Kosten | Empfehlung |
|----------|-------------|--------|-----------|
| **GFK-Isolationsschicht** | Hoch | +€5–15/m² | ✅ **MUSS bei CFK-Skins** |
| **Nomex statt Alu verwenden** | Eliminiert Problem | +15–30% Kernkosten | ✅ Bevorzugt |
| **Epoxid-Grundierung + PU-Decklack** | Temporär (2–5 Jahre) | Niedrig | ⚠️ Nur zusätzlich |
| **Opferanode** | Reaktiv, wartungsintensiv | Mittel | ⚠️ Nur Notlösung |
| **Alu-Skins + Alu-Kern** | Kein galvanisches Paar | Variable | ✅ Wenn Design erlaubt |

### 17.4 Galvanische Korrosion — Inspektionsprotokoll

| Prüfschritt | Methode | Akzeptanzkriterium | Häufigkeit |
|-------------|---------|-------------------|-----------|
| 1. Visuelle Inspektion | Suche: weißes Pulver, Blasen, Verfärbung | Keine sichtbaren Zeichen | Jährlich |
| 2. Klopftest Deck/Schott | Münze oder Klopfhammer | Einheitlicher Klang, kein Hohlton | Jährlich |
| 3. Feuchtemessung | Tramex Skipper Plus oder Sovereign | <5% Feuchte relativ | Jährlich |
| 4. Kanteninspektion | Lupe 10× an allen offenen Kanten | Keine weißen Ablagerungen/Kristalle | Halbjährlich |
| 5. Beschädigungsstellen | UV-Lampe + Lupe | Keine Korrosionsprodukte in Rissen | Nach jedem Impact |
| 6. Opferanoden prüfen | Visuell + Dickenmessung | >50% Restmaterial | Halbjährlich |
| 7. IR-Thermografie | Wärmebildkamera (FLIR) | Keine Hotspots (Korrosionsherde) | Alle 3 Jahre |

**Eskalationsprozedur bei Korrosionsverdacht:**

1. **Verdacht bestätigt** → Sofort alle offenen Kanten versiegeln
2. **Probenahme** → 25mm × 25mm Probe aus verdächtigem Bereich
3. **Labor** → REM/EDX-Analyse auf Korrosionsprodukte (Al₂O₃, Al(OH)₃)
4. **Bewertung** → Festigkeitstest der Probe vs. Referenzwert
5. **Entscheidung** → Reparatur lokal / Paneel-Austausch / Monitoring

### 17.5 Galvanische Reihe (Marine-relevant)

| Material | Potential (V vs. Ag/AgCl in Seewasser) | Rolle |
|----------|---------------------------------------|-------|
| Graphit/CFK | +0,20 bis +0,50 | Edelste Kathode |
| Titan | +0,10 bis +0,20 | Kathode |

| Blei | -0,50 bis -0,55 | — |

## 20. Aluminium — Schadensbilder & Versagen

### 20.1 Galvanische Korrosion (CFK + Alu)

- Dominanter Versagensmechanismus bei CFK-Sandwich

## 24. PP — Verarbeitung & Harzkompatibilität

### 24.1 Harzkompatibilität

| Harz | Kompatibilität | Anmerkung |
|------|---------------|-----------|
| **Polyester** | ✅ Hervorragend | Standard im Serienschiffbau |
| **Vinylester** | ✅ Hervorragend | Verbesserte Korrosionsbeständigkeit |
| **Epoxid** | ✅ Gut | Erfordert Oberflächenbehandlung |

**Schlüssel:** Das Polyester-Vlies auf der PP-Oberfläche fördert die Harzbenetzung und Haftung. Ohne Vlies haftet Harz schlecht auf reinem PP.

### 24.2 Vakuuminfusion

- Infusionsqualitäten (Plascore PCGA-XR, Nida-Core H8PP-I): für geschlossene Formprozesse optimiert
- Durchgehendes PET-Vlies versiegelt Zellstruktur
- Harz fließt entlang der Klebefuge mit minimaler Kernpenetration
- 50:50 Faser/Harz-Verhältnis erreichbar (vs. 40:60 bei offener Form)
- 60-Fuß-Rumpf in 2 Stunden infundierbar

### 24.3 Oberflächenbehandlung für PP

| Methode | Wirkung | Haltbarkeit |
|---------|---------|-------------|
| **Corona-Behandlung** | Erhöht Oberflächenenergie durch Oxidation | 2–8 Wochen (⚠️ degradiert!) |
| **Flammenbehandlung** | Oxidiert PP-Oberfläche | Ähnlich wie Corona |
| **Werksseitige Behandlung** | Integriert in Produktion (Nida-Core, Nidaplast) | Standard — bevorzugt |

**WICHTIG:** Corona-Effekt lässt nach! Laminierung muss innerhalb von 2 Wochen nach Behandlung erfolgen.

### 24.4 Thermoformbarkeit

PP-Honeycomb ist thermoplastisch → formbar zu gekrümmten Paneelen (Vorteil gegenüber Nomex/Alu). EconCore/ThermHex: kontinuierliche Formung integriert in Produktion.

### 24.5 Vakuuminfusion — Schritt-für-Schritt-Anleitung (Nida-Core H8PP)

**Materialien:**
- Nida-Core H8PP-I (Infusionsqualität, PET-Vlies beidseitig)
- Vinylester oder Polyester-Harz (niedrigviskos, 100–250 mPa·s)
- Biaxiales Glasfasergelege (z.B. 600 g/m² ±45°)
- Gelcoat, Abreißgewebe, Fließhilfe (Mesh), Vakuumfolie, Dichtband
- Spiralschläuche, T-Stücke, Harzbehälter

**Schritt-für-Schritt:**

| Schritt | Aktion | Kritische Parameter |
|---------|--------|-------------------|
| 1 | Form vorbereiten (Release, Gelcoat) | Gelcoat 0,5–0,8 mm, gleichmäßig |
| 2 | Äußere Skin auflegen (Glasfaser) | Orientierung beachten, 0°/90° oder ±45° |
| 3 | Nida-Core H8PP-I positionieren | Kanten dicht stoßen, kein Übermaß |
| 4 | Innere Skin auflegen | Symmetrischer Aufbau zur äußeren Skin |
| 5 | Abreißgewebe aufbringen | Vollflächig, überlappend |
| 6 | Fließhilfe (Mesh) auflegen | Abstand 20mm von Kante (kein Race-Tracking) |
| 7 | Spiralschläuche positionieren | Zufluss am niedrigsten Punkt, Absaugung oben |
| 8 | Vakuumfolie versiegeln | Dichtband prüfen, keine Falten über Kern |
| 9 | Vakuumtest (Dichtigkeit) | -0,9 bar, Druckabfall <50 mbar/5 min |
| 10 | Harz mischen (nach Herstellerangabe) | Topfzeit beachten! Typ. 40–90 min |
| 11 | Infusion starten | Harzfront gleichmäßig, max. 30 min für 1m |
| 12 | Harzfront beobachten | Vollständige Benetzung sicherstellen |
| 13 | Zulauf schließen, Vakuum halten | Überschuss-Harz in Falle |
| 14 | Aushärtung | RT 24h + Post-Cure 50°C/8h (empfohlen) |
| 15 | Entformen | Keile, nicht hebeln (Beschädigung) |

**Typische Fehler bei PP-Infusion:**

| Fehler | Ursache | Erkennung | Vermeidung |
|--------|---------|-----------|-----------|
| Trockenstellen | Fließhilfe fehlt lokal | Weiße Flecken durch Folie | Fließhilfe überlappend, Spiralschläuche korrekt |
| Race-Tracking | Harz fließt am Rand voraus | Ungleichmäßige Front | Fließhilfe 20mm vom Rand zurücksetzen |
| Kernpenetration | Vlies beschädigt/fehlend | Gewichtsüberprüfung (+30%) | Nur Infusionsqualität verwenden |
| Blasen unter Skin | Luft eingeschlossen | Klopftest, Ultraschall | Vakuum ≥-0,85 bar, Entlüftungskanäle |
| Harzgelierung vor Füllung | Topfzeit überschritten | Infusion stoppt | Harz kühlen, dünnere Schichten |
| Delamination an Kern | Unbehandeltes PP | Klopftest nach Aushärtung | NUR werksseitig behandeltes Material |

### 24.6 Handlaminierung — Schritt-für-Schritt-Anleitung (DIY)

| Schritt | Aktion | Tipp |
|---------|--------|------|
| 1 | Äußere Skin auf Form laminieren | Standardtechnik, Harz gut auspressen |
| 2 | Klebeharz (thixotrop) auf nasse Skin | 1mm Schicht, Zahnspachtel |
| 3 | H8PP-Kern einlegen, leicht andrücken | Harz muss in Vlies eindringen |
| 4 | Vakuumsack aufbringen (optional) | -0,5 bar, NICHT mehr (Core Crush!) |
| 5 | Aushärten lassen (12–24h bei RT) | Nicht zu früh entlasten |
| 6 | Kern-Oberfläche schleifen (P80) | Mechanische Verankerung |
| 7 | Innere Skin laminieren | Symmetrisch zur äußeren Skin |
| 8 | Kanten versiegeln | Epoxid + Glasfaser-Umwicklung |

## 26. PP — Schadensbilder & Versagensmechanismen

### 26.1 Klebversagen (Bond Delamination) — Hauptversagen

| Aspekt | Detail |
|--------|--------|
| **Ursache** | Niedrige Oberflächenenergie von PP, unzureichende Harzbenetzung |
| **Symptome** | Delamination an Skin-Kern-Grenzfläche, Skin lässt sich mit wenig Kraft ablösen |
| **Häufigkeit** | Besonders in Bereichen mit hohem Verkehr (Decks) oder hoher Feuchte (Nasszellen) |
| **Prävention** | Werksseitig behandelter Kern, saubere Oberfläche, ausreichende Harzsättigung |
| **Confidence** | measured (95%+) bei dokumentierter Laminierung; visual_medium (60–75%) bei reiner Inspektion |

### 26.2 Kriechen unter Wärme

| Aspekt | Detail |
|--------|--------|
| **Ursache** | PP erweicht ab 80°C, Dauerlast → permanente Verformung |
| **Typische Stellen** | Maschinenraumpaneele, sonnenbeschienene Interieur-Schotten (Tropen) |
| **Symptome** | Durchhängen über Jahre, Verformung großer ungestützter Paneele |
| **Prävention** | Temperaturgrenze <60°C, Zwischenstützen, dickere Skins |

### 26.3 Impact-Schäden & Kerndruckversagen

| Aspekt | Detail |
|--------|--------|
| **Ursache** | Niedrige Druckfestigkeit (0,5 MPa), Punktlasten |
| **Symptome** | Sichtbare Dellen, weiße Stressmarken, Wassereinbruch bei Skin-Riss |
| **Erfahrung** | 20+ Jahre Reparaturerfahrung: "extensive Dellen und Delaminationen" in Powerboats |
| **Prävention** | Nicht für Rumpfboden/Laufende Flächen, dickere Skins in Impact-Zonen |

### 26.4 Schlechte Schraubenhaltung

| Aspekt | Detail |
|--------|--------|
| **Ursache** | PP-Kern hat keinen mechanischen Griff für Gewinde |
| **Symptome** | Lose Beschläge, "Durchdrehen" der Schrauben, Versagen unter dynamischer Last |
| **Prävention** | KEINE direkte Verschraubung in PP-Kern, Backing Plates, lokale Verstärkung |

### 26.5 Kantendegradation & Wassereinbruch

| Aspekt | Detail |
|--------|--------|
| **Ursache** | Geschnittene Kanten legen offene Zellen frei |
| **Symptome** | Quellung, weiche Kanten, Wasserflecken auf benachbarten Flächen |
| **Prävention** | Alle Kanten versiegeln (Fräsung + GFK-Umwicklung, Spachtel, Schaumstopfen) |
| **PP-Paradox** | Kern selbst: 0% Feuchte — aber offene Kanten lassen Wasser in Luftraum der Zellen |

## 31A. Lagerung, Transport & Handhabung von Honeycomb-Materialien

### 31A.1 Lagerungsbedingungen

| Material | Temperatur | Luftfeuchte | Lagerort | Haltbarkeit |
|----------|-----------|-------------|----------|-------------|
| **Nomex (roh)** | 10–30°C | <60% RH | Trocken, geschlossen | 2+ Jahre |
| **Nomex (Prepreg)** | -18°C (Freezer) | N/A (versiegelt) | Tiefkühler | 6–12 Monate |
| **Aluminium** | 10–35°C | <70% RH | Trocken, beschichtungsfrei | 5+ Jahre |
| **PP (H8PP)** | 10–40°C | Unkritisch | Trocken, UV-geschützt | 5+ Jahre |
| **Klebfilme (Redux etc.)** | -18°C (Freezer) | N/A (versiegelt) | Tiefkühler | 6–12 Monate |
| **Prepregs (GFK/CFK)** | -18°C (Freezer) | N/A (versiegelt) | Tiefkühler | 6–12 Monate |

### 31A.2 Kritische Lagerungsfehler

| Fehler | Konsequenz | Betroffenes Material |
|--------|-----------|---------------------|
| Nomex feucht lagern | Wasseraufnahme → Blasenbildung bei Aushärtung | Nomex |
| Prepreg bei RT lagern | Voraushärtung → reduzierte Klebkraft, kurze Haltbarkeit | Prepreg, Klebfilm |
| Alu-Honeycomb unverpackt | Staub, Feuchte in Zellen → Korrosion | Aluminium |
| PP in Sonne lagern | UV-Degradation der Oberfläche | PP |
| Honeycomb stapeln ohne Zwischenlage | Zelldruckschäden (Core Crush) | Alle |
| Auf Kante stellen | Verformung der Wabenzellen | Nomex, PP |

### 31A.3 Transport

| Regel | Begründung |
|-------|-----------|
| Flach transportieren (horizontal) | Vermeidet Zellverformung durch Eigengewicht |
| Schutzecken an Kanten | Kanten sind empfindlichste Zone |
| Nicht biegen oder knicken | Irreversible Zellverformung |
| Folie umwickeln (Nomex) | Feuchtigkeitsschutz |
| Mindestens 3 Auflagepunkte | Gleichmäßige Lastverteilung |
| Nicht auf Palette mit spitzen Nägeln | Zellen werden lokal zerstört |

### 31A.4 Handhabung in der Werft

**Vor der Verarbeitung:**
- Nomex: **Trocknung** — 4h bei 70°C oder 24h bei 50°C im Umluftofen
- Prepreg: **Aufwärmen** — versiegelt auf RT bringen (2–4h), erst dann öffnen (Kondenswasser!)
- Alle: **Visuelle Prüfung** — Kanten auf Beschädigung, Zellen auf Deformation
- Alle: **Sauberkeit** — Staubfreie Umgebung, keine Kontamination durch Trennmittel/Öl

**Während der Verarbeitung:**
- Nomex: Handschuhe tragen (Hautfett verunreinigt Klebfläche)
- Alu: Handschuhe tragen (Grate → Schnittverletzung)
- PP: Innerhalb 2 Wochen nach Corona-Behandlung laminieren
- Alle: Restmaterial SOFORT versiegeln/verpacken

### 31A.5 Ausschussgründe (SOFORT entsorgen)

| Zustand | Warum Ausschuss |
|---------|----------------|
| Sichtbare Dellen (>5mm Ø) im Rohkern | Zellen irreversibel deformiert → lokale Schwäche |
| Nomex feucht (>3% Gewichtszunahme) | Nicht mehr trocknungsfähig ohne Strukturverlust (bei langer Exposition) |
| Alu mit weißem Pulver an Oberfläche | Korrosion bereits eingesetzt |
| PP mit verfärbter/spröder Oberfläche | UV-Degradation → Klebhaftung kompromittiert |
| Prepreg klebrig/nicht mehr drapierbar | Voraushärtung → mechanische Werte nicht erreichbar |
| Klebfilm trocken/brüchig | Out-of-Life → keine Klebkraft |

## 32. Diagnose-Leitfaden Honeycomb-Schäden

### 32.1 Visuelle Inspektion

| Befund | Mögliche Ursache | Schwere | Nächster Schritt |
|--------|-----------------|---------|-----------------|
| Delle (sichtbar) | Punktlast, Impact | Mittel | Klopftest um Delle |
| Weiße Stressmarken | Überlast, Biegung | Mittel | Feuchtemessung |
| Blasenbildung | Dampfdruck (Wasser im Kern) | Hoch | IR-Thermografie |
| Verfärbung an Kanten | Wassereinbruch | Hoch | Kern freilegen |
| Weiche Stellen beim Betreten | Kern-Delamination | Hoch | Professionelle Prüfung |
| Risse in Gelcoat/Skin | Impact, Ermüdung | Mittel–Hoch | Feuchtemessung + Klopftest |

### 32.2 Klopftest (Tap Testing)

| Sound | Bedeutung | Aktion |
|-------|-----------|--------|
| Scharf, klar | Intakt | OK |
| Dumpf, hohl | Delamination oder Kernversagen | Markieren, weiter prüfen |
| Weich, "tot" | Schwerer Schaden | Professionelle Prüfung |

**Einschränkung bei Honeycomb:** Klopftest kann nur die nächste Skin-Kern-Grenzfläche beurteilen, NICHT die Rückseite. Zweiseitiger Zugang für vollständige Prüfung erforderlich.

### 32.3 Feuchtemessung

| Methode | Gerät | Anwendung |
|---------|-------|-----------|
| Dielektrisch | Tramex, Protimeter | Oberflächen-Screening |
| Pin-Messung | Diverse | Punktuelle Bestätigung |

### 32.4 Infrarot-Thermografie

- **Passiv:** Umgebungstemperatur-Differenzen nutzen
- **Aktiv (Flash):** Kurzzeit-Wärmepuls
- **Transient:** Kontinuierliche Wärmequelle
- **Empfindlichkeit:** Erkennt 0,2 ml Wasser aus 20m Entfernung
- **Vorteil:** Vollflächige Inspektion gleichzeitig möglich

### 32.5 Ultraschall

- Ergänzend zu IR-Thermografie
- Wasser verändert Schalleigenschaften
- Quantitative Tiefenmessung möglich
- A-Scan, C-Scan für Flächendarstellung

## 33. Reparatur: Allgemeine Verfahren & Potting

### 33.1 Reparatur-Entscheidungsbaum

```
Schaden erkannt
├── Kleine Delle (<50mm Ø)
│   ├── Kein Wassereinbruch → Füllen (Epoxid + Mikroballons) + Skin reparieren
│   └── Mit Wassereinbruch → Trocknen → Füllen → Skin reparieren
├── Delamination (<300mm Ø)
│   ├── Einzelseitig → Skin entfernen → Kern prüfen → Füllen/Ersetzen → Neue Skin
│   └── Beidseitig → Beide Skins entfernen → Kern komplett ersetzen → Neue Skins
├── Großflächig (>300mm Ø)
│   └── Professionelle Reparatur → Kern komplett austauschen → Vakuumbag
└── Durchgehende Beschädigung
    └── Paneel ersetzen
```

### 33.2 Potting-Compounds (Vergussmassen)

| Produkt | Typ | Temperaturbereich | Anwendung |
|---------|-----|-------------------|-----------|
| BR 624 (Syensqo) | Einkomponent, niedrige Dichte | -70 bis +350°F | Kanten, Inserts |
| AP112 (PTM&W) | Zweikomponent Epoxid | Standard | Kanten, Inserts |
| Epoxid + Glaskügelchen | Selbstgemischt | Standard | Füllungen, Dellen |
| Epoxid + Baumwollflocken | Selbstgemischt | Standard | Strukturelle Füllungen |

### 33.3 Kantenversiegelung — Verfahren

| Methode | Beschreibung | Aufwand |
|---------|-------------|--------|
| Fräser + GFK-Umwicklung | Kante fräsen, mit GFK umwickeln | Professionell |
| Spachtelfüllung + GFK | Offene Zellen mit Spachtel füllen, überglasen | DIY möglich |
| Schaumstopfen + GFK | Schaumkeil einsetzen, überglasen | Standard-Marine |
| Vorlaminierte Produkte | Werksseitig versiegelt (Nidaplast) | Am einfachsten |

### 33.4 Through-Deck-Befestigungen — GOLDENE REGEL

> **"Kern muss bei Durchbrüchen entweder während der Laminierung ausgespart ODER nachträglich entfernt und mit verdicktem Epoxid verfüllt werden."**
> — Steve D'Antonio, stevedmarineconsulting.com

| Schritt | Verfahren |
|---------|-----------|
| 1 | Bohrung durch Sandwich |
| 2 | Kern um Bohrung entfernen (überdimensionierter Fräser) |
| 3 | Hohlraum mit verdicktem Epoxid füllen |
| 4 | Aushärten lassen |
| 5 | Durch massives Laminat bohren |
| 6 | Befestigung mit Dichtmittel einsetzen |
| 7 | Backing Plate auf Innenseite (bei hohen Lasten) |

### 34.6 Zusammenfassung Eigner-Erfahrungen

**Positive Berichte:**

- Massive Gewichtseinsparung (600–1.200 lbs auf 36–38' Boot vs. Sperrholz)
- Überlegenes Steifigkeit-zu-Gewicht-Verhältnis
- Hervorragende Leistung bei korrekter Neuinstallation
- Leise, gute Wärme-/Schallisolierung
- Relativ einfach zu verarbeiten (wenn Verklebung beherrscht)
- Reparabel bei frühzeitiger Wasserschaden-Erkennung
- PP-Kern selbst wird durch Wasser nicht beschädigt

**Negative Berichte:**

- Empfindlicher im Betrieb als erwartet
- Anfällig für Dellen und Delaminationen unter moderater Belastung
- Wassereinbruch schwer zu erkennen bis umfangreicher Schaden
- Reparatur arbeitsintensiv und teuer ($3.000–$7.500+ pro Weichstelle)
- Sonneneinstrahlung kann PP-Kerne erweichen
- Schlechte Schraubenhaltung — spezielle Installation nötig
- Nicht für alle strukturellen Anwendungen geeignet
- **Experten-Zitat:** "Nach 20 Jahren Reparatur an Honeycomb-Teilen: viele Dellen und Delaminationen in Teilen, die weniger Belastung aushalten als Rumpfböden"

## 38. Prüfverfahren & NDT

### 38.1 Zerstörungsfreie Prüfung (NDT) — Übersicht

| Methode | Erkennt | Empfindlichkeit | Kosten |
|---------|---------|-----------------|--------|
| **Klopftest** | Delamination, Kernversagen | Grob | Kostenlos |
| **Feuchtemesser** | Wassereinbruch | Mittel | €200–500 |
| **IR-Thermografie** | Wasser, Delamination, Hohlräume | Hoch (0,2ml) | €1.000–5.000 |
| **Ultraschall** | Tiefendefekte, Delamination | Hoch | €2.000–10.000 |
| **Röntgen** | Kernstruktur, Korrosion | Sehr hoch | Spezialist |

### 38.2 Prüfprotokoll Marine-Honeycomb

| Stufe | Methode | Zweck | Häufigkeit |
|-------|---------|-------|-----------|
| 1 | Visuelle Inspektion | Oberflächenbefunde, Dellen | Jährlich |
| 2 | Klopftest | Vorscreening Delamination | Alle 2–3 Jahre |
| 3 | Feuchtemessung | Wassereinbruch bestätigen | Bei Verdacht |
| 4 | IR-Thermografie | Flächige Bewertung | Bei Übernahme, alle 5 Jahre |
| 5 | Ultraschall | Quantitative Tiefenanalyse | Bei Befund |

### 38.3 Thermografie-Verfahren

| Verfahren | Beschreibung | Anwendung |
|-----------|-------------|-----------|
| Passiv | Natürliche Temperaturunterschiede | Screening |
| Aktiv (Flash) | Kurz-Wärmepuls + Kameraaufnahme | Delamination |
| Transient | Kontinuierliche Erwärmung | Tiefere Defekte |
| Lock-In | Modulierte Erwärmung + Fourier-Analyse | Höchste Empfindlichkeit |

### 38.4 Klopftest — Detaillierte Anleitung

**Werkzeuge:**
- Münze (€2-Stück, Durchmesser 25mm) für dünne Sandwich-Paneele
- Klopfhammer (100g, abgerundeter Kopf) für dickere Paneele
- Alternativ: Woodpecker NDT-Klopfgerät (elektronisch, €500–1.500)

**Durchführung:**

| Schritt | Aktion | Hinweis |
|---------|--------|---------|
| 1 | Systematisches Raster definieren (200mm × 200mm) | Sketch auf Deck zeichnen |
| 2 | Referenzton an bekannt gutem Bereich ermitteln | Hoher, klarer Ton = gesund |
| 3 | Raster systematisch abklopfen | Leichte, gleichmäßige Schläge |
| 4 | Tonänderungen markieren | Dumpf/hohl = Delamination verdächtig |
| 5 | Verdächtige Bereiche verifizieren | Feuchtemesser + Thermografie |
| 6 | Befunde dokumentieren | Foto + Position in Deckplan |

**Interpretations-Matrix:**

| Klopfton | Bedeutung | Nächster Schritt | AYDI Confidence |
|----------|-----------|-----------------|----------------|
| Hoch, klar | Gesundes Laminat + Kern | Kein Handlungsbedarf | visual_high |
| Leicht dumpf | Leichte Delamination möglich | Feuchtemesser | visual_medium |
| Deutlich hohl | Delamination wahrscheinlich | IR-Thermografie | visual_low |
| Totschlag (kein Nachhall) | Kern zerstört oder mit Wasser gefüllt | Ultraschall + Probenahme | visual_insufficient |

## 39. Langzeit-Erfahrungsberichte

### 39.1 Nomex auf Regattayachten

| Zeitraum | Erfahrung | Typische Befunde |
|----------|-----------|-----------------|
| 0–5 Jahre | Hervorragende Performance, keine Probleme bei korrekter Installation | Keine |
| 5–10 Jahre | Erste Inspektionsbefunde möglich | Impact-Dellen (Anker, Beschläge), Kanten-Feuchte |
| 10–15 Jahre | Regelmäßige Wartung erforderlich | Klebfugen-Alterung, UV-Degradation offener Stellen |
| 15–20 Jahre | Kernaustausch in stark beanspruchten Zonen möglich | Delamination Bugbereich, Kiel-Befestigungszone |
| 20+ Jahre | Grundlegende Entscheidung: Refit oder Neubau | Großflächige Feuchteaufnahme, Kernintegrität fraglich |

**Praxis-Beispiel IMOCA 60 (Vendée Globe 2008–2024):**
- 4 von 30 Booten mit dokumentierten Nomex-Deck-Problemen (Feuchte nach Impact)
- Häufigste Schadenszone: Bugbereich (Wellen-Impact), Mastspur (punktuelle Höchstlast)
- Erfolgreiche Boote: Hugo Boss, Banque Populaire — konsequente Trocknung + IR-Inspektion nach jeder Saison
- Problematische Boote: Late Repair, unzureichende Kantenversiegelung → kaskadierendes Versagen

**Praxis-Beispiel America's Cup (AC75, 2021–2024):**
- 100% Nomex/CFK-Sandwich in Rumpf + Foils
- Autoklav-Qualität, 100% NDT-geprüft nach Fertigung
- Standzeit: 1–3 Saisons (Kampagne), danach Neubau
- Kein dokumentiertes Kernversagen bei Autoklav-Teilen
- Schwachstelle: Impact-Schäden durch Foiling-Unfälle → lokaler Kernaustausch

### 39.2 PP (Nida-Core) auf Produktionsbooten

| Zeitraum | Erfahrung | Typische Befunde |
|----------|-----------|-----------------|
| 0–5 Jahre | Gute Performance, leise Decks, leichtes Boot | Keine |
| 5–10 Jahre | Erste Dellen an hochbelasteten Stellen | Befestigungs-Ausrisse, Dellen unter Winschen |
| 10–15 Jahre | Weichstellen am Deck, Transom-Probleme wenn PP verwendet | Skin-Kern-Delamination, Wasser in Zellen |
| 15–20 Jahre | Signifikante Reparaturen möglich | Deck-Neubau nicht ungewöhnlich ($20.000+) |
| 20+ Jahre | Erhebliche Alterserscheinungen | Großflächige Weichstellen, Kern-Austausch Standard |

**Praxis-Beispiel Grady-White (USA, PP-Honeycomb seit 1990er):**
- 30+ Jahre Erfahrung mit PP-Honeycomb in Decks und Cockpits
- Positive Bilanz: Gewichtsersparnis, Akustik, Wärmedämmung
- Probleme: Schraubenbefestigungen (nach 10+ Jahren ausgerissen)
- Lösung: Backing Plates + lokale Kernverstärkung ab Werk
- Hauptschwäche: Ältere Modelle ohne werksseitige Kantenversiegelung

**Praxis-Beispiel Mako Boats (USA):**
- Nida-Core in Decks + Cockpit-Boden
- Bis 15 Jahre: gute Performance
- Ab 15 Jahre: 15–20% der Boote zeigen Weichstellen (Surveyor-Daten)
- Korrelation: Boote in Tropen (FL, Bahamas) → höhere Schadensrate

### 39.3 Aluminium auf Marinebooten

| Zeitraum | Erfahrung | Typische Befunde |
|----------|-----------|-----------------|
| 0–5 Jahre | Robust, keine Probleme wenn Korrosionsschutz korrekt | Keine |
| 5–10 Jahre | Kantenversiegelungen prüfen | Beschichtungs-Degradation, Kantenkorrosion |
| 10–15 Jahre | Interne Korrosion möglich wenn Feuchtigkeit eingedrungen | Weißes Pulver an Kanten, lokale Festigkeitsverluste |
| 15–20 Jahre | Bei CFK-Kontakt ohne Isolation: potenziell katastrophal | Galvanische Korrosion, Totalversagen möglich |
| 20+ Jahre | Abhängig von Korrosionsschutz-Qualität | Alu-Kern ohne CFK: oft noch funktional |

**Praxis-Beispiel Militärschiffe (Royal Navy, US Navy):**
- Alu-Honeycomb in Radarmasten, Aufbauten, Hubschrauber-Decks
- 20+ Jahre Standzeit bei regelmäßiger Inspektion + Wartung
- Schlüssel: Kein CFK-Kontakt (GFK-Skins), regelmäßige Neubeschichtung
- IMO-Zertifizierung für Brandschutz (nicht brennbar)
- Erfahrung: Alu-Honeycomb ist extrem langlebig WENN galvanisches Risiko ausgeschlossen

**Praxis-Beispiel Superyacht-Interieur (55m+):**
- Alu-Honeycomb-Möbelpaneele: 15+ Jahre ohne Probleme
- Vorteil: Innenraum = keine Feuchte, keine UV, keine galvanische Korrosion
- Gewichtsersparnis: 40–60% vs. Marine-Sperrholz
- Einziges Problem: Knarrgeräusche bei Temperaturwechseln (WAK-Differenz Alu/Furnier)

### 39.4 Vergleichendes Langzeit-Fazit

| Kriterium | Nomex | PP | Aluminium |
|-----------|-------|-----|-----------|

| Surveyor-Akzeptanz | Hoch (bekannte Technik) | Mittel (Erfahrung variiert) | Hoch (Militär-Referenzen) |

## 42. FAQ — Häufige Fragen zu Honeycomb im Bootsbau

### F1: Kann ich Nida-Core für meinen Transom verwenden?
**A:** Nein. Nida-Core (PP-Honeycomb) hat unzureichende Druckfestigkeit für Motorlasten. Coosa Board oder Sperrholz-Laminat (Meranti) verwenden. Nida-Bond (Vergussmasse, anderes Produkt) kann für Recore-Reparaturen verwendet werden.

### F2: Ist Nomex oder PVC-Schaum besser für meinen Fahrtenyacht-Rumpf?
**A:** PVC-Schaum (Divinycell H80/H100) — geschlossene Zellen, infusionsfähig, keine Feuchteprobleme. Nomex ist 20–30% leichter, aber hygroskopisch, nicht infusionsfähig und teurer. Nomex lohnt sich bei Regattayachten, nicht bei Fahrtenbooten.

### F3: Warum ist Aluminium-Honeycomb mit Carbon-Skins problematisch?
**A:** Galvanische Korrosion — 1,2V Potentialdifferenz in Seewasser. Aluminium korrodiert bevorzugt. Lösung: GFK-Isolationsschicht (min. 0,5mm) zwischen CFK-Skin und Alu-Kern einfügen. Oder Nomex-Kern verwenden.

### F4: Kann ich Nomex-Honeycomb vakuuminfundieren?
**A:** Nein. Offene Zellen füllen sich mit Harz → keine Kernfunktion, massives Übergewicht. Prepreg + Vakuumbag oder Nasslamination + Vakuumbag verwenden.

### F5: Wie lange hält Honeycomb im Bootsbau?
**A:** Bei korrekter Installation und Wartung: PP 15–25 Jahre, Nomex 20–30+ Jahre, Alu 20–30+ Jahre (wenn Korrosionsschutz intakt). Hauptversagensursache ist immer Wassereinbruch durch äußere Beschädigung.

### F6: Welchen Honeycomb für DIY-Bootsbau?
**A:** PP (Nida-Core H8PP) — günstigster, mit allen Harzen kompatibel, vergebende Verarbeitung. Für Hochleistung: Professionelle Beratung für Nomex/Prepreg einholen.

### F7: Was kostet ein Quadratmeter Honeycomb-Kern?
**A:** PP: €3–8/m², Alu 5052: €12–18/m², Nomex 48: €18–35/m². Hinzu kommen Skins, Kleber, Verarbeitung. Gesamt-Sandwichplatte: €20–100/m² je nach Konfiguration.

### F8: Wie repariere ich eine Weichstelle im Honeycomb-Deck?
**A:** NICHT injizieren! Skin entfernen → beschädigten Kern herausnehmen → trocknen → neuen Kern oder Epoxid-Füllung einsetzen → neue Skin laminieren → Kanten versiegeln. Kosten: $2.500–7.500.

### F9: Welche Prüfung bei Gebrauchtboot-Kauf?
**A:** Klopftest (Vorscreening) → Feuchtemessung (Bestätigung) → IR-Thermografie (bei Verdacht). Budgetieren Sie €500–2.000 für professionelle Composite-Prüfung.

### F10: PP-Honeycomb im Maschinenraum?
**A:** NUR als Decke/Overhead, NICHT als Primärstruktur. Temperaturgrenze 60°C für Dauerlast. Ab 80°C: Kriechgefahr. Alternative: Alu-Honeycomb (nicht brennbar, temperaturbeständig).

### F11: Was ist der Unterschied zwischen Nida-Core und Nidaplast?
**A:** Nidaplast (Frankreich) ist der Rohmaterialhersteller. Nida-Core nutzt Nidaplast-Produkt für den US-Markt unter eigenem Branding. Technisch identisch.

### F12: Honeycomb vs. Schaum — wann Honeycomb?
**A:** Honeycomb wenn: maximale Gewichtsersparnis nötig (Regatta), PP-Kosten-Vorteil gewünscht (Serie), oder IMO-Brandschutz gefordert (Nomex/Alu). Schaum wenn: Infusion, einfachere Verarbeitung, bessere Feuchtebeständigkeit.

### F13: Wie erkenne ich galvanische Korrosion am Alu-Honeycomb?
**A:** Weiße/graue Ablagerungen an Kanten, Skin-Ablösung, Gewichtszunahme, Festigkeitsverlust. IR-Thermografie zeigt Feuchtigkeitsmuster. Worst Case: Sandwich fühlt sich "schwammig" an.

### F14: Kann ich verschiedene Kernmaterialien im selben Boot mischen?
**A:** Ja — Mixed-Core-Konzept ist Standard. PP für Decks/Interieur, PVC für Rumpf, Coosa für Transom, Nomex für Aufbau. Übergangszonen sauber gestalten (Scarf oder Überlappung, Skin durchlaufend).

### F15: Gibt es recycelbares Honeycomb?
**A:** PP ist generell recycelbar. EconCore/ThermHex entwickeln rPET-Honeycomb aus recycelten Flaschen. Nomex und Alu sind thermisch/chemisch recycelbar. GFK-Skins bleiben das Recycling-Problem.

### F16: Wie behandle ich Inserts / Befestigungspunkte in Honeycomb?
**A:** NIEMALS direkt in Honeycomb-Kern schrauben. Verfahren: Kern lokal aushöhlen (Stufenfräser) → Vergussmasse (Potting Compound) injizieren → Insert einsetzen → Aushärten → In massives Potting-Material bohren. Alternativ: Werksseitig eingelegte Inserts + Backing Plates.

### F17: Welche Zellweite für welche Anwendung?
**A:** Grundregel — kleine Zellen = höhere Festigkeit, aber teurer und schwerer. Marine-Standard: 3,2mm (Racing, Prepreg), 4,8mm (Regatta/Semi-Profi), 6,4mm (Standard-Marine), 8,0mm (PP Serie). Alu Marine: 6,4–12,7mm (Interieur/Möbel).

### F18: Muss ich Honeycomb trocknen vor der Verarbeitung?
**A:** Nomex: JA, unbedingt! 4h bei 70°C oder 24h im Trockenschrank. Nomex ist hygroskopisch. Feuchter Kern → Blasenbildung bei Warmhärtung, schwache Klebfuge. PP/Alu: In der Regel nicht nötig, aber trockene Lagerung sicherstellen.

### F19: Kann Honeycomb-Kern selbst Wasser aufnehmen?
**A:** Nomex-Kern: JA (4–8% bei langfristigem Kontakt). PP-Kern: NEIN (0%). Alu-Kern: NEIN (0%). ABER: Bei allen Typen kann Wasser in die Zellhohlräume eindringen wenn Kanten oder Skins beschädigt sind. Der Zellhohlraum ist das Problem, nicht das Material selbst (außer bei Nomex).

### F20: Was bedeutet "OX" oder "Over-Expanded" Honeycomb?
**A:** Über-expandierter Honeycomb hat rechteckige statt hexagonale Zellen. Er lässt sich um eine Achse biegen (Krümmungsradien bis 300mm). Ideal für zylindrische oder konische Formen (Masten, Ruderblätter). Festigkeit in einer Richtung reduziert (~50%). Verfügbar bei Hexcel, Plascore, Euro-Composites.

## ANHANG A — Detaillierte Berechnungsbeispiele

### A.1 Sandwich-Plattenberechnung: Deck mit Nomex-Kern

**Aufgabe:** Dimensionierung einer Deckplatte für eine 14m-Segelyacht (CE Kat. B)

**Eingangsdaten:**

| Parameter | Wert | Quelle |
|-----------|------|--------|
| Plattenlänge (a) | 800 mm | Spantenabstand |
| Plattenbreite (b) | 500 mm | Stringerabstand |
| Designdruck (p) | 12 kPa | ISO 12215-5 Kat. B Deck |
| Kern | Nomex HRH-78, 48 kg/m³ | Herstellerdaten |
| Skins | E-Glas/Epoxid Biax 600g/m² | Standardlaminat |
| Sicherheitsfaktor | 2,5 | ISO 12215 |

**Berechnung Kerndicke:**

Erforderliche Kernscherfestigkeit: τ_kern ≥ (p × b) / (2 × t_kern) × SF

Mit Nomex 48 kg/m³: τ_zulässig = 1,25 MPa (L-Richtung) / SF = 0,50 MPa

Erforderliche Kerndicke für Scherversagen:
t_kern ≥ (12.000 Pa × 0,5 m) / (2 × 500.000 Pa) = 6,0 mm

**Berechnung Skin-Dicke:**

Skin-Biegespannung: σ_skin = (p × b²) / (8 × t_skin × (t_kern + t_skin))

Mit E-Glas σ_zulässig = 120 MPa / SF = 48 MPa:
t_skin ≥ 1,0 mm pro Seite (2 Lagen Biax 300g/m² = ~0,6mm → 2 Lagen 600g/m² = ~1,2mm)

**Ergebnis:**

| Parameter | Wert | Bemerkung |
|-----------|------|-----------|
| Kerndicke | 10 mm | Nomex HRH-78, 48 kg/m³, 3,2mm Zelle |
| Skin-Dicke | 1,2 mm pro Seite | 2 × Biax 600g/m² E-Glas/Epoxid |
| Gesamtdicke | 12,4 mm | |
| Gewicht | ~4,5 kg/m² | vs. ~12 kg/m² massiv GFK |
| Gewichtsersparnis | ~63% | Signifikant |
| Steifigkeit | ~5× höher | Gegenüber 4mm massivem GFK gleichen Gewichts |
| Confidence | calculated | Aus gemessenen Daten berechnet |

```
  - Oberfläche: Polyester-Vlies beidseitig
  - Kerndicke: 25,0 ± 0,5 mm (H8PP-25)
  - DNV Marine Class zertifiziert

Skins: E-Glas/Polyester oder Vinylester
  - Ober-Skin: CSM 450 + Biax 600 (1,8 mm nom.)
  - Unter-Skin: CSM 450 + Biax 600 (1,8 mm nom.)

Verarbeitung: Handlaminierung oder Vakuuminfusion
  - Bei Infusion: PCGA-XR Infusionsqualität verwenden
  - Alle Kanten vor Laminierung versiegeln

Befestigungen: KEINE direkte Verschraubung in Kern
  - Alle Durchbrüche: Kern entfernen + Epoxid verfüllen
  - Backing Plates min. 50×50mm für leichte Last
  - Backing Plates min. 100×100mm für Decksbeschläge
  - Windlass, Winschen: Separate Strukturverstärkung

Temperatur: Max. 60°C Dauerbelastung
  - Nicht für Maschinenraum-Primärstruktur
  - Dunkel gestrichene Flächen in Tropen: Temperaturcheck
```

### F.2 Automatische AYDI-Regeln (Pseudo-Code)

```python
def honeycomb_empfehlung(zone, bootsklasse, ce_kategorie, budget_level):
    """
    AYDI-Empfehlungslogik für Honeycomb-Kernmaterial.
    Gibt (empfohlen: bool, typ: str, confidence: str) zurück.
    """

    # VERBOTENE Zonen
    if zone in ["transom", "stringer", "kiel", "rumpf_unterwasser"]:
        return (False, "kein_honeycomb", "measured")

    # MASCHINENRAUM → Nur Aluminium
    if zone == "maschinenraum_schott":
        return (True, "aluminium_5052_IMO", "measured")

    # REGATTA → Nomex bevorzugt
    if bootsklasse == "regatta":
        if zone in ["deck", "aufbau", "cockpit", "rumpf_ueberwasser"]:
            return (True, "nomex_48_prepreg", "measured")

    # SUPERYACHT → Nomex Struktur, Alu Möbel
    if bootsklasse == "superyacht":
        if zone in ["deck", "aufbau", "rumpf_ueberwasser"]:
            return (True, "nomex_64_oder_96", "measured")
        if zone in ["moebel", "schotten_nicht_tragend"]:
            return (True, "aluminium_5052", "measured")

    # SERIENBOOT → PP dominant
    if bootsklasse in ["produktionssegler", "motorboot_serie"]:
        if zone in ["deck", "aufbau", "moebel", "schotten_nicht_tragend", "luken", "tueren"]:
            return (True, "pp_h8pp", "measured")

    # SEMI-CUSTOM → Mixed
    if bootsklasse == "semi_custom":
        if zone in ["deck", "aufbau"]:
            return (True, "nomex_48_oder_pp", "estimated")
        if zone in ["moebel", "interieur"]:
            return (True, "pp_h8pp", "measured")

    # DEFAULT
    return (False, "keine_empfehlung", "estimated")
```

```python
                empfehlung["typ"], alter_jahre, skin_material
            )

            result["empfehlungen"].extend(degradation["empfehlungen"])

        # Zustandsdaten integrieren (wenn vorhanden)
        if zustand_daten:
            zustand = self._assess_condition(zustand_daten)
            result["warnungen"].extend(zustand["warnungen"])
            result["score"] = zustand["score"]

        return result
```
