## Zusammenfassung

Cyanacrylat-Klebstoffe (CA, umgangssprachlich "Sekundenkleber") sind einkomponentige, feuchtigkeitshärtende Klebstoffe auf Basis von Alkyl-2-Cyanoacrylat-Monomeren. Im Yachtbau spielen sie eine spezifische, wenn auch begrenzte Rolle: als Sofort-Fixierkleber, Notfallreparatur-Medium, Dichtungs-Eckenfixierung, Beschleuniger für andere Klebstoffe, und als hochspezialisierter Kleber für Gummi-, Kunststoff- und Metallverbindungen in nicht-strukturellen Anwendungen.

Die wichtigsten Hersteller im Marine-Umfeld sind Henkel (Loctite), 3M (Scotch-Weld), Permabond, Bohle, Panacol (Vitralit), Cyberbond, Delo, Kisling (Ergo), Bostik (Bison), Hobbyking/BSI (Bob Smith Industries), Satellite City, Great Planes (Pro CA), Starbond, Mercury Adhesives, Mitreapel, Infinity Bond, Pacer Technology (Zap), HB Fuller (Purbond), Weicon, Threebond, Loxeal, Permacol, und Parson Adhesives.

**Kernprinzip für Yachtbau:** CA-Klebstoffe sind KEINE Strukturkleber. Sie dienen zur temporären Fixierung, Notfallreparatur, und für spezifische nicht-lasttragende Verbindungen. Für strukturelle Marine-Verklebungen: MMA (Plexus, Scigrip) oder Epoxid verwenden.

## 1. Grundlagen der Cyanacrylat-Chemie

### 1.1 Chemische Basis

Cyanacrylate (CA) sind Ester der 2-Cyanacrylsäure. Die Polymerisation wird durch Spuren von Feuchtigkeit (OH⁻-Ionen) auf der Substratoberfläche initiiert — anionische Polymerisation.

| Parameter | Wert | Anmerkung |
|-----------|------|-----------|
| Monomer-Formel | CH₂=C(CN)(COOR) | R = Alkylgruppe |
| Polymerisations-Typ | Anionisch (Feuchtigkeit als Initiator) | Keine Mischung nötig |
| Aushärtezeit (dünn) | 5–30 s | Abhängig von Feuchte + Spalt |
| Aushärtezeit (dick) | 1–10 min | Tiefenaushärtung limitiert |
| Max. Spaltfüllung | 0,05–0,5mm (Standard) | Gel-Typen bis 5mm |
| Betriebstemperatur | -55°C bis +80°C (Standard) | Hochtemp.-Typen bis 120°C |
| Zugscherfestigkeit | 10–25 MPa | Substratabhängig |
| Bruchdehnung | 2–10% (Standard) | Flexible Typen bis 200% |
| Wasserbeständigkeit | Mäßig bis gut | Langzeit-Immersion problematisch |
| UV-Beständigkeit | Schlecht (Standard) | UV-Typen verfügbar |

### 1.2 Monomer-Typen und ihre Eigenschaften

| Monomer | Handelsbezeichnung | Viskosität | Flexibilität | Temperatur | Marine-Eignung |
|---------|-------------------|-----------|-------------|-----------|----------------|
| Methyl-CA | Standard-Sekundenkleber | Niedrig | Spröde | -55 bis +80°C | Eingeschränkt — spröde |
| Ethyl-CA | Loctite 4xx-Serie | Niedrig–Mittel | Mäßig | -55 bis +80°C | Standard-Marine |
| n-Butyl-CA | Medizinisch (Histoacryl) | Mittel | Flexibel | -20 bis +60°C | Nein (medizinisch) |
| 2-Octyl-CA | Dermabond | Mittel | Sehr flexibel | -20 bis +50°C | Nein (medizinisch) |
| Allyl-CA | Hochtemperatur | Niedrig | Spröde | -55 bis +120°C | Motorraum-Fixierung |
| Ethoxyethyl-CA | Loctite 4090 (Hybrid) | Hoch | Hoch | -55 bis +150°C | Ja — bester Marine-Typ |
| Methoxyethyl-CA | Geruchsarm | Niedrig | Mäßig | -55 bis +80°C | Ja — Innenraum |

### 1.3 Vor- und Nachteile im Yachtbau

**Vorteile:**
| Eigenschaft | Marine-Relevanz |
|------------|-----------------|
| Sofortige Fixierung (5–30s) | Notfallreparatur auf See |
| Einkomponentig, kein Mischen | Einfachste Verarbeitung |
| Kleinstmengen-Dosierung | Dichtungs-Ecken, Kabel, Beschriftungen |
| Gute Haftung auf Metall, Gummi, Kunststoff | Vielseitig auf dem Boot |
| Kein Lösemittel (reaktiv) | Geringe Gesundheitsgefahr |

**Nachteile:**
| Eigenschaft | Marine-Relevanz |
|------------|-----------------|
| Nicht strukturell (spröde) | KEIN Ersatz für MMA/Epoxid |
| Schlechte Wasserbeständigkeit | Langzeit-Immersion = Versagen |
| Schlechte UV-Beständigkeit | Deck/Außen = Versprödung |
| Begrenzte Spaltfüllung (≤0,5mm) | Nur für passgenaue Verbindungen |
| Blooming (weiße Ränder) | Ästhetisch problematisch |
| Verklebt Haut sofort | Sicherheitsrisiko |

### 14.3 CA + Backpulver (NaHCO₃) — Der Marine-Notfall-Füllstoff

Die Kombination CA + Backpulver erzeugt einen extrem harten, schleifbaren Füllstoff in Sekunden:

| Parameter | Detail |
|-----------|--------|
| Prinzip | NaHCO₃ beschleunigt CA-Polymerisation (alkalisch) + dient als Füllstoff |
| Mischverhältnis | Schichtweise: Backpulver einfüllen, CA drauf, Backpulver, CA... |
| Aushärtezeit | 5–10 s pro Schicht |
| Endhärte | Shore D 80+ (härter als GFK-Gelcoat) |
| Bearbeitbar | Schleifen (P80–P400), Bohren, Gewindeschneiden |
| Max. Tiefe | 10–15mm (tiefere Löcher schichtweise aufbauen) |
| Kosten | ~0,10 EUR pro Reparatur |
| Marine-Anwendungen | Ausgerissene Schraubenlöcher, Gelcoat-Chips, Teak-Löcher |

> **Experten-Zitat (E24):** „Ich habe damit mitten auf dem Atlantik ein ausgerissenes Winschen-Loch geflickt — Backpulver rein, Sekundenkleber drauf, 3 Schichten, Loch neu gebohrt. Hält seit 4 Jahren." — *Cruisers Forum, User 'Ocean_Dave', Thread "Emergency Repairs at Sea", 2022*

## 15. Fehlerbilder und Diagnose

### 15.1 Fehlerbild 1: Blooming (weiße Ränder um Klebefuge)

| Detail | Information |
|--------|------------|
| Symptom | Weiße, pudrige Ablagerung rund um CA-Klebefuge |
| Ursache | CA-Monomer verdampft, polymerisiert an feuchter Luft und lagert sich als weißer Film ab |
| Begünstigend | Hohe Luftfeuchte (>70%), viel überschüssiger CA, geschlossene Räume |
| Vermeidung | Wenig CA verwenden, Aktivator statt Überschuss, geruchsarme Typen (460, BSI Super-Gold+) |
| Beseitigung | Aceton-getränktes Tuch, vorsichtig abwischen (NICHT auf PMMA!) |
| AYDI Score-Impact | -5 bis -15 Punkte (ästhetisch, nicht strukturell) |

> **Experten-Zitat (E25):** „Blooming ist der Grund warum CA-Klebefugen auf polierten Oberflächen so schlecht aussehen — der weiße Belag ist polymerisiertes Cyanacrylat-Monomer. Lösung: geruchsarme CA verwenden (Loctite 460) oder Aktivator-Spray benutzen um die Reaktion schnell abzuschließen." — *Dr. Stefan König, Kunststoff-Institut Lüdenscheid, 2024*

### 15.2 Fehlerbild 2: CA härtet nicht aus

| Detail | Information |
|--------|------------|
| Symptom | CA bleibt flüssig oder klebrig |
| Ursache 1 | Saure Oberfläche (Holzgerbsäure, saurer GFK-Staub) — inhibiert anionische Polymerisation |
| Ursache 2 | Zu trockene Umgebung (<30% r.F.) |
| Ursache 3 | Abgelaufener CA (Polymerisation im Behälter) |
| Lösung 1 | Aktivator (7452) auf eine Fügefläche sprühen |
| Lösung 2 | Oberfläche anhauchen (Atemfeuchte) |
| Lösung 3 | Frischen CA verwenden |
| AYDI Score-Impact | -20 Punkte (Prozessfehler) |

### 15.3 Fehlerbild 3: Spröder Bruch der Klebefuge

| Detail | Information |
|--------|------------|
| Symptom | Klebefuge bricht glatt bei geringer Belastung |
| Ursache 1 | Standard-CA ist spröde (2–5% Bruchdehnung) |
| Ursache 2 | Vibration (Motor, Seegang) ermüdet spröde CA-Fuge |
| Ursache 3 | Temperaturschwankungen (CTE-Unterschiede) |
| Lösung | Toughened CA verwenden (480, 240, Cyberbond 2240) oder Hybrid (4090) |
| AYDI Score-Impact | -15 Punkte |

### 15.4 Fehlerbild 4: Klebefuge löst sich in feuchter Umgebung

| Detail | Information |
|--------|------------|
| Symptom | Klebefuge erweicht oder löst sich nach Wochen/Monaten im feuchten Bereich |
| Ursache | Standard-Ethyl-CA ist nicht wasserbeständig — hydrolysiert bei Dauerfeuchtigkeit |
| Begünstigend | Bilge, Nassbereich, unbelüftete Räume >80% r.F. |
| Lösung | Feuchtigkeitsresistenten CA verwenden (4062, Cyberbond 2240) oder Hybrid (4090) |
| Dauerhafte Lösung | Für dauerhafte Nassbereich-Verbindungen: MMA oder PU statt CA |
| AYDI Score-Impact | -20 Punkte |

### 15.5 Fehlerbild 5: Hautverklebung (Sicherheitsvorfall)

| Detail | Information |
|--------|------------|
| Symptom | Finger zusammengeklebt, CA auf Haut |
| Ursache | Sofortige Polymerisation auf feuchter Haut |
| Sofortmaßnahme 1 | Aceton (Nagellackentferner) auf Klebstelle, 2–3 min einwirken, vorsichtig lösen |
| Sofortmaßnahme 2 | BSI Un-Cure oder Loctite CA-Löser auftragen |
| Sofortmaßnahme 3 | NICHT mit Gewalt reißen! Warmes Seifenwasser, Geduld |
| Augenkontakt | Sofort mit Wasser spülen (15 min), ARZT — CA verklebt Augenlider aber löst sich in 1–3 Tagen von selbst |
| Prävention | Nitril-Handschuhe (NICHT Latex — CA verklebt Latex sofort!) |
| AYDI Score-Impact | N/A (Sicherheit) |

> **Experten-Zitat (E26):** „Die wichtigste Regel: Nitril-Handschuhe statt Latex. CA polymerisiert auf Latex-Handschuhen sofort und exotherm — man kann sich verbrennen. Nitril ist resistent." — *Ralf Niedermayer, IHK Klebstoff-Sachverständiger, 2024*

### 15.6 Fehlerbild 6: UV-Degradation (Außenbereich)

| Detail | Information |
|--------|------------|
| Symptom | Klebefuge wird gelb, dann braun, dann spröde |
| Zeitrahmen | 3–12 Monate bei direkter Sonneneinstrahlung |
| Ursache | Standard-CA hat keine UV-Stabilisatoren |
| Lösung | UV-stabile CA verwenden (4310, Panacol Vitralit) oder Klebefuge überlackieren |
| Dauerhafte Lösung | Für UV-exponierte Verbindungen: MMA oder PU statt CA |
| AYDI Score-Impact | -10 bis -25 Punkte |

### 15.7 Fehlerbild 7: Wärmeversagen (Motorraum)

| Detail | Information |
|--------|------------|
| Symptom | Klebefuge erweicht bei >80°C |
| Ursache | Standard-CA hat Tg ~80°C |
| Lösung | Hochtemperatur-CA verwenden (Permabond 105, Cyberbond 2999, Weicon VA 1500 HT) |
| Alternativ | Loctite 4090 Hybrid (bis 150°C) |
| AYDI Score-Impact | -15 Punkte |

### 15.8 Fehlerbild 8: Exotherme Reaktion auf Baumwolle/Textil

| Detail | Information |
|--------|------------|
| Symptom | CA auf Textil/Baumwolle erzeugt starke Hitze, Rauchentwicklung, Brandgefahr! |
| Ursache | Große Oberfläche der Fasern → extrem schnelle Polymerisation → starke Exothermie |
| WARNUNG | **BRANDGEFAHR! CA nie auf Baumwolle, Wolle, Leder in großen Mengen!** |
| Marine-Relevanz | Segeltuch, Polster, Taue — NUR Kleinstmengen, NIE einweichen |
| AYDI Score-Impact | N/A (Sicherheit) |

> **Experten-Zitat (E27):** „Ich habe einen Werftmitarbeiter gesehen der sich mit CA-getränktem Lappen die Hand verbrannt hat — die exotherme Reaktion auf Baumwolle kann 200°C+ erreichen. Steht auf jedem Sicherheitsdatenblatt, aber keiner liest es." — *Dr. Andrea Rossi, ISVA, 2024*

## Anhang D: Troubleshooting-Entscheidungsbaum

### Problem: CA härtet nicht aus
```
CA härtet nicht?
├── Oberfläche sauer? (Teak, Eiche, saurer GFK)
│   └── JA → Aktivator 7452 verwenden
├── Luftfeuchte <30%?
│   └── JA → Oberfläche anhauchen oder Aktivator
├── CA abgelaufen?
│   └── JA → Neuen CA verwenden
├── Spalt >0,5mm?
│   └── JA → Gel-Typ (454, 910) oder 3090 verwenden
└── Oberfläche kontaminiert?
    └── JA → IPA reinigen, trocknen, erneut versuchen
```

### Problem: Klebefuge versagt
```
Klebefuge gebrochen?
├── Glatter Bruch (adhäsiv)?
│   ├── Oberfläche kontaminiert? → Reinigen + Primer
│   └── Falscher CA für Substrat? → Cross-Referenz prüfen
├── Spröder Bruch (kohäsiv)?
│   ├── Vibration/Schock? → Toughened CA (480, 240)
│   └── Temperaturwechsel? → 4090 Hybrid
├── CA weich geworden?
│   ├── Feuchtigkeit? → 4062 (feuchteresistent)
│   └── Temperatur >80°C? → Permabond 105 / 4090 Hybrid
└── Blooming (weiß)?
    └── Geruchsarmen CA (460, BSI Super-Gold+) verwenden
```

## Anhang F: FAQ — Häufig gestellte Fragen

### FAQ 1: Kann ich mit Sekundenkleber ein Leck im Boot abdichten?
**Antwort:** Nur als absolute Notlösung für Mikrolecks. CA härtet nicht unter Wasser und hat keine Wasserbeständigkeit für Langzeit-Immersion. Für Leckdichtung: Belzona 1111, MarineTex, oder Epoxid-Unterwasserkitt (SubCoat). CA kann bestenfalls einen Haarriss von der trockenen Seite füllen.

### FAQ 2: Hält Sekundenkleber auf nassem Substrat?
**Antwort:** Standard-CA nein — Feuchtigkeit auf der Oberfläche startet die Polymerisation bevor der CA ins Substrat fließen kann → schlechte Haftung. Loctite 4062 (feuchtetolerant) funktioniert etwas besser, aber trockene Substrate sind immer vorzuziehen.

### FAQ 3: Warum ist Loctite 406 besser als Pattex Sekundenkleber?
**Antwort:** Loctite 406 ist ein Industrie-CA mit definierter Viskosität, reproduzierbaren Festigkeitswerten, und Chargenqualität. Consumer-CA (Pattex, UHU) hat variable Qualität, kürzere Haltbarkeit, und geringere Feuchtebeständigkeit. Für Marine: immer Industrie-CA.

### FAQ 4: Muss CA im Kühlschrank gelagert werden?
**Antwort:** Empfohlen, nicht zwingend. Kühlschranklagerung (2–8°C) verlängert die Haltbarkeit auf 12+ Monate. Bei Raumtemperatur: 6 Monate ungeöffnet, 2–3 Monate geöffnet. Auf dem Boot: Ziplock mit Silica-Gel im kühlsten Schapp.

### FAQ 5: Was ist der Unterschied zwischen 406 und 480?
**Antwort:** 406 ist dünnflüssig, klar, schnell (5s), spröde. 480 ist mittelviskos, schwarz, langsamer (30s), aber zäh (toughened) mit 3–5× höherer Schälfestigkeit. 406 für Fixierung, 480 für Gummi und vibrationsbeanspruchte Verbindungen.

### FAQ 6: Kann ich CA als Schraubensicherung verwenden?
**Antwort:** Ja, als Notlösung auf Kunststoff- und Holz-Gewinden. Für Metall-Gewinde ist Loctite Threadlocker (222/243/272) besser geeignet — CA füllt nicht das gesamte Gewindefreispiel und ist spröde.

### FAQ 7: Wie entferne ich ausgehärteten CA?
**Antwort:** (1) Aceton — erweicht CA in 5–15 min, dann mechanisch abkratzen. (2) Wärme — >150°C erweicht CA. (3) Spezial-Löser: BSI Un-Cure, Loctite CA Remover. (4) Mechanisch: Cutter, Schaber. Auf PMMA/PC: KEIN Aceton — nur mechanisch!

### FAQ 8: Gibt es CA der unter Wasser härtet?
**Antwort:** Nein. CA benötigt Oberflächenfeuchtigkeit zur Initiation, aber kann nicht in Wasser-Immersion polymerisieren — das Wasser verdünnt das Monomer bevor Polymerisation stattfindet. Für Unterwasser-Verklebung: Epoxid-Unterwasser-Kits (Belzona, SplashZone).

### FAQ 9: Ist CA giftig?
**Antwort:** Gering giftig. CA-Monomerdämpfe reizen Augen und Atemwege (H315, H319, H335), sind aber nicht krebserregend und nicht chronisch toxisch. MAK-Wert: 0,2 ppm (Ethyl-CA; ACGIH TLV-TWA, 1 ppm STEL). Bei normaler Verwendung (kleine Mengen, belüftet) kein Gesundheitsrisiko. Problematisch nur in geschlossenen Räumen bei großen Mengen.

### FAQ 10: Kann ich CA auf PVC-Schläuchen verwenden?
**Antwort:** Ja — CA haftet gut auf Hart-PVC. Auf Weich-PVC (Schläuche) ist die Haftung mäßig da der Weichmacher die Oberfläche kontaminiert. Oberfläche mit IPA reinigen und schnell kleben. Für dauerhafte Schlauchverbindungen: Schlauchschelle + Dichtmittel.

### FAQ 11: Was kostet eine typische CA-Bordapotheke?
**Antwort:** Minimal: 406 + 454 + Backpulver = ~25 EUR. Empfohlen: 406 + 480 + 454 + 7452 + Un-Cure = ~76 EUR. Premium: + 4090 Hybrid + 4310 UV + 770 Primer = ~200 EUR. Haltbarkeit beachten — kleine Flaschen kaufen und nach 6 Monaten ersetzen.

### FAQ 12: Wie verhindere ich Blooming?
**Antwort:** (1) Geruchsarmen CA verwenden (460, BSI Super-Gold+). (2) Aktivator-Spray auf offene Klebefugen — beschleunigt Oberflächenhärtung. (3) Weniger CA verwenden — ein Tropfen reicht meist. (4) Gute Belüftung — reduziert Monomer-Konzentration.

### FAQ 13: Kann ich CA-Klebefugen lackieren?
**Antwort:** Ja — nach vollständiger Aushärtung (24h+) kann CA geschliffen (P320+) und mit 2K-PU-Lack oder Epoxid-Primer beschichtet werden. Einschränkung: Lösemittel im Lack können frische CA-Fuge anlösen — erst nach Vollhärtung lackieren.

### FAQ 14: Warum klebt CA nicht auf meinem Teakholz?
**Antwort:** Teak enthält natürliche Öle und Gerbsäuren die CA-Polymerisation hemmen. Lösung: (1) Oberfläche mit Aceton entfetten (2) Aktivator 7452 auftragen (3) CA sofort auftragen. Alternativ: Medium-CA (Starbond EM-150) der besser in Holzporen eindringt.

### FAQ 15: CA vs. UV-Kleber — was ist besser für optische Verklebungen?
**Antwort:** UV-Kleber (Loctite 4310, Panacol Vitralit) geben die optisch klarste, UV-stabilste Verbindung. Standard-CA vergilbt im UV-Licht. Für Instrumenten-Linsen, Display-Abdeckungen: UV-CA. Für unsichtbare Arbeiten hinter Verkleidung: Standard-CA ausreichend.

### CA-spezifische Scoring-Funktion

```python
def calculate_ca_bond_score(
    visual_score: float,
    age_months: float,
    product_tier: str,  # "industrial", "consumer"
    environment: str,  # "interior_dry", "interior_humid", "exterior", "engine_room", "bilge"
    is_toughened: bool,
    boat_class: str
) -> dict:
    """
    AYDI Scoring-Funktion für CA-Klebefugen.

    model_config = {"from_attributes": True}
    """
    base = visual_score

    # CA altert schneller als andere Klebstoffe
    age_factors = {
        "industrial": 0.04,   # 4% pro Monat (48%/Jahr)
        "consumer": 0.08      # 8% pro Monat (96%/Jahr)
    }
    age_penalty = min(age_months * age_factors.get(product_tier, 0.06) * 100, 60)

    # Umgebungsfaktor (CA ist empfindlich)
    env_factors = {
        "interior_dry": 1.00,
        "interior_humid": 0.80,
        "exterior": 0.70,
        "engine_room": 0.75,
        "bilge": 0.60
    }
    env = env_factors.get(environment, 0.75)

    # Toughened-Bonus
    tough_factor = 1.10 if is_toughened else 1.00

    # Bootklassen-Erwartung
    class_factors = {
        "production": 0.95,
        "semi_custom": 1.00,
        "custom_superyacht": 1.05
    }
    cls = class_factors.get(boat_class, 1.00)

    adjusted = (base - age_penalty) * env * tough_factor * cls
    final_score = max(0, min(100, round(adjusted)))

    if final_score >= 90:
        action = "Routinekontrolle — CA-Klebefuge intakt"
    elif final_score >= 75:
        action = "Monitoring — bei nächster Wartung prüfen"
    elif final_score >= 60:
        action = "Erneuerung planen — CA erneuern in nächster Saison"
    elif final_score >= 40:
        action = "Reparatur nötig — CA erneuern"
    elif final_score >= 20:
        action = "Sofortige Reparatur — CA versagt"
    else:
        action = "Neuklebung mit geeignetem Klebstofftyp prüfen (MMA/PU statt CA?)"

    return {
        "score": final_score,
        "action": action,
        "confidence": "calculated",
        "components": {
            "visual_base": base,
            "age_penalty": round(age_penalty, 1),
            "environment_factor": env,
            "toughened_factor": tough_factor,
            "class_factor": cls
        }
    }
```

## Anhang U: Erweiterte FAQ 16–30

### FAQ 16: Kann ich CA für Unterwasser-Zinkanoden verwenden?
**Antwort:** Nein — CA hydrolysiert in Salzwasser-Immersion. Zinkanoden mechanisch befestigen (Schrauben, Bolzen) oder mit Unterwasser-Epoxid (Belzona 1111). CA-Klebefugen lösen sich in 2–6 Monaten unter Wasser vollständig auf.

### FAQ 17: Warum klebt mein CA nicht auf nassem Edelstahl?
**Antwort:** Wasserfilm auf Stahl verhindert CA-Haftung. Oberfläche trocken wischen, mit IPA reinigen, sofort kleben. CA braucht SPUREN von Feuchtigkeit (Luftfeuchte), aber KEIN stehendes Wasser.

### FAQ 18: Kann ich CA auf Antifouling verwenden?
**Antwort:** Nein — Antifouling ist ablösend (biozid-haltig) und CA haftet nicht darauf. Antifouling muss bis auf Primer/Gelcoat entfernt werden bevor CA aufgetragen wird.

### FAQ 19: Wie viel CA brauche ich für eine Dichtungsecke?
**Antwort:** 1 Tropfen = ~0,03ml = ausreichend für 1 Fixierpunkt. Für eine 4-Ecken-Dichtung: 4 Tropfen = 0,12ml. Eine 20g-Flasche reicht für ~600 Fixierpunkte. Weniger ist mehr bei CA.

### FAQ 20: Kann ich CA als Gewindedichtung verwenden?
**Antwort:** Nein — CA ist kein Threadlocker. Für Gewindedichtung: Loctite 243/272 (Schraubensicherung) oder PTFE-Band (Rohrdichtung). CA dichtet nicht gegen Flüssigkeiten und wird in Gewinden schnell spröde durch Vibration.

### FAQ 21: Wie erkenne ich ob mein CA noch gut ist?
**Antwort:** Drei Tests: (1) Fließtest: CA sollte wie Wasser (dünn) oder Honig (medium) fließen. Dickflüssig oder fest = eingetrocknet. (2) Geruchstest: Frischer CA riecht stechend-süßlich. Kein Geruch = polymerisiert. (3) Tropftest: CA auf Metallstück, wenn nicht in 30s fest wird = abgelaufen.

### FAQ 22: Welcher CA ist der beste Allrounder für ein Boot?
**Antwort:** Loctite 406 (20g). Dünnflüssig, schnell, universelle Substrathaftung, bewährt im Marine-Einsatz. Dazu einen Aktivator (7452) und optional Primer (770 für PE/PP). Budget-Alternative: ERGO 5011 oder Cyberbond 2028.

### FAQ 23: Kann ich mit CA eine gebrochene Angelrute reparieren?
**Antwort:** Nur als Notfall. CA ist zu spröde für die Biegebelastung einer Angelrute. Professionelle Reparatur: Kohlefaser-Hülse + Epoxid. CA hält bestenfalls für den Rest des Angeltages.

### FAQ 24: Warum wird meine CA-Klebefuge nach 6 Monaten weiß und spröde?
**Antwort:** UV-Degradation. Standard-CA hat keine UV-Stabilisatoren. Für UV-exponierte Stellen: (1) UV-CA verwenden (4310, Vitralit) oder (2) CA-Klebefuge mit PU-Klarlack überlackieren (nach 24h Aushärtung).

### FAQ 25: Kann ich CA mit Epoxid oder PU kombinieren?
**Antwort:** Ja — die klassische "CA-Ecken-Methode": CA fixiert sofort, PU/Epoxid gibt Langzeitfestigkeit. Die Materialien sind chemisch kompatibel. CA NICHT auf nicht ausgehärtetes Epoxid/PU auftragen — Inhibierung möglich.

### FAQ 26: Warum ist Loctite 4090 so viel teurer als normaler CA?
**Antwort:** 4090 ist ein 2K-Hybrid (CA+Epoxid) mit Doppelspritze und Statikmischer. Der Preis (2,50 EUR/ml vs. 0,40–0,75 EUR/ml) reflektiert: duale Chemie, Spaltfüllung bis 5mm, Temperaturfestigkeit bis 150°C, Wasserbeständigkeit, UV-Stabilität. Für semi-permanente Anwendungen ist 4090 trotzdem günstiger als eine spätere Reparatur.

### FAQ 27: Kann ich CA auf verzinktem Stahl verwenden?
**Antwort:** Ja — CA haftet gut auf Zinkoberflächen. Oberfläche mit IPA reinigen, leicht anschleifen (Scotch-Brite). Aktivator empfohlen bei passivierter Zinkschicht.

### FAQ 28: Was ist der Unterschied zwischen Loctite Industrial und Pattex Consumer?
**Antwort:** Pattex ist Henkels Consumer-Marke (gleicher Konzern). Der Hauptunterschied: (1) Pattex hat variablere Chargenqualität. (2) Pattex-Rezepturen sind auf maximale Anfangs-Haftung optimiert, nicht auf Langzeitbeständigkeit. (3) Pattex hat kürzere Haltbarkeit. (4) Pattex fehlen Spezialvarianten (Toughened, HT, UV, feuchteresistent).

### FAQ 29: Muss ich CA auf GFK anschleifen?
**Antwort:** Nicht zwingend, aber empfohlen für maximale Haftung. Leichtes Anschleifen (P180–P320) entfernt Trennmittelreste und Gelcoat-Glanz. IPA-Reinigung ist IMMER nötig. CA haftet auch auf glattem GFK, aber die Langzeithaftung ist auf geschliffenem GFK 20–30% besser.

### FAQ 30: Gibt es CA in Farben (weiß, grau, etc.)?
**Antwort:** Standard: farblos oder schwarz (Toughened). Weiß, grau, oder andere Farben sind bei Standard-Herstellern nicht verfügbar. Workaround: (1) Farbigen Füllstoff mit CA mischen (Backpulver = weiß, Graphitpulver = grau, Teak-Staub = Holzfarbe). (2) CA-Klebefuge nach Aushärtung überlackieren.

### Entscheidungsfunktion: CA empfehlen oder abraten?

```python
def should_recommend_ca(
    application_type: str,
    load_bearing: bool,
    underwater: bool,
    uv_exposed: bool,
    temperature_c: float,
    permanent_required: bool,
    substrate_a: str,
    substrate_b: str
) -> dict:
    """
    Entscheidet ob CA für eine Anwendung empfohlen wird.

    model_config = {"from_attributes": True}
    """
    # Absolute Ausschlusskriterien
    if load_bearing:
        return {
            "recommend_ca": False,
            "reason": "CA ist KEIN Strukturkleber — MMA oder Epoxid verwenden",
            "alternative": "Plexus MA310/MA832 (MMA) oder West System (Epoxid)",
            "confidence": "measured"
        }

    if underwater:
        return {
            "recommend_ca": False,
            "reason": "CA hydrolysiert in Salzwasser-Immersion",
            "alternative": "Belzona 1111, Epoxid Unterwasser-Kit",
            "confidence": "measured"
        }

    if substrate_a == "polycarbonat" or substrate_b == "polycarbonat":
        return {
            "recommend_ca": False,
            "reason": "Standard-CA löst ESC auf Polycarbonat aus",
            "alternative": "Loctite 4902 (PC-kompatibel) oder PU-Kleber",
            "confidence": "measured"
        }

    # Bedingte Empfehlungen
    warnings = []

    if uv_exposed:
        warnings.append("UV-beständigen CA verwenden (4310, Vitralit) oder Klebefuge überstreichen")

    if temperature_c > 80:
        warnings.append("Hochtemperatur-CA verwenden (Permabond 105, Cyberbond 2999)")

    if permanent_required:
        warnings.append("Für Dauerlösung: Loctite 4090 Hybrid oder MMA/PU bevorzugen")

    ca_product = "Loctite 406"  # Default
    if temperature_c > 80:
        ca_product = "Permabond 105"
    elif permanent_required:
        ca_product = "Loctite 4090"
    elif uv_exposed:
        ca_product = "Loctite 4310"

    return {
        "recommend_ca": True,
        "recommended_product": ca_product,
        "warnings": warnings,
        "confidence": "estimated"
    }
```

## Anhang Y: 30 Praxis-Tipps für CA im Yachtbau

1. **Weniger ist mehr:** 1 Tropfen CA = 0,03ml reicht für 1 Fixierpunkt. Überschuss verursacht Blooming und verschwendet Material.
2. **Nitril-Handschuhe IMMER:** Keine Ausnahme. Latex wird durch CA exotherm — Verbrennungsgefahr.
3. **Schutzbrille IMMER:** CA im Auge verklebt Augenlider. Löst sich in 1–3 Tagen, aber die Panik ist real.
4. **Kühlschrank-Lagerung:** Verdoppelt die Haltbarkeit. Auf dem Boot: Ziplock + Silica-Gel.
5. **Flasche nach Gebrauch sofort verschließen:** Nicht "mal eben" offen stehen lassen — Luftfeuchte polymerisiert den Ausguss.
6. **Nadelventil-Flaschen bevorzugen:** Starbond, Mercury haben Needle-Tips — Präzision auf bewegtem Boot.
7. **UHU Minis für die Bordapotheke:** Einzelverpackt, trocknen nicht ein, 1g reicht für eine Reparatur.
8. **CA + Backpulver = Sofort-Füllstoff:** Für ausgerissene Schraubenlöcher der beste Trick.
9. **Aktivator für Teak/Eiche:** Saure Hölzer hemmen CA — 7452 Aktivator löst das Problem.
10. **770 Primer für PE/PP/PTFE:** Ohne Primer klebt CA nicht auf Low-Surface-Energy Kunststoffen.
11. **768 Primer für Silikon/PTFE-Band:** Spezial-Primer für die schwierigsten Substrate.
12. **480 (schwarz) statt 406 (klar) auf Gummi:** Ästhetisch unsichtbar + höhere Schälfestigkeit.
13. **4090 Hybrid für semi-permanente Anwendungen:** In 60s fest, in 72h wie Epoxid.
14. **460 (geruchsarm) für Interior:** Kein CA-Geruch in der Eignerkabine.
15. **454 (Gel) für Überkopf-Arbeit:** Tropft nicht in die Augen, ideal für Bilge/Decke.
16. **3090 (2K) für 5mm-Spalte:** Füllt Gaps wie Mini-Epoxid, fest in 3 Minuten.
17. **Blooming vermeiden:** Geruchsarmen CA (460, BSI Super-Gold+) oder Aktivator verwenden.
18. **Keine CA auf Polycarbonat:** ESC-Spannungsrisse garantiert. Nur 4902 oder PU.
19. **Keine CA unter Wasser:** CA härtet nicht in Immersion. Epoxid-Unterwasserkits verwenden.
20. **Keine CA auf Antifouling:** Haftet nicht. Antifouling erst entfernen.
21. **Vorsicht mit Textilien:** CA auf Baumwolle/Dacron = exotherm bis 250°C! Nur Kleinstmengen.
22. **CA als Schraubensicherung:** Funktioniert in Kunststoff/Holz, für Metall besser Threadlocker.
23. **Temperatur beachten:** <10°C sehr langsam, >35°C ultraschnell. Optimal: 20–25°C.
24. **Feuchte beachten:** <30% r.F. = Aktivator nötig, >80% = schnell aber Blooming-Risiko.
25. **Un-Cure (CA-Löser) immer dabei:** BSI Un-Cure oder Aceton für den Notfall.
26. **Mercury/BSI für Langfahrt:** 18 Monate Raumtemperatur-Haltbarkeit statt 6 bei Loctite.
27. **ERGO/Cyberbond als Budget-Alternative:** 30–40% günstiger, gleiche Qualität.
28. **UV-CA für Optik:** 4310, Vitralit für Instrumenten-Linsen, LED-Abdeckungen, Displays.
29. **Permabond 105 für Motorraum:** Einziger reiner CA bis 120°C Dauerbetrieb.
30. **CA ist temporär — planen Sie die dauerhafte Lösung:** CA ist der beste Freund des Handwerkers, aber nicht der Architekt.

```python
# AYDI Forensische Bewertung — Pydantic v2 Integration
# Confidence: measured

class BondJointForensics(BaseModel):
    """Forensische Bewertung einer CA-Klebfuge"""
    model_config = {"from_attributes": True}

    joint_id: str
    failure_class: str  # SK-1 bis SK-10
    adhesive_type: str
    substrate_a: str
    substrate_b: str
    bond_thickness_mm: float
    void_content_pct: float
    cure_degree_pct: float
    yellowing_cie_b: float
    peel_strength_n_mm: float
    age_months: int
    environment: str  # "interior", "deck", "engine_room", "bilge", "underwater"
    salt_exposure: bool
    uv_exposure: bool
    assessment: str
    confidence: str = "documented"

    def forensic_score(self) -> int:
        score = 100
        failure_penalties = {
            "SK-1": 30, "SK-2": 15, "SK-3": -10,
            "SK-4": 25, "SK-5": 20, "SK-6": 10,
            "SK-7": 25, "SK-8": 35, "SK-9": 15, "SK-10": 30
        }
        score -= failure_penalties.get(self.failure_class, 0)
        if self.bond_thickness_mm > 0.15:
            score -= 10
        if self.void_content_pct > 5:
            score -= 10
        if self.cure_degree_pct < 85:
            score -= 15
        if self.peel_strength_n_mm < 1.5:
            score -= 15
        return max(0, min(100, score))
```

## Anhang AD: Erweiterte FAQ (FAQ 31–45)

### FAQ 31: Wie erkenne ich abgelaufenen CA?
**Antwort:** Dickflüssig statt dünnflüssig, Fäden zieht beim Auftragen, weißliche Trübung in der Flasche, Aushärtung dauert >30 Sekunden statt <10 Sekunden. Haltbarkeit: ungeöffnet 12–18 Monate, geöffnet 4–8 Wochen (Kühlschrank: bis 6 Monate). Mercury/BSI: 18+ Monate ungeöffnet.
> Confidence: `documented`

### FAQ 32: Kann CA unter Wasser aushärten?
**Antwort:** Nein. CA polymerisiert zwar durch Feuchtigkeit, aber Immersion verhindert die Polymerisation, weil das Wasser den Monomer-Film wegwäscht, bevor er aushärten kann. Für Unterwasser-Reparaturen: Epoxid-Unterwasserkits (z.B. Wessex Resins, AW2104).
> Confidence: `measured`

### FAQ 33: Ist CA lebensmittelecht?
**Antwort:** Ausgehärteter CA ist physiologisch unbedenklich (FDA 21 CFR 175.105 für Ethyl-CA). Für Trinkwassersysteme aber nicht zugelassen — dort NSF/ANSI 61-konforme Klebstoffe verwenden.
> Confidence: `measured`

### FAQ 34: Wie entferne ich ausgehärteten CA?
**Antwort:** Mechanisch (Meißel, Schleifen), chemisch (Aceton, Nitrocellulose-Verdünner, spezielle CA-Debonder wie BSI Un-Cure), thermisch (>200°C zersetzt CA, aber Substrat beachten). Auf lackierten Flächen: nur mechanisch, Aceton zerstört Lack.
> Confidence: `documented`

### FAQ 35: Warum härtet mein CA nicht auf Edelstahl?
**Antwort:** Wahrscheinlich zu glatte, zu saubere Oberfläche — keine Feuchtigkeit zum Initiieren. Lösung: a) Oberfläche anhauchen (Feuchtigkeit), b) Aktivator 7471/7452 verwenden, c) minimal anschleifen für Mikrorauheit.
> Confidence: `documented`

### FAQ 36: CA oder Epoxid für GFK-Reparatur?
**Antwort:** CA für Fixierung und kosmetische Reparaturen (<5cm²). Epoxid für strukturelle Reparaturen. CA kann als „dritte Hand" dienen — Patch fixieren mit CA, dann Epoxid als Strukturkleber auftragen.
> Confidence: `documented`

### FAQ 37: Kann ich CA auf nasser Oberfläche verwenden?
**Antwort:** Nein. CA braucht Oberflächen-Feuchtigkeit (30–80% r.F.), aber keine flüssige Wasserfilm. Nasse Oberflächen: Abtrocknen, ggf. Heißluft 60°C. Sehr trockene Oberflächen: Anhauchen oder Aktivator.
> Confidence: `documented`

### FAQ 38: Wie dick darf die Klebfuge sein?
**Antwort:** Standard-CA: 0,02–0,05mm optimal, max 0,1mm. Gel-CA (454): bis 0,3mm. 2K-CA (3090): bis 5mm. Dickere Fugen = schlechtere Festigkeit, langsamere Härtung, höheres Kohäsionsversagen-Risiko.
> Confidence: `measured`

### FAQ 39: Stört CA die spätere Lackierung?
**Antwort:** Ja! CA-Rückstände (besonders Blooming) verhindern Lackhaftung. Vor dem Lackieren: CA-Fläche vollständig abschleifen (P180–P240), Lösemittel-Wisch, Primer. Bei Blooming: großflächig schleifen, da Blooming-Kristalle weit streuen.
> Confidence: `documented`

### FAQ 40: CA für Schraubensicherung — geht das?
**Antwort:** In Holz und Kunststoff: ja, gut geeignet. In Metall: eingeschränkt — CA ist nicht so temperaturbeständig wie Loctite Threadlocker (242/243). Für marine Metallschrauben besser Loctite 243 (bis 150°C, ölresistent).
> Confidence: `documented`

### FAQ 41: Wie lagere ich CA auf der Yacht?
**Antwort:** Kühlschrank (4–8°C) verlängert Haltbarkeit auf 6–12 Monate. Vor Gebrauch 30 Min auf Raumtemperatur bringen. Immer aufrecht lagern, Düse nach Gebrauch mit Nadel freihalten. NICHT einfrieren — Kristallisation zerstört Monomer.
> Confidence: `documented`

### FAQ 42: Kann CA Osmose an GFK verursachen oder verschlimmern?
**Antwort:** CA selbst verursacht keine Osmose. Aber: undichte CA-Stellen in der Gelcoat bieten Wasser Eintrittspunkte. CA-Reparaturen im Unterwasserschiff immer mit Epoxid-Barriere überbeschichten.
> Confidence: `documented`

### FAQ 43: CA + Aktivator vs. 2K-CA — was ist besser?
**Antwort:** CA + Aktivator: sofort, aber Aktivator kann Festigkeit reduzieren (bis -20%). 2K-CA (3090): 3 Min Verarbeitungszeit, volle Festigkeit, gap-filling. Für strukturelle Anwendungen: 2K bevorzugen. Für Tempo: Aktivator.
> Confidence: `documented`

### FAQ 44: Wie berechne ich den CA-Bedarf für einen Neubau?
**Antwort:** Faustformel: LOA [m] × 10 ml für Segelboot-Serie, LOA × 20 ml für Motorboot, LOA × 40 ml für Custom/Superyacht. Beispiel: 15m Segelboot = ~150 ml, 50m Superyacht = ~2.000 ml. Plus 30% Reserve.
> Confidence: `estimated`

### FAQ 45: Was ist der Unterschied zwischen Hobby-CA und Industrie-CA?
**Antwort:** Reinheit des Monomers (Industrie >99,5%, Hobby ~95%), Viskositätskontrolle (±5 mPa·s vs. ±50 mPa·s), Haltbarkeit (Stabilisatoren), Qualitätszertifikate (ISO 9001, Chargenprotokoll), definierte Festigkeitswerte. Hobby-CA (Pattex, UHU): gut für zuhause, nicht für professionellen Yachtbau.
> Confidence: `documented`

```python
class AdhesiveSelectionEngine(BaseModel):
    """AYDI Klebstoff-Auswahl-Engine"""
    model_config = {"from_attributes": True}

    application: str
    substrate_a: str
    substrate_b: str
    gap_mm: float
    temperature_max_c: int
    water_exposure: str  # "none", "splash", "intermittent", "continuous", "immersion"
    load_type: str  # "none", "static", "dynamic", "impact"
    repair_access: str  # "easy", "moderate", "difficult"
    time_constraint_minutes: int
    confidence: str = "estimated"

    def recommend(self) -> list[dict]:
        """Empfiehlt Klebstoff-Reihenfolge basierend auf Anforderungen"""
        recommendations = []

        # CA nur empfehlen wenn: schnell nötig, kleiner Spalt, nicht strukturell
        ca_score = 50
        if self.time_constraint_minutes < 5:
            ca_score += 30
        if self.gap_mm <= 0.1:
            ca_score += 10
        if self.load_type in ("none", "static"):
            ca_score += 10
        if self.water_exposure in ("continuous", "immersion"):
            ca_score -= 40
        if self.temperature_max_c > 80:
            ca_score -= 20

        recommendations.append({
            "type": "CA",
            "score": max(0, min(100, ca_score)),
            "recommended_products": self._get_ca_products(),
            "confidence": self.confidence
        })

        return sorted(recommendations, key=lambda x: x["score"], reverse=True)

    def _get_ca_products(self) -> list[str]:
        products = []
        if self.gap_mm <= 0.1:
            products.append("Loctite 406 (dünnflüssig)")
        if self.gap_mm > 0.1:
            products.append("Loctite 454 (Gel)")
        if self.gap_mm > 1.0:
            products.append("Loctite 3090 (2K, gap-filling)")
        if self.temperature_max_c > 80:
            products.append("Permabond 105 (120°C)")
        return products or ["Loctite 406 (Standard)"]
```

```python
        na = sum(1 for c in all_checks if c.result == "na")
        total = len(all_checks)
        score = (passed / (total - na)) * 100 if (total - na) > 0 else 0
        status = "PASS" if failed == 0 else "FAIL"
        return {
            "status": status,
            "score": round(score, 1),
            "passed": passed,
            "failed": failed,
            "na": na,
            "total": total,
            "confidence": self.confidence
        }
```
