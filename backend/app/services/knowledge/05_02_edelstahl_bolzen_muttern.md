## 1. Grundlagen: Bolzen vs. Schrauben — Terminologie

### 1.1 Definition nach DIN/ISO

| Begriff | Definition | Typisch im Yachtbau |
|---------|-----------|-------------------|
| **Schraube (Screw)** | Verbindungselement mit Kopf und Gewinde, wird in Gewindeloch oder Material geschraubt | Holzschrauben, Blechschrauben, Maschinenschrauben in Gewindebohrung |
| **Bolzen (Bolt)** | Verbindungselement mit Kopf und Gewinde, wird DURCH beide Teile gesteckt, Mutter auf Gegenseite | Deckbeschläge, Klampen, Winschen, Relingshalter |
| **Gewindebolzen (Stud Bolt)** | Bolzen OHNE Kopf, beidseitig Gewinde | Motorenbefestigung, Flanschverbindungen |
| **Stiftschraube (Stud)** | Ein Ende eingeschraubt, anderes Ende mit Mutter | Auspuffkrümmer, Seeventile |
| **Gewindestange (Threaded Rod)** | Durchgehend Gewinde, kein Kopf | Kielbolzen, Durchgangsverbindungen, Anker |

### 1.2 Die Goldene Regel im Yachtbau

> **„Wenn es unter Last steht und nicht geschweißt ist — Durchgangsbolzen."**
> — Konsens von CruisersForum, YBW, ThehullTruth

Diese Regel gilt besonders für:
- Alle Beschläge, die Rigg-Lasten tragen (Wanten, Stagen, Winschen)
- Alle Deck-Durchbrüche mit Beschlag (Klampen, Poller, Relingshalter)
- Alle sicherheitsrelevanten Verbindungen (Ruder, Kiel, Steuerung)
- Motorbefestigung (Motorlager, Getriebe)

**Ausnahme:** Rein dekorative Beschläge und leichtbelastete Innenausstattung dürfen mit Maschinenschrauben in Einpressmuttern oder Gewindeeinsätze befestigt werden.

### 1.3 Warum Durchgangsbolzen?

| Argument | Erklärung |
|----------|-----------|
| **Lastverteilung** | Kraft wird auf Schraubenkopf + Unterlegscheibe + Backing Plate (oben) UND Mutter + Unterlegscheibe + Backing Plate (unten) verteilt |
| **Inspizierbarkeit** | Beide Seiten sichtbar → Korrosion, Risse, Lockerung erkennbar |
| **Demontierbarkeit** | Mutter lösen → Bolzen rausziehen. Kein Gewindeschneiden nötig |
| **Redundanz** | Selbst wenn Vorspannung verloren geht → Bolzen hält mechanisch durch Kopf+Mutter |
| **Auszugskraft** | Nicht vom Basismaterial abhängig (wie bei Schrauben in GFK) |

| Beschlag | Typischer Bolzen-Ø | Material | Bemerkung |

### 2.5 Warum 304 (A2) NIEMALS für Bolzen im Außenbereich

**Forum-Konsens (CruisersForum, YBW, ThehullTruth):**

304 (V2A) fehlt Molybdän. Ohne Mo keine Beständigkeit gegen Chlorid-Ionen im Seewasser. Folgen:
1. **Lochfraß (Pitting)**: Beginnt innerhalb von 6–18 Monaten in Salzwasser
2. **Spaltkorrosion**: Unter Scheiben, in Gewinden — dort wo Bolzen am meisten beansprucht werden
3. **SCC**: Bei Vorspannung + Temperatur + Chlorid → katastrophaler Bruch OHNE Vorwarnung
4. **Galvanische Korrosion**: 304 ist weniger edel als 316 → in Mischverbindung wird 304 bevorzugt angegriffen

> „Jede Saison fischen wir mindestens zwei Boote mit gebrochenen 304er Klampenbolzen aus dem Hafen. Die Eigner wussten nicht mal, dass es 304 war — stand halt ‚Edelstahl' drauf."
> — Hafenmeister-Bericht, CruisersForum 2019

## 4. Bolzentypen im Yachtbau

### 4.1 Sechskantschrauben (Hex Bolts) — DIN 931 / DIN 933 / ISO 4014 / ISO 4017

**Der Standard-Bolzen für alle tragenden Verbindungen.**

| Norm | Beschreibung | Gewinde | Anwendung |
|------|-------------|---------|-----------|
| **DIN 931 / ISO 4014** | Sechskant, Teilgewinde | Gewinde endet vor Kopf | Standard: Beschläge, Flansche |
| **DIN 933 / ISO 4017** | Sechskant, Vollgewinde | Gewinde bis Kopf | Wenn gesamte Länge Gewindebereich |
| **DIN 960 / ISO 8765** | Sechskant, Feingewinde | Feiner Steigung | Vibrations-empfindlich, Präzision |
| **DIN 961 / ISO 8676** | Sechskant, Feingewinde, Vollgewinde | Fein, bis Kopf | Motorlager, Präzisionsanwendung |

**Teilgewinde vs. Vollgewinde — Die Entscheidung:**

| Kriterium | Teilgewinde (DIN 931) | Vollgewinde (DIN 933) |
|-----------|----------------------|----------------------|
| **Scherbelastung** | ✅ Schaft im Scherloch → voller Ø | ❌ Gewinde im Scherloch → reduzierter Ø |
| **Zugbelastung** | Gleich (Gewinde ist schwächste Stelle) | Gleich |
| **Korrosion** | ✅ Glatter Schaft = weniger Angriffsfläche | ❌ Gewinde = mehr Spaltkorrosion |
| **Flexibilität** | ✅ Passt auch wenn Materialstärke variiert | ✅ Mutter überall positionierbar |
| **Marine-Empfehlung** | ✅✅ BEVORZUGT für tragende Verbindungen | OK für leichte Beschläge |

### 4.2 Schlossschrauben (Carriage Bolts) — DIN 603 / ISO 8677

Runder Kopf mit Vierkantansatz — dreht sich nicht im Bohrloch.

| Merkmal | Detail |
|---------|--------|
| **Norm** | DIN 603 / ISO 8677 |
| **Kopf** | Flachrund mit Vierkantansatz |
| **Antrieb** | Keiner (Vierkant verhindert Mitdrehen) |
| **Marine-Einsatz** | Teakdeck-Befestigung, Innenausbau, Niedergang-Stufen |
| **Vorteil** | Glatter Kopf = keine Verletzungsgefahr, Diebstahlschutz |
| **Nachteil** | Schwer lösbar (muss von Mutternseite), Vierkant kann in GFK ausschlagen |
| **Verfügbar in A4** | Ja, aber eingeschränktes Sortiment |

### 4.3 Innensechskant-Schrauben (Socket Head Cap Screws) — DIN 912 / ISO 4762

| Merkmal | Detail |
|---------|--------|
| **Norm** | DIN 912 / ISO 4762 |
| **Kopf** | Zylindrisch mit Innensechskant (Allen) |
| **Marine-Einsatz** | Winschen, Genuaschienen, Hardware mit versenktem Kopf |
| **Vorteil** | Hohe Festigkeit, kleiner Kopf, versenkt möglich |

| **Praxistipp** | Torx-Version (ISO 14579) bevorzugen → weniger Cam-Out, besseres Lösen |

| **Norm** | ISO 7380 |

| **Vorteil** | Niedriger Kopf = wenig Stolpergefahr |

### 4.5 Senkkopfschrauben (Countersunk / Flathead) — DIN 7991 / ISO 10642

| Merkmal | Detail |
|---------|--------|
| **Norm** | DIN 7991 / ISO 10642 |

### 4.6 Gewindestifte (Set Screws / Grub Screws) — DIN 913–916 / ISO 4026–4029

| Merkmal | Detail |
|---------|--------|

## 5. Durchgangsbolzen — Das Goldene Prinzip

### 5.1 Aufbau einer korrekten Durchgangsverbindung

```
OBEN (Deckseite):
┌─────────────────────────────┐
│  Beschlag (Klampe, Winsch)  │
│  ┌───┐                     │
│  │ ○ │ ← Bolzenloch im Beschlag │
│  └───┘                     │
└─────────────────────────────┘
│  Dichtmittel (Sikaflex/Butyl) │
┌─────────────────────────────┐
│  Gelcoat                    │
│  Deck-Laminat (GFK)        │
│  Kernmaterial (Balsa/Schaum)│  ← IMMER auffüllen bei Bolzen!
│  Inneres Laminat            │
└─────────────────────────────┘
│  Dichtmittel (optional)    │
┌─────────────────────────────┐
│  Backing Plate              │
│  (Edelstahl oder Aluminium) │
└─────────────────────────────┘
│  Große Unterlegscheibe (A4) │
│  Mutter (A4-80, Nyloc)     │
```

### 5.2 Dimensionierungsregeln

| Parameter | Regel | Erklärung |
|-----------|-------|-----------|
| **Bolzen-Ø** | Gemäß Beschlag-Spezifikation | Beschlaghersteller gibt vor |
| **Bohrloch-Ø** | Bolzen-Ø + 0,5–1,0 mm | Passung, nicht Presspassung |
| **Laminat-Mindestdicke** | 2 × Bolzen-Ø | Darunter: Backing Plate vergrößern |
| **Bolzenlänge** | Materialstärke + Scheibe + Mutter + 2–3 Gewindegänge | Gewinde muss komplett durch Mutter |
| **Backing-Plate-Dicke** | ≥ 0,5 × Bolzen-Ø (min 3mm) | A4 Edelstahl oder 5083 Aluminium |
| **Backing-Plate-Größe** | Mindestens Beschlag-Grundfläche | Lastverteilung über großen Bereich |
| **Randabstand** | ≥ 2 × Bolzen-Ø vom Laminatrand | Sonst Ausbruchgefahr |
| **Lochabstand** | ≥ 3 × Bolzen-Ø zwischen Bolzen | Sonst Laminat-Schwächung |

### 5.3 Kernbereich-Behandlung bei Sandwich-Decks

**DAS häufigste Problem bei Deck-Durchbrüchen auf modernen Yachten:**

Sandwich-Decks (GFK-Balsa-GFK oder GFK-Schaum-GFK) haben einen Kernbereich, der Wasser aufnehmen kann. JEDER Bolzen durch einen Sandwich-Kern muss den Kern im Bohrlochbereich ersetzen:

**Verfahren (nach Gougeon Brothers / West System):**

1. Bohrloch mit Überbmaß bohren (3–4× Bolzen-Ø)
2. Kernmaterial im Bohrlochbereich entfernen (Dremel mit Kugelfräser)
3. Hohlraum mit Epoxid + Microballons oder Colloidal Silica füllen
4. Aushärten lassen (24h bei 20°C oder 48h bei 15°C)
5. Bolzenloch auf Soll-Ø bohren
6. Dichtmittel aufbringen (Sikaflex 291 oder Butylband)
7. Bolzen mit Dichtmittel einsetzen

**Zeitaufwand pro Bolzenloch:** 30–45 Minuten (plus Aushärtezeit)

> „Wer ein Loch durch Balsa-Sandwich bohrt ohne den Kern zu versiegeln, baut eine Wasserleitung in sein Deck. Innerhalb von 2–3 Jahren hast du weichen Kern und knirschende Beschläge."
> — Steve D'Antonio, stevedmarineconsulting.com

### 5.4 Fehlerbeispiele: Falsch montierte Durchgangsbolzen

| Fehler | Folge | Lösung |
|--------|-------|--------|
| **Keine Backing Plate** | GFK wird durchgezogen unter Last | Immer Backing Plate nachrüsten |
| **Zu kleine Scheiben** | Punktlast auf Mutter → Laminat bricht | Große Scheiben (≥ 2,5× Bolzen-Ø) |
| **Kern nicht versiegelt** | Wasserintrusion, Kernfäule | Demontieren, Kern ersetzen, neu setzen |
| **Kein Dichtmittel** | Deck-Leck bei jedem Regen | Demontieren, dichten, neu montieren |
| **Bolzen zu kurz** | Weniger als 2 Gewindegänge in Mutter | Längeren Bolzen verwenden |
| **Bolzen zu lang** | Gewindestange steht weit über → Verletzung | Absägen, entgraten, oder kürzeren Bolzen |
| **304 statt 316** | Korrosion, Bruch | Komplett durch A4-80 ersetzen |
| **Mischung 304/316** | Galvanische Korrosion 304→316 | ALLES in 316 (A4) |

## 10. Kielbolzen — Die kritischsten Bolzen am Boot

### 10.1 Warum Kielbolzen lebenswichtig sind

Der Kiel ist die schwerste Einzelkomponente einer Segelyacht. Er hängt NUR an den Kielbolzen. Wenn Kielbolzen versagen, sinkt das Boot in Minuten — ohne Vorwarnung, ohne Rettungsmöglichkeit.

**Typische Kielgewichte:**

| Bootslänge | Kielgewicht (ungefähr) | Typischer Bolzen-Ø | Anzahl |
|-----------|----------------------|-------------------|--------|
| 8m (26ft) | 800–1200 kg | M16–M20 | 4–6 |
| 10m (33ft) | 1500–2500 kg | M20–M24 | 6–8 |
| 12m (40ft) | 2500–4500 kg | M24–M30 | 8–12 |
| 14m (46ft) | 4000–7000 kg | M30–M36 | 10–14 |
| 16m (52ft) | 6000–10000 kg | M36–M42 | 12–16 |

### 10.2 Kielbolzen-Materialien

| Material | Verwendung | Vorteile | Nachteile |
|----------|-----------|----------|-----------|
| **A4-80 (316L)** | Standard moderner GFK-Yachten | Verfügbar, bekannt, inspizierbar | SCC-Risiko bei hoher Vorspannung |
| **Duplex 2205** | Premium-Yachten, Refit | SCC-resistent, stärker, höhere PREN | Teuer, schwer beschaffbar |
| **Monel 400** | Traditionelle/klassische Yachten | Exzellente Seewasserbeständigkeit, kein SCC | Niedrigere Festigkeit, sehr teuer |
| **Monel K-500** | Premium-Refit | Höhere Festigkeit als 400, kein SCC | Extrem teuer, Spezialanfertigung |
| **Silicon Bronze** | Holz-Yachten, traditionell | Holzverträglich, kein SCC | Niedrige Festigkeit, nicht für GFK |
| **Titan Grade 5** | Racing, Mega-Yachten | Leicht, extrem korrosionsbeständig | Astronomisch teuer, Spezialwerkzeug |

### 10.3 SCC-Risiko bei Kielbolzen (DAS Schreckensszenario)

Spannungsrisskorrosion (Stress Corrosion Cracking) bei Edelstahl-Kielbolzen:

**Bedingungen:**
1. Zugspannung (Vorspannung der Kielbolzen — IMMER vorhanden)
2. Chlorid-Ionen (Seewasser — IMMER vorhanden)
3. Temperatur >50°C (Bilge im Sommer, insbesondere Mittelmeer/Tropen)

**Verlauf:** Risse beginnen unsichtbar im Inneren des Bolzens. Kein äußerer Rost, keine Verformung, keine Warnung. Bolzen bricht plötzlich unter Last.

**Berühmte Fälle:**
- Segelmagazin Deutschland, 2012: Systematische Untersuchung von Kielbolzen verschiedener Serienyachten. Ergebnis: 30% der >15 Jahre alten 316-Kielbolzen zeigten SCC-Anfänge.
- Practical Sailor (USA), 2015: Empfehlung, 316-Kielbolzen spätestens nach 20 Jahren prophylaktisch zu ersetzen, unabhängig vom Zustand.
- CruisersForum, Thread "Keel bolt inspection", 40+ Seiten: Dutzende Berichte von SCC bei 316-Kielbolzen in Tropeneinsatz.

### 10.4 Kielbolzen-Inspektion

| Methode | Was wird geprüft | Intervall | Kosten |
|---------|-----------------|-----------|--------|
| **Visuelle Inspektion** | Rost, Verfärbung, Feuchtigkeit an Muttern | Jährlich (Slip) | €0 (Eigenleistung) |
| **Drehmoment-Prüfung** | Vorspannung noch vorhanden | Alle 2 Jahre | €0 (Eigenleistung) |
| **Ultraschall (UT)** | Querschnittsreduktion, Risse | Alle 5 Jahre oder bei Verdacht | €200–500 pro Prüfung |
| **Magnetpulverprüfung** | Oberflächenrisse (nur ferrit. Stahl) | Bei Verdacht | €150–300 |
| **Röntgen** | Innere Risse, Hohlräume | Bei schwerem Verdacht | €500–1500 |
| **Herausziehen + Prüfen** | Gesamtzustand | Alle 15–20 Jahre | €1000–5000+ (Kiel abnehmen) |

### 10.5 Kielbolzen-Austausch: Anleitung

**Phase 1: Vorbereitung**
1. Boot auf stabilem Kiel-Gestell / Kielwagen
2. Kiel ZUSÄTZLICH abstützen (Hydraulikstempel)
3. Alle Muttern markieren (Strich über Mutter+Unterlegscheibe)
4. Drehmoment dokumentieren (IST-Wert jeder Mutter)

**Phase 2: Demontage**
1. Muttern lösen: KREUZWEISE, jeweils 1/4 Umdrehung
2. Penetrieröl verwenden wenn festsitzend (Kroil, PB Blaster)
3. Alte Bolzen herausdrücken (NIEMALS hämmern! → Laminat-Schaden)
4. Zustand dokumentieren: Fotos, Durchmesser messen, Risse notieren

**Phase 3: Neue Bolzen**
1. Material: Duplex 2205 oder Monel bevorzugt (Upgrade von 316)
2. Oberfläche: geschliffen, keine Werkzeugspuren (Rissausgangspunkte!)
3. Gewinde: gerollt (nicht geschnitten) — höhere Ermüdungsfestigkeit
4. Länge: exakt vermessen, 3–4 Gewindegänge über Mutter

**Phase 4: Montage**
1. Bolzenlöcher reinigen, trocknen
2. Bolzen mit Tef-Gel oder Duralac auf gesamter Länge beschichten
3. Große Unterlegscheiben (3× Bolzen-Ø) + Backing Plate
4. Muttern KREUZWEISE anziehen in 3–4 Schritten
5. Drehmoment gemäß Hersteller-Spezifikation
6. Nach 24h: Drehmoment nachprüfen
7. Nach erster Ausfahrt: Drehmoment nachprüfen

### 11.2 Mutter-Höhe und Gewindeeingriff

**Warum Mutter-Höhe wichtig ist:**

Die effektive Gewindehöhe bestimmt, wie viel Last die Verbindung tragen kann. Bei Edelstahl-Bolzen mit Edelstahl-Mutter wird die volle Festigkeit bei einer Mutterhöhe von 0,8d (d = Bolzen-Ø) erreicht.

| Mutterhöhe | Gewindeeingriff | Tragfähigkeit | Empfehlung |
|-----------|----------------|--------------|-----------|
| 0,5d (DIN 439) | ~62% | Reduziert | Nur als Konter-Mutter |
| 0,8d (DIN 934) | ~100% | Voll | Standard-Anwendung |
| 1,0d | >100% | Überdimensioniert | Sicherheitskritisch |
| 1,5d (DIN 6330) | >100% | Deutlich überdimensioniert | Kielbolzen, Schwerlasten |

### 12.1 Nyloc-Muttern (Selbstsichernde Muttern mit Kunststoffeinsatz)

> „Nyloc-Muttern sind Einmal-Teile. Punkt. Wer sie wiederverwendet, spart 30 Cent und riskiert seinen Beschlag."

ABER: Die Klemmkraft nimmt ab, und es gibt KEINE Garantie. Auf sicherheitsrelevanten Verbindungen (Rigging, Kiel, Ruder): IMMER neue Nyloc verwenden!

### 14.3 Warum Federringe (DIN 127) NICHT funktionieren

Der Junker-Vibrations-Test (1969, seitdem vielfach bestätigt) hat bewiesen:

> Federringe (Grower/Spring Washers) bieten KEINE messbare Sicherung gegen Vibrationslockerung. Sie platten bei der ersten Belastung zusammen und wirken dann nur noch als flache Unterlegscheibe.

**Was stattdessen verwenden:**

1. **Nyloc-Mutter** — Für 90% aller Marine-Anwendungen die richtige Wahl
2. **Nord-Lock Keilsicherung** — Für hochbelastete Verbindungen
3. **Kronenmutter + Splint** — Für sicherheitskritische Verbindungen
4. **Loctite 243 (mittelfest)** — Für Gewindestifte und nicht-demontierbaren Verbindungen

## 16. Backing Plates — Lastverteilung auf GFK

### 16.1 Warum Backing Plates unverzichtbar sind

GFK-Laminat hat eine zulässige Flächenpressung von nur 80–180 MPa (je nach Laminataufbau). Eine M10-Mutter mit Standard-Scheibe erzeugt bei 40 Nm Anzugsmoment eine Flächenpressung von >200 MPa → Laminat wird komprimiert, Verbindung lockert sich.

Backing Plate verteilt die Last auf eine viel größere Fläche.

### 16.2 Dimensionierung

| Beschlag | Backing Plate Material | Dicke (mm) | Mindestgröße |
|----------|----------------------|-----------|-------------|
| **Klampe 150mm** | A4 Edelstahl | 3–4 | 80 × 30 mm |
| **Klampe 200mm** | A4 Edelstahl | 5 | 100 × 40 mm |
| **Klampe 250mm+** | A4 Edelstahl oder Alu 5083 | 6–8 | 120 × 50 mm |

| **Wantenbeschlag** | A4 Edelstahl | 8–12 | Überlappend mit Beschlag |
| **Relingshalter** | A4 Edelstahl | 4–5 | Rechteckig, 80 × 50mm |

### 16.3 Material-Alternativen

| Material | Vorteil | Nachteil | Preis |
|----------|---------|----------|-------|
| **A4 Edelstahl** | Stark, korrosionsbeständig | Schwer, teuer | ★★★★ |
| **Alu 5083 (marine)** | Leichter, gute Korrosionsbeständigkeit | Galvanisches Potential zu SS-Bolzen! | ★★★ |
| **Alu 6082-T6** | Leicht, gut bearbeitbar | Weniger seewasserbeständig als 5083 | ★★ |
| **G10/FR4 Glasfaser** | Isolierend, kein galv. Problem | Niedrigere Festigkeit | ★★ |

### 16.4 Isolation bei Mischmetall-Verbindungen

Wenn A4-Bolzen durch Aluminium-Beschlag oder Aluminium-Backing-Plate gehen:

**Problem:** Edelstahl (edel) + Aluminium (unedel) + Elektrolyt (Seewasser) = galvanische Korrosion des Aluminiums

**Lösung:** Vollständige Isolation:
1. Kunststoff-Buchse im Bohrloch (PTFE oder Nylon)
2. Isolierscheiben unter Kopf und unter Mutter
3. Duralac (Tef-Gel) auf gesamter Kontaktfläche
4. Dichtmittel an Deckdurchführung (Sikaflex) = Elektrolyt-Barriere

## 19. Galling (Kaltverschweißung) bei Bolzen

### 19.1 Mechanismus

Galling entsteht, wenn zwei Edelstahl-Oberflächen unter hohem Druck und Reibung aufeinander gleiten (Anziehen einer Mutter auf Bolzen):

1. Chromoxid-Passivschicht wird lokal abgerieben
2. Blanke Metalloberflächen kommen in Kontakt
3. Hoher Druck + Wärme → Mikroverschweißung (Kaltverschweißung)
4. Weiteres Drehen reißt verschweißte Stellen → MEHR blankes Metall
5. Eskalation → Verbindung blockiert komplett oder Bolzen bricht

### 19.2 Galling-Risikofaktoren

| Faktor | Hohes Risiko | Niedriges Risiko |
|--------|-------------|-----------------|
| **Material** | Gleiche Legierung (SS auf SS) | Unterschiedliche Härte |
| **Oberflächengüte** | Rau, bearbeitet | Poliert, gerollt |
| **Geschwindigkeit** | Schnelles Anziehen (Akkuschrauber!) | Langsames Handanziehen |
| **Schmierung** | Trocken | Tef-Gel, Anti-Seize |
| **Temperatur** | Warm (>30°C) | Kalt |
| **Gewinde-Passform** | Eng (Toleranzklasse 6H/6g) | Locker (8H/8g) |
| **Wiederholtes Lösen/Anziehen** | Steigendes Risiko pro Zyklus | Erster Zyklus am sichersten |

### 19.3 Prävention

1. **IMMER Anti-Seize verwenden** — Tef-Gel, Duralac, oder Nickel-Anti-Seize
2. **Langsam anziehen** — Hand oder langsamer Drehmomentschlüssel, NIEMALS Schlagschrauber
3. **Unterschiedliche Härte** — Mutter leicht härter als Bolzen (A4-80 Bolzen + A4-80 Mutter ist OK, weil Mutter kaltverformt bei Herstellung)
4. **Gerollte Gewinde bevorzugen** — Glattere Oberfläche als geschnittene Gewinde
5. **Bolzen NICHT wiederverwendbar** — Wenn einmal gegalled: Bolzen + Mutter ERSETZEN
6. **Bei Festsitzen: STOPP** — NICHT weiter drehen! Eindringöl, warten, vorsichtig lösen. Sonst Bruch.

### 19.4 Rettungsmaßnahme bei festgefressenem Bolzen

| Schritt | Aktion | Werkzeug |
|---------|--------|----------|
| 1 | STOPP — nicht weiter drehen | — |
| 2 | Eindringöl reichlich aufsprühen | Kroil, PB Blaster, WD-40 Specialist |
| 3 | 15–30 Minuten einwirken lassen | Geduld |
| 4 | Vorsichtig IN LÖSERICHTUNG versuchen | Ringschlüssel, NICHT Maulschlüssel |
| 5 | Wenn immer noch fest: Kälte + Wärme | Eisspray auf Mutter, Lötlampe auf Bolzen (nicht GFK!) |
| 6 | Wenn nichts hilft: Mutter spalten | Mutternsprenger (Nut Splitter) |
| 7 | Letzter Ausweg: Abbohren | HSS-Bohrer, Ausdreher (Screw Extractor) |

### 20.3 Die Eichen-Holz-Falle

**WARNUNG:** Edelstahl in Eichenholz ist eine GEFÄHRLICHE Kombination!

Eiche enthält Tannine (Gerbsäure), die in Kombination mit Feuchtigkeit eine saure Umgebung schaffen. Diese Säure greift die Passivschicht des Edelstahls an → beschleunigte Korrosion, SCC-Risiko.

**Lösung für Eichenholz:**
- Silicon Bronze verwenden (CDA 655) — traditionell und bewährt
- Wenn SS notwendig: Duplex 2205 oder höher
- Bolzenloch mit Epoxid versiegeln → Holzsäure-Barriere
- Regelmäßige Inspektion (jährlich)

### 34.4 Toggle-Gelenke: Warum sie IMMER nötig sind

Ein Toggle-Gelenk am Wantenfuß erlaubt seitliches Kippen des Wants ohne Biegemoment auf den Terminal-Bolzen.

**Ohne Toggle:** Wantendraht + Terminal-Bolzen wird bei seitlicher Schiffsbewegung auf Biegung belastet → Ermüdungsbruch nach 5–15 Jahren (je nach Segelrevier).

**Mit Toggle:** Bolzen nimmt NUR Scherbelastung auf → Lebensdauer 3–5× länger.

> „Ein 6mm Clevis Pin ohne Toggle am Wantenfuß bricht statistisch nach 8.000–12.000 Seemeilen im Hochseebereich. Mit Toggle: 40.000+ Seemeilen."
> — Rigging-Statistik, Sail Magazine

## 35. Inspektion und Wartung von Bolzenverbindungen

### 35.1 Inspektionsintervalle

| Verbindungstyp | Sichtprüfung | Drehmoment-Check | Professionelle Prüfung |
|---------------|-------------|-----------------|----------------------|
| **Kielbolzen** | Jährlich (Slip) | Alle 2 Jahre | UT alle 5 Jahre (>10 J.) |
| **Wantenbeschläge** | Jährlich | Alle 2 Jahre | UT alle 5 Jahre |
| **Deckbeschläge (Klampen, Winschen)** | Jährlich | Alle 3 Jahre | Bei Verdacht |
| **Motorlager** | Jährlich | Jährlich | Bei Vibrations-Änderung |
| **Seeventil-Flansche** | Jährlich (Slip) | Alle 2 Jahre | Bei Leckage |
| **Rigging-Bolzen (Clevis, Toggle)** | Vor jeder Saison | — | Alle 5 Jahre |
| **Innenausbau** | Alle 5 Jahre | — | — |

### 35.2 Visuelle Inspektions-Checkliste

| Prüfpunkt | Was suchen | Bewertung |
|-----------|-----------|----------|
| **Rostspuren** | Braune Verfärbung um Bolzen | ⚠️ Sofort untersuchen |
| **Risse** | Haarrisse in Bolzenkopf oder Mutter | ❌ Sofort ersetzen |
| **Verformung** | Bolzenkopf oder Mutter oval/verformt | ❌ Sofort ersetzen |
| **Lockerung** | Mutter verdreht, Markierung verschoben | ⚠️ Nachziehen + Ursache suchen |
| **Wasseraustritt** | Feuchtigkeit um Deckdurchbruch | ⚠️ Abdichten |
| **Elektrolytische Korrosion** | Weißes Pulver (Alu), grüne Patina (Bronze) | ⚠️ Isolation prüfen |
| **Ermüdungsanzeichen** | Riefen, Verfärbung im Übergangsradius | ❌ Fachmännisch prüfen |

### 35.3 Drehmoment-Nachkontrolle

**Verfahren:**
1. Markierung auf Mutter + Unterlegscheibe + Grundmaterial (Edding)
2. Drehmomentschlüssel auf 80% des Soll-Wertes einstellen
3. LANGSAM anziehen → Mutter darf sich NICHT bewegen
4. Wenn Mutter sich bewegt: auf 100% Soll-Drehmoment nachziehen
5. Wenn Mutter sich bei <50% des Soll-Drehmoments bewegt: DEMONTIEREN + untersuchen

### 35.4 Austauschkriterien

| Zustand | Maßnahme |
|---------|----------|
| Korrosionsnarben >0,5mm Tiefe | Ersetzen |
| Gewindebeschädigung (deformiert, aufgeraut) | Ersetzen |
| Querschnittsreduktion >5% (gemessen) | Ersetzen |
| Sichtbare Risse (auch Haarrisse) | SOFORT ersetzen |
| Nyloc-Mutter bereits demontiert gewesen | Neue Nyloc-Mutter |
| Bolzen >15 Jahre in Seewasser (316) | Prüfung empfohlen |
| Bolzen >20 Jahre in Seewasser (316) | Prophylaktischer Austausch empfehlen |

### 41.3 Demontagetechniken für festsitzende Bolzen

#### 41.3.1 Methode 1: Thermische Dehnung (Hitze-Kälte-Verfahren)

**Anwendung:** Galling, Rostung, Kaltverschweißung ohne SCC-Verdacht.

**Schritt-für-Schritt:**
1. Eindringöl (Kroil, PB Blaster) 30 Min einwirken lassen
2. Heißluftfön oder Heat Gun auf 80–100°C erhitzen (nicht über 120°C auf Edelstahl)
3. 3–5 Min halten, dann Bolzen sanft (nicht ruckweise) drehen
4. Mit Drehmomentschlüssel oder Rohrzange arbeiten, NICHT Schlag-Werkzeug
5. Bolzen um 1/8 Umdrehung drehen, erneut wärmen, nächste 1/8 Umdrehung
6. Mutter wird dabei von außen mit Rohrzange gehalten
7. Bei hartnäckigen Fällen: Zyklisch wärmen (5 Min heiß, 2 Min Pausen) über 20–30 Min
8. Nach dem Lösen: Bolzen und Mutter sofort mit Tef-Gel neu montieren

**Warnung:** GFK-Laminat verträgt >120°C nicht. Ausstrahlung vom Bolzen mit feuchtem Tuch begrenzen.

#### 41.3.2 Methode 2: Chemisches Eindringen (Anti-Galling-Öle)

**Beste Öle nach Wirksamkeit:**
- **Kroil** (USA, Premium, €30–40/Liter): dünnflüssig, kapillare Eindringung in 10–15 Min
- **PB Blaster** (€15–20/Liter): guter Kompromiss, schnell wirkend
- **WD-40 Specialist** (€8–12): ausreichend für leichte Fälle, aber langsamer (45+ Min)

**Anwendungsprotokoll:**
1. Bolzen mit Bürste/Tuch grob reinigen
2. Öl großzügig auftragen, ringsum Mutter/Kopf durchdränken
3. Wartezeit:
   - Kroil: 10–15 Min
   - PB Blaster: 20–30 Min
   - WD-40: 60+ Min
4. Bolzen sanft hin-und-her bewegen (nicht drehen), nur 5–10° Winkel
5. Wartezeit wiederholen
6. Mit Rohrzange versuchen zu drehen — nicht mit Kraft erzwingen
7. Nach Erfolg: mit Lösungsmittel auswaschen, trocknen, sofort Tef-Gel neu auftragen

#### 41.3.3 Methode 3: Mechanische Sprengung (Mutternsprenger)

**Anwendung:** Wenn thermisch und chemisch nicht funktioniert. Zerstört Mutter, bewahrt Bolzen.

**Schritte mit Mutternsprenger (z.B. Gedore, Hazet):**
1. Mutternsprenger korrekt auf Mutter aufschieben (muss 100% sitzen)
2. Spannschraube langsam (!) anziehen — nicht schnell ruckweise pressen
3. Bei 2–3 Umdrehungen sollte Mutter sichtbar reißen (hörbarer Knacks)
4. Spannschraube vollständig losdrehen, Mutter entfernen
5. Bolzen mit Kroil nachbehandeln, dann herausziehen
6. Bolzen reinigen und inspizieren (Bruchkanten, Verformung?)
7. Neue A4-80 Mutter montieren mit Tef-Gel

**Warnung:** Zu schnelles Spannen kann Bolzen verbiegen oder brechen → langsam justieren!

#### 41.3.4 Methode 4: Ausbohren und Ausdreher

**Anwendung:** Bolzen ist bereits gebrochen oder kann nicht mehr gedreht werden.

**Ablauf:**
1. Links-Bohrer (HSS, <Bolzendurchmesser) mittig ansetzen
2. Mit moderater Drehzahl (150–300 U/min bei M8+) bohren
3. Links-Bohrer dreht Bolzen oft selbst heraus — Glücksfall
4. Falls nicht: Grabit- oder SpeedOut-Ausdreher verwenden:
   - Ausdreher-Bohrer durch Bolzenrest bohren
   - Ausdreher-Spitze einschrauben, dann dreht Bolzen-Bruchstück heraus
5. Loch nachbohren (nächste Standard-Größe) und Gewindeeinsatz (Helicoil) einsetzen
6. Mit neuem Bolzen montieren

#### 41.3.5 Methode 5: Hochfrequenz-Ultraschall (professionell)

**Anwendung:** Extreme Fälle bei großen Industrie-Bolzen (z.B. Kielbolzen).

**Prinzip:** Ultraschall-Vibrationen lockern Rost und Galling mechanisch auf.

**Verfügbarkeit:** Spezialwerkstätten, nicht DIY. Kosten: €100–300 pro Bolzen. Wirksamkeit: 85–95%.

### 41.4 Ersatzteil-Identifikation — Unbekannte Bolzen bestimmen

| Merkmal | Prüfmethode | Bedeutung |
|---------|-----------|-----------|
| **Durchmesser** | Schieblehre oder Gewindeprüfer | M6, M8, M10 etc. |
| **Gewindetyp** | Gewindelehre (metrisch vs UNC vs BSP) | ISO metrisch, UNC (USA), BSP (Rohre) |
| **Gewinde-Steigung** | Steigungsmesser oder Vergleichsmuster | Grobgewinde (0,75–1,5mm) vs Feingewinde (0,5–1,0mm) |
| **Kopfform** | Sichtprüfung | Sechskant (DIN 931/933), Zylinderkopf (ISO 4762), Flachkopf, Hutbolzen |
| **Material-Farbe** | Oberflächen-Inspektion | Silbergrau = wahrsch. Edelstahl; Gold-Schimmer = Kupfer-Legierung; Matt = Stahl |
| **Magnettest** | Magnet an Oberfläche | Nicht magnetisch = 316/304; schwach magnetisch = 304 (nach Kaltverformung); Stark magnetisch = Stahl (FALSCH für Yacht!) |
| **Gewicht-Test** | 2–3 identische Bolzen wiegen | Monel ist ~schwerer als SS. Titanium leichter. |
| **Hersteller-Stempel** | Kopf-Markierung lesen | z.B. „A4-80" (Edelstahl), „8.8" (Stahl), „Bumax" (Marke) |
| **Bruchprobe (Letztes Mittel)** | Überlast bis Bruch, Bruchfläche analysieren | Glatte kristalline Bruchfläche = Sprödbruch (SCC). Verformte/silberglänzende = duktiler Bruch (normal) |

**Entscheidungsmatrix nach Identifikation:**
- Unbekannter Bolzen im Rigging, Kiel, Steuerung? → Ersetzen durch dokumentierte A4-80/DIN 931
- Unbekannter Bolzen in Innenausbau? → Wenn nicht magnetisch + sichtbar in Ordnung → beobachten
- Verdacht auf Stahl-Bolzen (magnetisch)? → SOFORT ersetzen, kann SCC bekommen und brechen
- Monel-Bolzen gefunden? → BEWAHREN, nicht ersetzen; Monel ist kostbar, korrosionsresistent, wiederverkäuflich

## 42. FAQ — 20 häufigste Fragen

### FAQ 1: Muss ich wirklich Durchgangsbolzen verwenden? Blechschrauben halten doch auch.

**Ja. Durchgangsbolzen sind PFLICHT für alle lastragenden Beschläge.** Blechschrauben haben in GFK nur 30–50% der Auszugskraft eines Durchgangsbolzens. Bei dynamischer Last (Wellen, Wind, Rigg) lockern sich Blechschrauben und reißen aus. Durchgangsbolzen mit Backing Plate verteilen die Last auf beiden Seiten des Laminats und können sich nicht herausziehen.

### FAQ 2: A4-70 oder A4-80 — welche Festigkeitsklasse brauche ich?

**A4-80 ist der Marine-Standard.** A4-70 ist für leicht belastete Innenausbau-Verbindungen akzeptabel. Für alles, was unter Last steht (Beschläge, Rigging, Kiel, Motor): A4-80. Kostenunterschied: ca. 20–30% mehr für A4-80 — bei den Kosten einer Yacht-Reparatur vernachlässigbar.

### FAQ 3: Kann ich Nyloc-Muttern wiederverwenden?

**Offiziell: Nein. Nyloc-Muttern sind Einmal-Teile.** In der Praxis: Wenn der Nylon-Ring noch spürbaren Widerstand beim Aufschrauben bietet (2–3 Umdrehungen vor Anlage), wird die Mutter oft wiederverwendet. Auf sicherheitsrelevanten Verbindungen (Rigging, Kiel, Ruder, Steuerung): IMMER neue Nyloc verwenden. Kosten: €0,15–0,60 pro Mutter.

### FAQ 4: Brauche ich Anti-Seize auf Edelstahl-Bolzen?

**Ja, IMMER.** Edelstahl auf Edelstahl ohne Schmierung = Galling (Kaltverschweißung) ist nahezu garantiert. Tef-Gel ist der Marine-Standard. Eine 10g-Tube (€12–18) reicht für 50–100 Verbindungen.

### FAQ 5: Was ist der Unterschied zwischen DIN 931 und DIN 933?

**DIN 931 = Teilgewinde** (glatter Schaft unter Kopf). **DIN 933 = Vollgewinde** (Gewinde bis Kopf). Für tragende Verbindungen DIN 931 bevorzugen: der glatte Schaft hat den vollen Querschnitt im Scherloch.

### FAQ 6: Wie erkenne ich, ob mein Bolzen 304 oder 316 ist?

**Schwierig ohne Labor-Test.** Magnet-Test hilft bedingt: 304 ist nach Kaltverformung leicht magnetisch, 316 weniger. Sicherste Methode: Beim Kauf auf Materialzertifikat (3.1) bestehen. Für existierende Bolzen: XRF-Handgerät (Röntgenfluoreszenz) beim Materialprüfer.

### FAQ 7: Kielbolzen: 316 oder Duplex?

**Für Neubau und Refit: Duplex 2205 empfohlen.** 316/316L hat SCC-Risiko unter Vorspannung in warmen Gewässern. Duplex 2205 ist SCC-resistent, stärker, und hat höhere PREN. Mehrkosten: 2–3× gegenüber 316, aber bei Kielbolzen geht es um Lebenssicherheit.

### FAQ 8: Wie oft müssen Kielbolzen geprüft werden?

**Jährlich Sichtprüfung (Slip). UT-Prüfung alle 5 Jahre ab 10 Jahren Alter. Prophylaktischer Austausch nach 20 Jahren (316) oder bei SCC-Verdacht.**

### FAQ 9: Warum funktionieren Federringe (Grower) nicht?

Der Junker-Vibrations-Test hat 1969 bewiesen, dass Federringe KEINE messbare Sicherung gegen Vibrationslockerung bieten. Sie platten bei der ersten Belastung zusammen. Alternativen: Nyloc-Mutter, Nord-Lock Keilsicherung, Kronenmutter + Splint.

### FAQ 10: Welche Unterlegscheibe für GFK?

**Mindestens DIN 9021 (große Scheibe, 3× Bolzen-Ø).** Bei Sandwich-Deck: Backing Plate verwenden. Nie DIN 125 (kleine Scheibe) direkt auf GFK — Flächenpressung zu hoch.

### FAQ 11: Edelstahl-Bolzen in Aluminium — geht das?

**Ja, ABER nur mit vollständiger galvanischer Isolation.** Kunststoff-Buchse im Bohrloch + Isolierscheiben + Tef-Gel/Duralac. Ohne Isolation wird das Aluminium innerhalb von 1–3 Jahren zerstört.

### FAQ 12: Warum kein Kupfer-Anti-Seize im Marine-Bereich?

Kupferpartikel sind galvanisch aktiv. In Salzwasser-Umgebung bilden sie mit Edelstahl und Aluminium galvanische Paare → beschleunigte Korrosion. Nickel-basiertes oder PTFE-basiertes Anti-Seize verwenden.

### FAQ 13: Wie lang muss mein Bolzen sein?

**Bolzenlänge = Materialstärke(n) + Scheibe(n) + Mutter + 2–3 Gewindegänge.** Der Bolzen muss so weit durch die Mutter ragen, dass mindestens 2 komplette Gewindegänge sichtbar sind.

### FAQ 14: Darf ich metrische und Zoll-Gewinde mischen?

**NIEMALS.** Metrische und Zoll-Gewinde sind NICHT kompatibel, auch wenn der Durchmesser ähnlich ist. Steigung und Flankenwinkel unterscheiden sich. Falsches Gewinde = Gewindezerstörung.

### FAQ 15: Was ist eine Backing Plate und brauche ich eine?

**Ja, für jeden Durchgangsbolzen in GFK.** Eine Backing Plate ist eine Metallplatte (SS oder Alu) auf der Unterseite des Laminats, die die Kraft der Mutter auf eine große Fläche verteilt. Ohne Backing Plate wird GFK unter Last durchgezogen.

### FAQ 16: Wie dichte ich einen Bolzen-Durchbruch im Deck ab?

**Kernbereich mit Epoxid versiegeln (bei Sandwich-Deck). Bolzenloch mit Sikaflex 291 oder Butylband abdichten. Beschlag-Unterseite mit Sikaflex einbetten. Bolzen mit Tef-Gel einsetzen.** KEIN Silikon! Silikon haftet nicht dauerhaft auf GFK.

### FAQ 17: Mein Bolzen hat bräunliche Verfärbung — ist das gefährlich?

**Sofort untersuchen!** Bräunliche Verfärbung kann sein: Fremdrost (harmlos, abwischbar), Lochfraß (ernst), Spaltkorrosion (ernst), oder SCC-Anfang (kritisch). Bolzen lösen, inspizieren, Ursache feststellen.

### FAQ 18: Kann ich Bolzen kürzen?

**Ja, aber korrekt.** Gewindestange/Bolzen mit Metallsäge oder Trennscheibe kürzen. WICHTIG: Mutter VOR dem Schnitt auf den Bolzen drehen, dann nach dem Schnitt ABDREHEN → richtet beschädigtes Gewinde. Schnittkante entgraten.

### FAQ 19: Wo kaufe ich Edelstahl-Bolzen in der Karibik?

**Budget Marine** (Filialen in St. Maarten, Grenada, Trinidad, Curaçao) hat begrenztes Sortiment. Besser: **McMaster-Carr oder Bolt Depot** über US-Versand nach USPS-Adresse (z.B. in USVI). Oder Vorrat vor Atlantik-Überquerung anlegen.

### FAQ 20: Wann ist ein Bolzen zu alt?

**316-Bolzen in Seewasser: Ab 15 Jahren inspektionspflichtig, ab 20 Jahren Austausch empfohlen.** Duplex: Ab 25 Jahren inspizieren. Monel/Bronze: 30+ Jahre möglich. Entscheidend ist nicht nur das Alter, sondern die Belastung, das Segelrevier (Tropen = aggressiver), und die Wartungshistorie.

## 43. Glossar

- **A2**: Austenitischer Edelstahl der Gruppe 304 (V2A), NICHT für Seewasser geeignet
- **A4**: Austenitischer Edelstahl der Gruppe 316 (V4A), Marine-Standard
- **A4-70 / A4-80**: Festigkeitsklasse nach ISO 3506, Zugfestigkeit 700 bzw. 800 MPa
- **Anti-Seize**: Montagepaste zur Verhinderung von Galling und Korrosion
- **Backing Plate**: Lastverteilungsplatte unter Bolzenverbindungen auf GFK
- **Belleville-Scheibe**: Federnd-konische Unterlegscheibe (Tellerfeder)
- **BSP**: British Standard Pipe — Gewindestandard für Rohrverschraubungen
- **Clevis Pin**: Gabelbolzen, verbindet Gabel mit Öse im Rigging
- **Cotter Pin**: Splint, sichert Bolzen gegen Herausfallen
- **Durchgangsbolzen**: Bolzen, der DURCH alle Materialschichten geht, mit Mutter auf Gegenseite
- **Duplex**: Zweiphasiger Edelstahl (Austenit + Ferrit), höhere Festigkeit und Korrosionsbeständigkeit
- **Ermüdungsfestigkeit**: Belastungsgrenze bei zyklischer Last (Fatigue Limit)
- **Flanschmutter**: Mutter mit integrierter Unterlegscheibe
- **Galling**: Kaltverschweißung von Edelstahl-Gewinden durch Reibung unter Druck
- **Gewindestange**: Stange mit durchgehendem Gewinde, kein Kopf
- **Helicoil**: Gewindedraht-Einsatz zur Reparatur beschädigter Gewinde
- **Hutmutter**: Mutter mit geschlossenem Dom, schützt Gewindeende
- **Kielbolzen**: Bolzen, die den Kiel am Rumpf befestigen — lebensicherheitsrelevant
- **Kronenmutter**: Mutter mit Schlitzen für Splint-Sicherung
- **Monel**: Nickel-Kupfer-Legierung, exzellent in Seewasser, kein SCC
- **Nord-Lock**: Keilsicherungsscheibe gegen Vibrationslockerung
- **Nyloc**: Selbstsichernde Mutter mit Polyamid (Nylon)-Einsatz
- **PREN**: Pitting Resistance Equivalent Number = %Cr + 3,3×%Mo + 16×%N
- **Rp0,2**: 0,2%-Dehngrenze (Streckgrenze) in MPa
- **Rm**: Zugfestigkeit (Bruchfestigkeit) in MPa
- **SCC**: Stress Corrosion Cracking — Spannungsrisskorrosion, katastrophales Versagen
- **Sicherungsmutter**: Oberbegriff für alle Muttern mit Verdrehsicherung
- **Stiftschraube (Stud)**: Bolzen ohne Kopf, ein Ende eingeschraubt
- **Tef-Gel**: PTFE-basiertes Anti-Seize-Produkt, Marine-Standard #1
- **Toggle**: Gelenkverbindung am Wantenfuß, verhindert Biegemoment
- **Turnbuckle**: Wantenspanner (Spannschloss)
- **U-Bolt**: U-förmiger Bolzen zur Rohr- oder Stangenbefestigung
- **UNC**: Unified National Coarse — US-Grobgewinde
- **UNF**: Unified National Fine — US-Feingewinde
- **WLL**: Working Load Limit — zulässige Arbeitslast (= 1/3 Bruchlast)
- **Flankenwinkel**: Steigung der Gewindeflanken, typisch 60° (metrisch) oder 55° (Whitworth)
- **Kernloch**: Bohrung für Gewindeeinsatz, Durchmesser = Nenndurchmesser - 2×Steigung
- **Klemmlänge**: Effektive Verbindungslänge über die sich der Bolzen verteilt
- **Kopfauflage**: Kontaktfläche zwischen Bolzenkopf und Oberfläche, bei Blockverteilung entscheidend
- **Prüfkraft**: Bei Keilsicherungen die Kraft, bei der Verdrehsicherung eintritt (typically 50% der Nennlast)
- **Querkraft**: Scherkraft senkrecht zur Bolzenachse, führt zu Schubspannungen
- **Schaftlänge**: Reine Länge des ungewindeten Bolzenbereichs
- **Scherfläche**: Querschnittsfläche zur Berechnung von Schubspannungen (As = π/4 × d²)
- **Spannungsquerschnitt**: Effektiver Querschnitt des Gewindes unter Zugspannung (As aus ISO 898-1)
- **Streckgrenzensteuerung**: Kontrolliertes Anzugsverfahren bis zur Streckgrenze statt festes Drehmoment
- **Vorspannkraft**: Initiale Zugkraft im Bolzen unmittelbar nach dem Anziehen (Fi)
- **Gewindelänge**: Länge des gewindeten Bereichs am Bolzen
- **Selbsthemmung**: Bolzen-Gewindepaar, bei dem Rückwärtsbewegung durch Reibung blockiert ist
- **Rippenbolzen**: Bolzen mit Rippen zur Verdrehsicherung ohne Sicherungsmutter
- **Splintsicherung**: Sicherung durch Splint durch Kronenmutter oder Ausnehmungen

class KeelBoltRisk(BaseModel):

    bolt_age_years: int
    sailing_area: str
    bilge_temp_max_c: float
    last_ut_inspection_years: Optional[int]
    last_torque_check_years: Optional[int]

    risk_level: str  # "low", "medium", "high", "critical"
    scc_risk: str
    recommendation: str
    next_action: str

def assess_keel_bolt_risk(
    material: str,
    age_years: int,
    sailing_area: str,
    bilge_temp_c: float,
    last_ut: Optional[int] = None,
    last_torque: Optional[int] = None,
) -> KeelBoltRisk:
    """Bewertet das Risiko von Kielbolzen basierend auf Material, Alter und Einsatz."""
    risk = 0

    # Material-Risiko
    if material in ("304", "A2"):
        risk += 50  # SOFORT ersetzen
    elif material in ("316", "316L", "A4-70", "A4-80"):
        risk += age_years * 2  # Linearer Anstieg mit Alter
    elif material in ("Duplex", "2205"):
        risk += age_years * 0.8
    elif material in ("Monel", "Bronze"):
        risk += age_years * 0.5

    # Temperatur-Risiko (SCC)
    if material in ("316", "316L", "A4-70", "A4-80"):
        if bilge_temp_c > 50:
            risk += 20
        elif bilge_temp_c > 40:
            risk += 10

    # Segelrevier
    if sailing_area in ("tropics", "mediterranean"):
        risk += 10
    elif sailing_area in ("caribbean", "red_sea"):
        risk += 15

    # Inspektions-Defizite
    if last_ut is None or last_ut > 5:
        risk += 10 if age_years > 10 else 5
    if last_torque is None or last_torque > 3:
        risk += 5

    risk = min(risk, 100)

    if risk >= 70:
        level = "critical"
        rec = "Sofortiger Austausch empfohlen"
        action = "Kielbolzen-Austausch planen, UT-Prüfung sofort"
    elif risk >= 50:
        level = "high"
        rec = "UT-Prüfung dringend empfohlen"
        action = "UT-Prüfung innerhalb 3 Monate"
    elif risk >= 30:
        level = "medium"
        rec = "Regelmäßige Überwachung"
        action = "Nächste UT-Prüfung bei nächstem Slip"
    else:
        level = "low"
        rec = "Kein akuter Handlungsbedarf"
        action = "Weiter jährliche Sichtprüfung"

    scc = "high" if material in ("316", "316L") and bilge_temp_c > 40 and age_years > 10 else "medium" if material in ("316", "316L") else "low"

    return KeelBoltRisk(
        bolt_material=material,
        bolt_age_years=age_years,
        sailing_area=sailing_area,
        bilge_temp_max_c=bilge_temp_c,
        last_ut_inspection_years=last_ut,
        last_torque_check_years=last_torque,
        risk_score=risk,
        risk_level=level,
        scc_risk=scc,
        recommendation=rec,
        next_action=action,
        confidence="calculated",
    )

### E.5 Drehmoment-Rechner

```python
import math


class TorqueResult(BaseModel):
    model_config = {"from_attributes": True}

    bolt_diameter_mm: float
    pitch_mm: float
    material_class: str
    friction_coefficient: float
    preload_force_kn: float
    torque_nm: float
    stress_ratio_pct: float
    warnings: List[str]
    confidence: str


def calculate_torque(
    diameter_mm: float,
    pitch_mm: float,
    material_class: str = "A4-80",
    friction_mu: float = 0.14,
) -> TorqueResult:
    """Berechnet Anzugsmoment für Edelstahl-Bolzen nach VDI 2230."""
    # Spannungsquerschnitt nach ISO 898
    d2 = diameter_mm - 0.6495 * pitch_mm  # Flankendurchmesser
    d3 = diameter_mm - 1.2269 * pitch_mm  # Kerndurchmesser
    As = math.pi / 4 * ((d2 + d3) / 2) ** 2  # mm²

    # Streckgrenze nach Material
    rp02_map = {"A4-50": 210, "A4-70": 450, "A4-80": 600, "C4-80": 600}
    rp02 = rp02_map.get(material_class, 600)

    # Ausnutzungsgrad 90% der Streckgrenze
    sigma_target = rp02 * 0.9
    F_preload = sigma_target * As / 1000  # kN

    # Drehmoment nach VDI 2230 (vereinfacht)
    # M = F * (0.16 * P + 0.58 * d2 * µ + 0.5 * D_km * µ)
    D_km = (diameter_mm + d2) / 2  # Mittlerer Kopfauflagedurchmesser
    M = F_preload * 1000 * (0.16 * pitch_mm + 0.58 * d2 * friction_mu + 0.5 * D_km * friction_mu) / 1000  # Nm

    warnings = []
    if friction_mu > 0.20:
        warnings.append("WARNUNG: Hohe Reibung — Galling-Risiko!")
    if friction_mu < 0.08:
        warnings.append("WARNUNG: Sehr niedrige Reibung — Bolzen kann überspannt werden!")
    stress_pct = sigma_target / rp02 * 100

    return TorqueResult(
        bolt_diameter_mm=diameter_mm,
        pitch_mm=pitch_mm,
        material_class=material_class,
        friction_coefficient=friction_mu,
        preload_force_kn=round(F_preload, 1),
        torque_nm=round(M, 1),
        stress_ratio_pct=round(stress_pct, 1),
        warnings=warnings,
        confidence="calculated",
    )
```

## ANHANG F — Fehlerbild-Atlas

### F.1 Galling (Kaltverschweißung)

| Merkmal | Detail |
|---------|--------|
| **Aussehen** | Aufgerissene Gewindeflanken, Material-Aufwürfe, verschmiertes Metall |
| **Typische Stelle** | Edelstahl-Mutter auf Edelstahl-Bolzen, nach dem Anziehen |
| **Ursache** | Fehlende Schmierung, zu schnelles Anziehen, gleiche Härte |
| **Diagnose** | Mutter dreht schwer, dann plötzlich blockiert. Weiteres Drehen = Bolzenbruch |
| **Verwechslungsgefahr** | Korrosion (nein: Galling ist MECHANISCH, nicht chemisch) |
| **Maßnahme** | Mutternsprenger, Eindringöl. Prävention: IMMER Anti-Seize |

### F.2 Lochfraß (Pitting)

| Merkmal | Detail |
|---------|--------|
| **Aussehen** | Kleine, tiefe Löcher in der Oberfläche, oft unter Ablagerungen |
| **Typische Stelle** | Bolzenkopf-Unterseite, Gewindegrund, unter Scheiben |
| **Ursache** | Chlorid-Ionen durchbrechen Passivschicht lokal |
| **Diagnose** | Braune/schwarze Punkte, Lupe zeigt Krater |
| **Maßnahme** | Leicht: Passivieren. Tief: Bolzen ersetzen |

### F.3 Spaltkorrosion (Crevice Corrosion)

| Merkmal | Detail |
|---------|--------|
| **Aussehen** | Braune Verfärbung, Material-Abtrag in Spaltbereichen |
| **Typische Stelle** | Unter Unterlegscheiben, im Gewinde, in Flanschspalten |
| **Ursache** | O₂-Verarmung im Spalt + Chlorid-Konzentration |
| **Diagnose** | Mutter/Scheibe entfernen: Spaltbereich deutlich angegriffen |
| **Maßnahme** | Bolzen ersetzen, Spalt mit Dichtmittel eliminieren |

### F.4 SCC (Spannungsrisskorrosion)

| Merkmal | Detail |
|---------|--------|
| **Aussehen** | Haarrisse, oft unsichtbar von außen. Bruchfläche: intergranular, körnig |
| **Typische Stelle** | Kielbolzen, Wantenbolzen — überall unter HOHER Vorspannung |
| **Ursache** | Zugspannung + Chlorid + Temperatur (die SCC-Trias) |
| **Diagnose** | UT-Prüfung, Magnetpulver. Bruchfläche zeigt verzweigte Risse |
| **WARNUNG** | KEIN äußerlich sichtbares Zeichen vor dem Versagen! |
| **Maßnahme** | Sofortiger Austausch. Material auf Duplex/Monel upgraden |

### F.5 Galvanische Korrosion

| Merkmal | Detail |
|---------|--------|
| **Aussehen** | Weißes Pulver (Alu), grüne Patina (Kupfer/Bronze), schwarze Flecken (Stahl) |
| **Typische Stelle** | SS-Bolzen in Alu-Beschlag, SS-Bolzen in Bronze-Seeventil |
| **Ursache** | Potentialdifferenz zwischen Metallen + Elektrolyt (Seewasser) |
| **Diagnose** | Das UNEDLERE Material wird angegriffen (Alu, Bronze, Stahl) |
| **Maßnahme** | Isolation nachrüsten (Hülse + Scheibe + Tef-Gel) |

### F.6 Ermüdungsbruch (Fatigue Failure)

| Merkmal | Detail |
|---------|--------|
| **Aussehen** | Glatte Bruchfläche (Rissfortschritt) + raue Zone (Restbruch). „Muschelform" |
| **Typische Stelle** | Clevis Pins ohne Toggle, Rigging-Bolzen, Motor-Lager |
| **Ursache** | Zyklische Belastung (Wellengang, Vibration, Rigg-Pumpen) |
| **Diagnose** | Bruchfläche zeigt konzentrische Wachstumslinien (Beach Marks) |
| **Maßnahme** | Toggle nachrüsten, Bolzen größer dimensionieren, Nyloc/Nord-Lock |

| Merkmal | Detail |

| Merkmal | Detail |
