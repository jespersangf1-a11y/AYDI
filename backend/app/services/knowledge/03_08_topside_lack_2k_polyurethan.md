## 1. Grundlagen: 2K Polyurethan-Lacke im Yachtbau

### 1.1 Was ist ein 2K-PU-Topside-Lack?

2K-Polyurethan-Topside-Lacke (auch „Linear Polyurethan" oder „LPU" genannt) sind zweikomponentige Beschichtungssysteme, bei denen eine Polyol-Harz-Komponente (Basis) mit einer Isocyanat-Härter-Komponente (Converter/Activator) chemisch reagiert. Diese Reaktion bildet ein dreidimensional vernetztes Polyurethan-Netzwerk mit außergewöhnlicher Härte, Glanz, UV-Beständigkeit und chemischer Resistenz.

| Parameter | Detail |
|-----------|--------|
| **Vollständige Bezeichnung** | 2-Komponenten Lineares Polyurethan (2K LPU) |
| **Chemische Reaktion** | Polyol + Isocyanat → Polyurethan + CO₂ (Nebenreaktion bei Feuchtigkeit) |
| **Typischer Glanzgrad** | 85–95 GU bei 60° (neu) |
| **UV-Beständigkeit** | 5–10+ Jahre ohne signifikanten Glanzverlust |
| **Temperaturbeständigkeit** | Dauerhaft bis 80–120°C |
| **Chemische Resistenz** | Beständig gegen Kraftstoffe, Öle, Reinigungsmittel |
| **Verarbeitungstemperatur** | 15–30°C (optimal 18–25°C) |
| **Relative Luftfeuchtigkeit** | <65% (kritisch: Isocyanat reagiert mit Feuchtigkeit) |

> Confidence: `measured`

### 1.2 Abgrenzung: 2K-PU vs. 1K-PU vs. Alkyd

| Eigenschaft | 2K Polyurethan | 1K Polyurethan | Alkyd (1K) |
|-------------|---------------|----------------|------------|
| **Komponenten** | Basis + Härter | Feuchtigkeitshärtend | Lufttrocknend |
| **Glanz (60° GU)** | 85–95 | 70–85 | 60–80 |
| **UV-Beständigkeit** | 5–10+ Jahre | 3–5 Jahre | 1–3 Jahre |
| **Härte** | Sehr hoch (4H–6H Bleistift) | Mittel (2H–4H) | Gering (HB–2H) |
| **Chemische Resistenz** | Exzellent | Gut | Mäßig |
| **Verarbeitung** | Komplex, Isocyanat-Gefahr | Einfach | Sehr einfach |
| **Preis (€/m²)** | 25–80 | 15–30 | 5–15 |
| **Reparierbarkeit** | Schwieriger | Mittel | Einfach |
| **Beispielprodukte** | Awlgrip, Alexseal, Perfection | Toplac, EZ-Poxy | International Toplac (alt) |

> Confidence: `measured`

### 1.3 Warum 2K-PU der Standard für Yachten ist

Die Yacht-Industrie hat sich seit den 1980er Jahren auf 2K-PU als Standard-Topside-Finish etabliert, weil kein anderes System die Kombination aus Tiefenglanz, UV-Beständigkeit und Langlebigkeit bietet. Ein professionell applizierter Awlgrip- oder Alexseal-Lack hält 10–15 Jahre auf Superyachten und 5–10 Jahre auf Fahrtenyachten, bevor eine Auffrischung nötig wird.

| Markt-Segment | Typisches 2K-PU-System | Budget (€/m²) |
|---------------|----------------------|---------------|
| **Superyacht (>24m)** | Awlgrip / Alexseal (Spray) | 60–120 |
| **Semi-Custom (15–24m)** | Awlgrip / Alexseal / Perfection (Spray) | 40–80 |
| **Fahrtenyacht (10–15m)** | Perfection / Awlcraft SE / Hempathane (Roll&Tip) | 20–50 |
| **Produktionssegler (8–12m)** | Perfection / De IJssel / Toplac (Roll&Tip) | 15–35 |
| **Eigenleistung/Refit** | Perfection / Alexseal Roll (Roll&Tip) | 15–30 |

## 2. Chemie der Polyurethan-Beschichtung

### 2.1 Isocyanat-Polyol-Reaktion

| Parameter | Detail |
|-----------|--------|
| **Reaktionsgleichung** | R-NCO + R'-OH → R-NH-CO-O-R' (Urethan-Bindung) |
| **Isocyanat-Typen** | HDI (Hexamethylendiisocyanat), IPDI (Isophorondiisocyanat) |
| **Polyol-Typen** | Acryl-Polyole (Awlgrip), Polyester-Polyole (Alexseal) |
| **NCO:OH-Verhältnis** | Stöchiometrisch 1:1 — Abweichung führt zu Qualitätsverlust |
| **Nebenreaktion** | R-NCO + H₂O → R-NH₂ + CO₂ (Blasenbildung bei Feuchtigkeit!) |
| **Topfzeit** | Produktabhängig: 15 min (Awlgrip) bis 4h (Hempathane) |
| **Durchhärtung** | 7–14 Tage für volle chemische Resistenz |

> Confidence: `measured`

### 2.2 Aliphatische vs. Aromatische Isocyanate

| Eigenschaft | Aliphatisch (HDI/IPDI) | Aromatisch (MDI/TDI) |
|-------------|----------------------|---------------------|
| **UV-Beständigkeit** | Exzellent — kein Vergilben | Schlecht — vergilbt schnell |
| **Verwendung** | Topcoats (Awlgrip, Alexseal, Perfection) | Primer, Industriebeschichtungen |
| **Kosten** | Höher | Niedriger |
| **Gesundheitsrisiko** | Hoch (Isocyanat-Dampf) | Sehr hoch |
| **Aushärtungsgeschwindigkeit** | Langsamer | Schneller |
| **Glanzretention** | 5–10+ Jahre | 1–2 Jahre (dann Kreidebildung) |

> Confidence: `measured`

### 2.3 Vernetzungsdichte und mechanische Eigenschaften

| DFT (µm) | Härte (Bleistift) | Flexibilität | Schlagfestigkeit |
|-----------|-------------------|-------------|-----------------|
| **25–35** | 4H–6H | Gut | Gut |
| **35–50** | 4H–5H | Mittel | Mittel |
| **50–75** | 3H–4H | Gering | Gering (Rissrisiko) |
| **>75** | 3H | Sehr gering | Sprödbruch-Risiko |
| **Optimal** | 30–50 µm DFT pro Schicht, 2–3 Schichten = 60–150 µm gesamt |

> Confidence: `measured`

## 3. Gesundheit und Sicherheit: Isocyanat-Gefahren

### 3.1 Warum Isocyanate gefährlich sind

| Parameter | Detail |
|-----------|--------|
| **Hauptgefahr** | Isocyanat-Aerosole und -Dampf verursachen Berufsasthma |
| **Sensibilisierung** | Einmalige Exposition kann lebenslange Allergie auslösen |
| **GHS-Einstufung** | H334 (Sensibilisierung Atemwege), H317 (Hautsensibilisierung) |
| **MAK-Wert** | HDI: 0.005 ppm (5 µg/m³) — extrem niedrig |
| **Latenzzeit** | Symptome können 4–12 Stunden nach Exposition auftreten |
| **Irreversibilität** | Isocyanat-induziertes Asthma ist NICHT reversibel |

> Confidence: `measured`

### 3.2 Pflicht-Schutzausrüstung (PSA)

| PSA | Spezifikation | Pflicht bei |
|-----|--------------|-------------|
| **Atemschutz (Spritzen)** | Fremdbelüftete Vollmaske oder Druckluft-Atemschutz (EN 14594) | IMMER beim Spritzen |
| **Atemschutz (Rollen)** | Mindestens A2P3-Kombinationsfilter (EN 405) | Empfohlen, bei schlechter Belüftung Pflicht |
| **Handschuhe** | Nitril, Dicke ≥0.3mm, EN 374 | IMMER |
| **Schutzanzug** | Einweg-Overall Typ 5/6 (EN 13982/EN 13034) | IMMER beim Spritzen |
| **Augenschutz** | Dichtschließende Schutzbrille (EN 166) oder Vollmaske | IMMER |
| **Hautschutz** | Langarm, kein offenes Hautareal | IMMER |

> Confidence: `measured`

### 3.3 Belüftungsanforderungen

| Methode | Mindest-Luftwechsel | Empfehlung |
|---------|---------------------|------------|
| **Spritzkabine (Profi)** | 60× pro Stunde (EN 16985) | Standard für Werften |
| **Temporäre Einhausung** | 20× pro Stunde, Absaugung | Refit-Werften, DIY mit Setup |
| **Offene Halle** | 10× pro Stunde + Querlüftung | Minimum für Rolle+Tip |
| **Im Freien** | Natürliche Belüftung + Windrichtung beachten | Nur für Rolle+Tip, NIE spritzen |
| **Geschlossener Raum** | NICHT zulässig ohne Absaugung | NIE ohne aktive Belüftung |

> Confidence: `measured`

### 3.4 Isocyanat-Schulung und Vorschriften

| Land | Vorschrift | Anforderung |
|------|-----------|-------------|
| **DE** | TRGS 430 (Isocyanate) | Schulung Pflicht vor Umgang, Betriebsanweisung, Substitutionsprüfung |
| **UK** | COSHH + HSE ASTM (seit 2023) | Isocyanat-Zertifikat Pflicht für gewerbliche Nutzung |
| **EU** | REACH Restriction (EU 2020/1149) | Ab 08/2023: Isocyanat-Schulung Pflicht für alle gewerblichen Nutzer |
| **US** | OSHA 29 CFR 1910.1000 | PEL: 0.02 ppm TDI ceiling, Action Level: 0.005 ppm |
| **AUS** | WHS Regulations | Atmospheric Monitoring + Health Surveillance bei >0.003 ppm |

> Confidence: `measured`

### 3.5 Erste Hilfe bei Isocyanat-Exposition

| Exposition | Sofortmaßnahme |
|-----------|----------------|
| **Einatmen** | Sofort an frische Luft, ruhig lagern, Arzt rufen (auch bei Symptomfreiheit!) |
| **Hautkontakt** | Sofort mit Wasser und Seife waschen (KEIN Lösemittel!), kontaminierte Kleidung entfernen |
| **Augenkontakt** | 15 Minuten mit Wasser spülen, Augenklinik aufsuchen |
| **Verschlucken** | KEIN Erbrechen auslösen, sofort Notarzt |
| **Wichtig** | Symptome können verzögert auftreten — IMMER ärztliche Kontrolle nach Exposition |

> Confidence: `measured`

## 31. Rolle+Tip-Methode (Roll & Tip)

### 31.1 Was ist Roll & Tip?

Roll & Tip ist die bevorzugte DIY-Applikationsmethode für 2K-PU-Topside-Lacke. Eine Person rollt den Lack auf, eine zweite Person „tippt" sofort mit einem trockenen Lackpinsel die Rollenstruktur glatt. Bei korrekter Ausführung ist das Ergebnis nahezu sprühlackiert — 85–92 GU Glanz sind erreichbar.

> Confidence: `measured`

| Equipment | Spezifikation | Bezugsquelle |
|-----------|--------------|-------------|
| **Schaumrolle** | Geschlossenporiger Schaum, 10 cm, KEIN Velour | West Marine, SVB, Compass24 |

| **Rollenhalter** | Mini-Bügel passend für 10cm Rolle | Baumarkt |

| **Mischbecher** | Graduiert, lösemittelbeständig | PPG-Becher, ToolPRO |

## 40. Fehlerbilder und Troubleshooting

### Fehlerbild 1: Orangenhaut (Orange Peel)

| Parameter | Detail |
|-----------|--------|
| **Symptom** | Oberfläche hat Textur wie Orangenschale statt Spiegelglanz |
| **Ursache Spray** | Zu wenig Verdünner, zu hoher Druck, zu weit entfernt, zu schnelle Verdunstung |
| **Ursache Roll** | Rolle zu grob, zu wenig Verdünner, zu schnell getippt |
| **Lösung** | P1000–P1500 nass schleifen, polieren — oder schleifen + neue Schicht |
| **Vermeidung** | Korrekter Verdünner für Temperatur, richtige Düse, richtiger Abstand |

> Confidence: `measured`

### Fehlerbild 2: Laufnasen (Runs/Sagging)

| Parameter | Detail |
|-----------|--------|
| **Symptom** | Tropfen und Laufnasen auf vertikalen Flächen |
| **Ursache** | Zu dick aufgetragen, zu viel Verdünner, Substrat zu kalt |
| **Lösung** | Aushärten lassen, P320 schleifen, neu lackieren |
| **Vermeidung** | Dünnere Schichten, korrekte Verdünnung, Substrat vorwärmen |

> Confidence: `measured`

### Fehlerbild 3: Blasenbildung (Solvent Pop / Outgassing)

| Parameter | Detail |
|-----------|--------|
| **Symptom** | Kleine Blasen/Krater in der Oberfläche (wie Nadelstiche) |
| **Ursache** | Lösemittel im Primer nicht vollständig verdampft, Substrat porös, Schicht zu dick |
| **Lösung** | Schleifen P400, dünnere Schichten mit mehr Flash-Off-Zeit |
| **Vermeidung** | Primer 48h+ trocknen lassen, dünnere Schichten, mehr Flash-Off |

> Confidence: `measured`

### Fehlerbild 4: Fisheyes (Krater)

| Parameter | Detail |
|-----------|--------|
| **Symptom** | Kreisförmige Krater, Lack weicht vom Substrat zurück |
| **Ursache** | Silikone oder Öle auf der Oberfläche (Fingerabdrücke, Wachsreste, Politur) |
| **Lösung** | Komplett abtragen, gründlich entfetten, neu aufbauen |
| **Vermeidung** | GRÜNDLICHE Entfettung mit Awl-Prep/Silikonentferner, Nitrilhandschuhe |

> Confidence: `measured`

### Fehlerbild 5: Mattierung / Haze (Blushing)

### FAQ 2: Brauche ich wirklich einen Isocyanat-Atemschutz beim Rollen?

| Aspekt | Antwort |
|--------|---------|
| **Beim Spritzen** | JA — Fremdbelüftete Vollmaske PFLICHT, ohne Ausnahme |
| **Beim Rollen** | DRINGEND empfohlen — A2P3-Filter mindestens |
| **Begründung** | Auch beim Rollen entstehen Isocyanat-Dämpfe durch Lösemittel-Verdunstung |
| **Risiko** | Einmalige Sensibilisierung = lebenslang Isocyanat-Asthma |
| **Empfehlung** | Immer A2P3-Maske beim Rollen, draußen bei Wind akzeptabel ohne (aber riskant) |

> Confidence: `measured`

| **2K-PU (Perfection/Hempathane)** | 5–10 Jahre | Sehr gut | 8–15 Jahre |

## 45. Glossar

| Nr. | **Begriff** | Definition |
|-----|-----------|-----------|
| 1 | **2K-PU** | Zweikomponenten-Polyurethan — Basis (Polyol) + Härter (Isocyanat) |
| 2 | **LPU** | Linear Polyurethan — Fachbegriff für 2K-PU-Topside-Lacke |
| 3 | **Isocyanat** | Reaktive Komponente (R-NCO) im 2K-PU-System — gesundheitsgefährlich |
| 4 | **HDI** | Hexamethylendiisocyanat — aliphatisches Isocyanat für UV-stabile Lacke |
| 5 | **IPDI** | Isophorondiisocyanat — cycloaliphatisches Isocyanat, langsamer als HDI |
| 6 | **Polyol** | Hydroxylgruppen-haltiges Harz — Basis-Komponente des 2K-PU |
| 7 | **Converter** | Härter/Aktivator im 2K-System (enthält Isocyanat) |
| 8 | **DFT** | Dry Film Thickness — Trockenschichtdicke in µm |
| 9 | **WFT** | Wet Film Thickness — Nassschichtdicke in µm |
| 10 | **GU** | Gloss Units — Glanzeinheit, gemessen bei 60° Einfallswinkel |
| 11 | **Topfzeit** | Verarbeitungszeitraum nach Mischen von Basis + Converter |
| 12 | **Flash-Off** | Wartezeit zwischen Spritzgängen zum Abdunsten von Lösemittel |
| 13 | **Recoat-Fenster** | Zeitraum für nächste Schicht ohne Zwischenschleifen |
| 14 | **Feststoffgehalt** | Anteil nicht-flüchtiger Bestandteile (bestimmt DFT aus WFT) |
| 15 | **VOC** | Volatile Organic Compounds — flüchtige organische Verbindungen (g/l) |
| 16 | **Orangenhaut** | Textur wie Orangenschale — häufigster Roll&Tip-Fehler |
| 17 | **Fisheye** | Kraterförmige Defekte durch Silikone/Öle auf dem Substrat |
| 18 | **Blushing** | Milchig-matte Trübung durch Feuchtigkeitseinwirkung |
| 19 | **Chalking** | Kreidebildung — UV-Degradation des Bindemittels |
| 20 | **Solvent Pop** | Blasen durch eingeschlossenes Lösemittel |
| 21 | **Crazing** | Netzartige Haarrisse in der Oberfläche |
| 22 | **Roll&Tip** | Applikationsmethode: Rollen + sofortiges Glätten mit Pinsel |
| 23 | **Airless Spray** | Spritzverfahren ohne Luft — hydraulischer Druck atomisiert Lack |
| 24 | **HVLP** | High Volume Low Pressure — Spritzverfahren mit reduziertem Overspray |
| 25 | **Fairing** | Spachteln und Schleifen zur Herstellung einer perfekten Oberfläche |
| 26 | **Tie-Coat** | Verbindungsschicht zwischen verschiedenen Beschichtungssystemen |
| 27 | **Undercoat** | Grundierungsschicht unter dem Topcoat für Farbaufbau |
| 28 | **Etch Primer** | Haftungsgrundierung mit Phosphorsäure für Aluminium |
| 29 | **Tack Cloth** | Klebetuch zum Aufnehmen von Staub vor dem Lackieren |
| 30 | **Fine Line Tape** | Flexibles Abklebeband für scharfe Farbkanten |
| 31 | **Cross-Coat** | Kreuzweise Spritzgänge (horizontal + vertikal) |
| 32 | **Mil** | Tausendstel Zoll (1 mil = 25.4 µm) — US-Maßeinheit für DFT |
| 33 | **Reducer** | Verdünner — senkt Viskosität, verlängert offene Zeit |
| 34 | **Brushing Reducer** | Spezieller Verdünner für Rolle+Tip (langsamer als Spray-Reducer) |
| 35 | **Pot Life** | = Topfzeit — Verarbeitungsdauer nach Mischen |
| 36 | **Induction Time** | Wartezeit nach Mischen vor Verarbeitung (einige Produkte) |
| 37 | **Primer-Sealer** | Versiegelungsschicht über Spachtel/Primer vor Topcoat |
| 38 | **Wet Edge** | Nasskante — noch nicht angetrocknete Kante der letzten Bahn |
| 39 | **Overcoat Window** | = Recoat-Fenster |
| 40 | **PSA** | Persönliche Schutzausrüstung (Atemschutz, Handschuhe etc.) |

### Hinweis zur Klima-Matrix

Die Klima-Empfehlung basiert auf der Kombination aus Topfzeit, Verdünner-Verfügbarkeit, minimaler Applikationstemperatur und UV-Beständigkeit. Lokale Verfügbarkeit und Preis können die Empfehlung im Einzelfall überlagern — insbesondere in Regionen mit eingeschränktem Import (Karibik, Pazifik, Südostasien).
