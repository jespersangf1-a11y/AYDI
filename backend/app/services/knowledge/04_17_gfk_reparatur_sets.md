## 1. Grundlagen & Einführung

### 1.1 Definition

Ein **GFK-Reparatur-Set** (engl. Fiberglass Repair Kit) ist eine zusammengestellte Sammlung von Materialien und Werkzeugen, die es dem Bootsbesitzer oder Werftmitarbeiter ermöglicht, Schäden an glasfaserverstärktem Kunststoff (GFK/GRP/FRP) zu reparieren. Sets reichen von einfachen Gelcoat-Ausbesserungskits für €15 bis zu professionellen Komplettsystemen für €500+.

### 1.2 Warum GFK-Reparatur im Bootsbau kritisch ist

| Faktor | Relevanz |
|--------|---------|
| **90%+ aller Boote** sind aus GFK gefertigt | Riesiger Reparaturbedarf weltweit |
| **Osmose** betrifft 30–60% aller Polyester-Boote >15 Jahre | Strukturelle Reparatur unvermeidlich |
| **Grundberührung** ist häufigster Schadensfall | Jeder Eigner braucht Grundwissen |
| **Reparatur unterwegs** kann Reise retten | Bordwerkstatt ist Pflicht für Langfahrtsegler |
| **Professionelle Reparatur** kostet €50–150/h | Selbstreparatur spart erheblich |
| **Falsche Reparatur** kann gefährlich werden | Strukturelle Integrität muss gewährleistet sein |

### 1.3 Kategorien von GFK-Reparatur-Sets

| Kategorie | Inhalt | Für wen | Preis (€) |
|-----------|--------|---------|----------|
| **Gelcoat-Ausbesserung** | Gelcoat + Härter + Spachtel | Kratzer, Chips | 15–40 |
| **Kleines Reparatur-Set** | Harz + Härter + Glasfaser + Zubehör | Kleine Löcher, Risse | 25–60 |
| **Standard Reparatur-Set** | Harz + Härter + Glasfaser + Füller + Zubehör | Mittlere Schäden | 50–150 |
| **Profi-Reparatur-System** | Epoxid-System + alle Füller + Glasfaser + Werkzeug | Strukturelle Reparaturen | 150–500+ |
| **Osmose-Reparatur-Set** | Epoxid + Barrier Coat + Füller + Schleifmittel | Osmose-Sanierung | 200–800+ |
| **Notfall-Set (Bordwerkstatt)** | Schnellhärtendes Epoxid + Glasfaser + Tape | Unterwegs, Notfälle | 50–150 |

### 1.4 Harz-Grundlagen für Reparaturen

| Eigenschaft | Polyester | Vinyl-Ester | Epoxid |
|------------|-----------|------------|--------|
| **Haftung auf Altlaminat** | Moderat (60–70%) | Gut (75–85%) | **Exzellent (90–100%)** |
| **Feuchtigkeitsbeständigkeit** | Gering | **Sehr gut** | **Exzellent** |
| **Schrumpfung** | 6–8% | 3–5% | **1–2%** |
| **Festigkeit (Zugfestigkeit)** | 40–60 MPa | 60–80 MPa | **60–90 MPa** |
| **Preis pro kg** | €5–10 | €10–20 | **€15–40** |
| **Geruch/Styrol** | Stark (Styrol) | Mittel (Styrol) | **Minimal** |
| **Verarbeitungszeit** | 10–30 min | 15–30 min | **20–120 min** |
| **Verfügbarkeit weltweit** | Überall | Gut | **Überall** |
| **Empfehlung für Reparatur** | Budget, kosmetisch | Unterwasser, Osmose | **Strukturell, Premium** |

### 1.5 Glasfaser-Typen für Reparaturen

| Typ | Gewicht (g/m²) | Harz-Aufnahme | Stärke | Anwendung |
|-----|---------------|-------------|--------|-----------|
| **Chopped Strand Mat (CSM)** | 225, 300, 450, 600 | Hoch (2,5:1 Harz:Glas) | Moderat | Schichtaufbau, Füllmaterial, **NUR mit Polyester** |
| **Woven Roving** | 300, 600, 800 | Mittel (1,5:1) | Hoch | Strukturelle Reparatur, Verstärkung |
| **Biaxial (±45°)** | 300, 450, 600, 800 | Niedrig (1:1) | **Sehr hoch** | Strukturelle Reparatur, Infusion |
| **E-Glas Gewebe (Köper/Leinwand)** | 80, 160, 200, 300 | Niedrig | Gut | Drapierbar, Ecken, Oberfläche |
| **Glasfaser-Tape** | 50–300 g/m², 25–100 mm Breite | Niedrig | Gut | Kanten, Nähte, Verbindungen |
| **Glasfaser-Roving** | — | — | Hoch | Füllen von Hohlräumen |

**KRITISCH:** Chopped Strand Mat (CSM) ist mit Polyester-Binder versehen und löst sich **NUR** in Styrol (Polyester/VE). CSM mit Epoxid = VERSAGEN. Für Epoxid: Woven Roving oder Biaxial verwenden!

### 1.6 Verarbeitungsbedingungen

| Parameter | Minimum | Ideal | Maximum | Hinweis |
|-----------|---------|-------|---------|--------|

### 1.8 Reparatur-Hierarchie (Entscheidungsbaum)

```
Schaden am GFK-Boot erkannt
├── Nur Gelcoat betroffen?
│   ├── JA → Gelcoat-Reparatur (Kap. 19, 20.1)
│   └── NEIN ↓
├── Strukturelles Laminat betroffen?
│   ├── Kleiner Riss (<10 cm) → Epoxid + Glasfaser-Tape (Kap. 21)
│   ├── Größerer Riss (>10 cm) → Schäftung + Biaxial-Layup (Kap. 21.3)
│   ├── Loch/Durchbruch → Professionell ODER Schäftung mit Stützform
│   └── NEIN ↓
├── Sandwichkern betroffen?
│   ├── Kern nass aber intakt → Trocknen + Epoxid-Injektion
│   ├── Kern verrottet → Kern ersetzen (Kap. 20.5, 20.6)
│   └── NEIN ↓
├── Kiel/Ruder betroffen?
│   ├── JA → PROFESSIONELLE REPARATUR (Kap. 20.4, 20.7)
│   └── NEIN → Neue Diagnose
└── Osmose?
    └── JA → Vollständige Osmose-Sanierung (Kap. 20.3, Anhang C.5)
```

## 3. Polyester vs. Vinyl-Ester vs. Epoxid für Reparaturen

### 3.1 Detaillierter Vergleich

| Eigenschaft | Polyester | Vinyl-Ester | Epoxid |
|------------|-----------|------------|--------|
| **Haftung auf altem PE-Laminat** | 65% | 80% | **95%** |
| **Haftung auf altem Epoxid-Laminat** | 50% | 65% | **95%** |
| **Haftung auf nassem/feuchtem GFK** | 30% | 45% | **70% (G/flex: 85%)** |
| **Feuchtigkeitsaufnahme** | 2–3% | 0,5–1% | **0,3–0,5%** |
| **Schrumpfung bei Aushärtung** | 6–8% | 3–5% | **1–2%** |
| **Zugfestigkeit** | 40–60 MPa | 60–80 MPa | **60–90 MPa** |
| **Dehnung bis Bruch** | 2–3% | 3–5% | **3–6%** |
| **Ermüdungsbeständigkeit** | Gering | Mittel | **Hoch** |
| **UV-Beständigkeit** | Gut | Gut | **Moderat** (Schutz nötig) |
| **Chemische Beständigkeit** | Moderat | **Sehr gut** (Kraftstoff, Lösemittel) | Gut |
| **Topfzeit (25°C)** | 10–20 min | 15–25 min | **20–120 min** |
| **Volle Aushärtung** | 24–48 h | 24–48 h | **48–72 h (7 Tage optimal)** |
| **Styrol-Emission** | **Stark** | Moderat | **Keine** |
| **CSM-kompatibel?** | ✅ Ja | ✅ Ja | **❌ Nein** |
| **Preis (€/kg)** | 5–10 | 10–20 | **15–40** |
| **Verfügbarkeit Karibik** | Überall | Begrenzt | West System überall |

### 3.2 Entscheidungsbaum: Welches Harz?

```
Welcher Schaden?
│
├─ Kosmetisch (Gelcoat-Kratzer, Chips)
│  └─ Gelcoat (Polyester) — passend zum Original
│
├─ Strukturell (Risse, Löcher, Delamination)
│  ├─ Budget → Polyester (wenn Original = Polyester)
│  └─ **Empfehlung → EPOXID** (West 105, TotalBoat 5:1)
│
├─ Unterwasser (Osmose, Blistering)
│  └─ **EPOXID** (West 105 + 422 Barrier Coat) oder Vinyl-Ester
│
├─ Notfall unterwegs
│  └─ **West G/flex 655** (klebt auf nassem GFK) oder **Marine-Tex**
│
└─ Kiel/Ruder (strukturell kritisch)
   └─ **EPOXID** (West 105 + 404 oder 406) — NIEMALS Polyester
```

### 3.3 Kompatibilitätsregeln

| Altlaminat | Reparatur-Harz | Kompatibilität | Hinweis |
|-----------|---------------|---------------|--------|
| Polyester | Polyester | ✅ Gut | Billigstes, aber schwächstes |
| Polyester | Vinyl-Ester | ✅ Gut | Bessere Festigkeit |
| Polyester | **Epoxid** | ✅ **Sehr gut** | **Empfohlen** |
| Epoxid | Polyester | ⚠️ Schlecht | Haftung nur 50%; nicht empfohlen |
| Epoxid | Vinyl-Ester | ⚠️ Moderat | Akzeptabel, nicht ideal |
| Epoxid | **Epoxid** | ✅ **Exzellent** | Einzige richtige Wahl |
| Vinyl-Ester | Polyester | ✅ Akzeptabel | Schwächer als Original |
| Vinyl-Ester | **Epoxid** | ✅ **Sehr gut** | **Empfohlen** |

**Goldene Regel:** Epoxid haftet auf ALLEM. Polyester haftet nur gut auf Polyester. Im Zweifel: **EPOXID**.

### 3.4 Mechanische Eigenschaften im Vergleich

| Eigenschaft | Polyester (ISO) | Vinyl-Ester (VE) | Epoxid | Einheit |
|------------|----------------|-----------------|--------|---------|
| **Zugfestigkeit** | 40–65 | 65–82 | 65–90 | MPa |
| **Druckfestigkeit** | 90–120 | 110–140 | 110–150 | MPa |
| **Biegefestigkeit** | 70–110 | 120–140 | 120–160 | MPa |
| **Dehnung** | 1,5–3,0 | 3,0–5,0 | 3,0–6,0 | % |
| **E-Modul** | 3.000–3.500 | 3.200–3.600 | 3.000–3.800 | MPa |
| **Wasseraufnahme (24h)** | 0,5–1,5 | 0,1–0,3 | 0,05–0,15 | % |
| **Glasübergangstemperatur (Tg)** | 65–80 | 100–120 | 50–80 (Standard) / 120+ (post-cured) | °C |
| **Haftung auf geschliffenem GFK** | 60–70% der Kohäsion | 75–85% | 90–100% | Scherfestigkeit |

| Szenario | Polyester-Kosten | Epoxid-Kosten | Empfehlung |

### 12.4 Verarbeitungshinweise für Polyester-Kits

**Mischverhältnis MEKP:**
- Standard: 1–2% MEKP bezogen auf Harzgewicht (10–20 ml pro kg)
- Temperaturanpassung: <15°C → 2%, 15–25°C → 1,5%, >25°C → 1%
- NIEMALS mehr als 3% MEKP → exotherme Reaktion, Verbrennung, Brandgefahr!
- MEKP-Tröpfchen-Methode: 1 Tropfen ≈ 0,05 ml → für 100 g Harz: 15–30 Tropfen

**Schichtaufbau mit Polyester:**
1. CSM 300 g/m² als erste Lage (Bindeschicht zum Untergrund)
2. Woven Roving 450–600 g/m² als Strukturlage
3. CSM 300 g/m² als Abschlusslage (glatte Oberfläche)
4. Styrol-Wachs-Lösung (Surfacing Agent) auf letzte Lage → verhindert Tacky-Oberfläche

**Typische Fehler bei Polyester-Verarbeitung:**
- Zu viel Härter → spröde, rissig, verfärbt (gelb-braun)
- Zu wenig Härter → nicht ausgehärtet, klebrig, muss entfernt werden
- Mischen bei >30°C → zu schnelle Gelierung, keine Zeit für saubere Verarbeitung
- CSM mit Epoxid verwenden → Styrol-Binder löst sich nicht → Delaminierung garantiert

| **Zu langsam** | Zu wenig MEKP oder zu kalt | Min. 1% MEKP, Temperatur >18°C |

### 20.8 Stringer-Ablösung (Stringer Detachment)

**Ursachen:** Vibration (Motorboote), Alterung der PE-Klebung, Impact-Schäden, mangelhafter Bauaufbau.

**Diagnose:**
- Hohler Klang beim Abklopfen entlang des Stringers
- Sichtbare Rissbildung am Stringer-Übergang zur Außenhaut
- Wassereinlagerung im Hohlraum → Feuchtemessung >12%
- Bei Belastung (Motor laufen lassen): spürbare Vibration/Bewegung

**Reparatur-Prozedur:**
1. Freilegen: Stringer beidseits zugänglich machen, Verkleidung entfernen
2. Reinigung: Alte Klebung entfernen bis zum gesunden Laminat
3. Trocknung: Mindestens 2 Wochen bei >20°C, <60% rF (Feuchtemessung kontrollieren)
4. Vorbereitung: Schleifen P40, Aceton-Reinigung, Staub absaugen
5. Kleben: West 105 + 206 + 406 (Colloidal Silica) als thixotroper Kleber → Kehlnaht beidseits
6. Laminieren: 2× Biax 450 g/m² über Kehlnaht + 200 mm auf Außenhaut beidseits
7. Aushärtung: 48 h bei >15°C, dann Nachtemperung 6 h bei 40°C (wenn möglich)

**Materialien:** West 105 + 206 Slow + 406 Colloidal Silica + Biax ±45° 450 g/m² + Glasgewebe 300 g/m²
**Kosten:** ~80–150 € Material pro Meter Stringer
**Schwierigkeitsgrad:** Fortgeschritten (Zugang oft extrem schwierig)

### 20.9 Deck-Hardware-Leckage (Deck Fitting Water Ingress)

**Ursachen:** Alterung der Dichtmasse, fehlende Hinterlegung, Schrauben-Korrosion, thermische Ausdehnung.

**Diagnose:**
- Wasserflecken unter Deck (Deckenverkleidung anheben)
- Weicher Kern um Beschlagschrauben → Klopftest
- Feuchtemessgerät: >15% = definitiv Wassereinlagerung

**Reparatur-Prozedur (Kern-Sanierung):**
1. Beschlag demontieren, Dichtmasse entfernen
2. Bohrlöcher auf 10–12 mm aufbohren
3. Nassen Kern mit spitzem Werkzeug entfernen (3–5 cm Radius)
4. Trocknen: Heißluftföhn vorsichtig, dann 1–2 Wochen offen lassen
5. West 105 + 205 (dünn) in Hohlraum fließen lassen → Kern konsolidieren
6. Nach Aushärtung: Hohlraum mit West 105 + 404 (High-Density) verfüllen
7. Beschlag mit Sikaflex 291i oder 3M 4200 neu montieren, Backing Plate prüfen

**Materialien:** West 105 + 205 (dünn) + 404 (High-Density Filler) + Sikaflex 291i
**Kosten:** ~15–30 € pro Beschlag (Material)
**Profi-Tipp:** Backing Plates aus Edelstahl 316L oder G10-Platte IMMER verwenden — kein Beschlag direkt in Kern!

### 21.2 Glasfaser-Layup für Reparatur

| Schritt | Aktion | Hinweis |
|---------|--------|--------|
| 1 | Glasfaser-Stücke VORHER zuschneiden (trocken) | Jede Lage 25 mm größer als vorherige |
| 2 | Lagen nummerieren und sortieren | Kleinste zuerst (auf Schadensstelle), größte zuletzt |
| 3 | Oberfläche mit reinem Epoxid benetzen ("Wet-Out Coat") | Dünn, gleichmäßig, keine Pfützen |
| 4 | Erste Lage auflegen, mit Entlüfterrolle tränken | Von Mitte nach außen rollen |
| 5 | Luftblasen ausrollen (Entlüfterrolle oder Stipple-Pinsel) | JEDE Blase muss raus! |
| 6 | Nächste Lage Wet-on-Wet (innerhalb der Topfzeit) | Fasern um 45° drehen wenn möglich |
| 7 | Zwischen Lagen: dünne Epoxid-Schicht falls nötig | Nur wenn Fasern trocken erscheinen |
| 8 | Letzte Lage: dünn überrollen, kein Epoxid-Überschuss | Harzreiche Oberfläche = schwächer |
| 9 | Peel Ply auflegen (optional, empfohlen) | Hinterlässt perfekte Oberfläche zum Schleifen |
| 10 | Aushärten lassen (NICHT berühren!) | Min. 24h bei >18°C |

### 21.3 Schäftung (Scarf Joint) — Strukturelle Reparatur

| Schritt | Aktion | Hinweis |
|---------|--------|--------|
| 1 | Schadensbereich markieren (Rand + 50 mm) | Ovale/runde Form bevorzugt |
| 2 | Laminat-Dicke messen | Tiefenmesser oder Bohrprobe am Rand |
| 3 | Schäftungslänge berechnen | L = Dicke × 12 (optimal) → z.B. 6 mm dick = 72 mm Schäftung |
| 4 | Konzentrische Ringe markieren (pro Lage) | Abstand = Glasfaser-Lagendicke × 12 |
| 5 | Material mit Schleifscheibe/Dremel abtragen | 80er Körnung, stufenweise Ringen folgen |
| 6 | Oberfläche reinigen (Aceton), trocknen | Kein Staub, kein Fett |
| 7 | Erste Lage (kleinste) auf Schadensgrund legen | Mit Epoxid benetzen |
| 8 | Folgende Lagen aufbauen (pyramidenförmig) | Jede Lage deckt Ring + vorherige Lage |
| 9 | Letzte Lage deckt gesamten Schäftungsbereich | +25 mm über gesundes Laminat |
| 10 | Peel Ply oder Mylar für Finish | Peel Ply für weitere Schichten; Mylar für Gelcoat |
| 11 | Aushärten (24–72h je nach Temperatur) | Nicht forciert heizen (Spannungen!) |
| 12 | Finish: Schleifen → Gelcoat → Polieren | 220→400→800→1000 |

### 21.4 Gelcoat-Reparatur — Schritt für Schritt

| Schritt | Aktion | Hinweis |
|---------|--------|--------|
| 1 | Schadensbereich reinigen (Aceton) | Kein Wachs, kein Fett |
| 2 | V-Nut schleifen (bei Riss) oder Rand anfasen | 220er Schleifpapier |
| 3 | Abkleben (Kreppband, 5 mm Abstand zum Rand) | Für saubere Kanten |
| 4 | Gelcoat anmischen + MEKP (1,5–2%) | Nicht zu viel MEKP (Verfärbung!) |
| 5 | Ggf. Farbpaste einmischen | Tropfenweise, gut verrühren |
| 6 | Gelcoat mit Pinsel/Spachtel auftragen | Max. 0,5 mm pro Schicht |
| 7 | Mylar-Folie auflegen (glatt streichen) | Verhindert Luftinhibition → tack-freie Oberfläche |
| 8 | Aushärten lassen (2–4h) | Nicht in der Sonne (zu schnell) |
| 9 | Mylar entfernen | Vorsichtig abziehen |
| 10 | Nassschliff 400→800→1000→1500 | Immer nass schleifen bei Gelcoat |
| 11 | Rubbing Compound | Mit Pad oder Polierscheibe |
| 12 | Finish-Politur | Für Hochglanz |
| 13 | Marine-Wachs | Schutz und Glanz |

### 21.5 Marine-Tex Notfall-Reparatur — Schritt für Schritt

| Schritt | Aktion | Hinweis |
|---------|--------|--------|
| 1 | Oberfläche so gut wie möglich reinigen | Frischwasser, trocknen |
| 2 | Marine-Tex aus Tube nehmen (Harz + Härter) | 5:1 Volumen, visuell dosieren |
| 3 | 2–3 Minuten gründlich kneten/mischen | Gleichmäßige Farbe = gut gemischt |
| 4 | Auf Schadensbereich drücken und formen | Überstehend auftragen (Schleifreserve) |
| 5 | Oberfläche glätten (angefeuchteter Finger) | Für weniger Schleifarbeit |
| 6 | 24h aushärten lassen | NICHT berühren |
| 7 | Schleifen (220→400→800) | Marine-Tex ist gut schleifbar |
| 8 | Gelcoat oder Lack als UV-Schutz | Marine-Tex ist NICHT UV-stabil |

## 22. Typische Fehler & Troubleshooting

### 22.1 Die 12 häufigsten Reparatur-Fehler

| # | Fehler | Konsequenz | Prävention |
|---|--------|-----------|-----------|
| 1 | **Unzureichende Vorbereitung** | Haftungsversagen | 80er schleifen, Aceton reinigen |
| 2 | **Falsches Mischverhältnis** | Epoxid härtet nicht oder wird spröde | Dosierpumpen oder Waage verwenden |
| 3 | **Feuchtigkeit im Laminat** | Blasen, Delamination | Feuchtemessgerät, TROCKNEN |
| 4 | **CSM mit Epoxid** | Versagen (Binder löst sich nicht) | Woven/Biaxial für Epoxid |
| 5 | **Zu dicke Gelcoat-Schicht** | Rissbildung | Max. 0,5 mm pro Schicht |
| 6 | **Reparatur bei Kälte (<15°C)** | Unvollständige Aushärtung | Heizstrahler, Min. 18°C |
| 7 | **Reparatur bei Hitze (>35°C)** | Zu schnelle Gelierung | Slow Hardener (206/209) |
| 8 | **Kein Schäftungsverhältnis** | Schwache Reparatur | 12:1 Schäftung (min. 8:1) |
| 9 | **Zu wenige Lagen** | Reparatur schwächer als Original | Originaldicke + 10% Zuschlag |
| 10 | **Amine-Blush nicht entfernt** | Nächste Schicht haftet nicht | Warmwasser + Scotch-Brite |
| 11 | **Polyester auf Epoxid** | Schwache Haftung (50%) | Immer Epoxid auf Epoxid |
| 12 | **Kein Finish (UV-Schutz)** | Epoxid degradiert unter UV | Gelcoat, Lack oder Antifouling |

### 22.2 Troubleshooting-Tabelle

| Symptom | Ursache | Lösung |
|---------|---------|--------|
| **Epoxid bleibt klebrig** | Falsches Verhältnis; zu kalt; überlagert | Abkratzen, neu mischen (exakt!); erwärmen |
| **Epoxid wird milchig** | Feuchtigkeit (Amine-Blush) | Abwaschen (Warmwasser + Scotch-Brite), dann schleifen |
| **Blasen im Laminat** | Luft eingeschlossen | Entlüfterrolle; langsamer arbeiten; von Mitte nach außen |
| **Glasfaser löst sich** | Oberfläche nicht vorbereitet | 80er schleifen + Aceton reinigen vor Layup |
| **Glasfaser weiße Stellen** | Nicht vollständig durchtränkt (dry spot) | Mehr Harz auftragen + Rolle → Wet-Out sicherstellen |
| **Gelcoat klebrig** | Luftinhibition (Sauerstoff) | Mylar-Folie oder wachshaltiges Finish-Harz verwenden |
| **Gelcoat-Risse** | Zu dicke Schicht; zu viel MEKP | Max. 0,5 mm/Schicht; 1–2% MEKP |
| **Farbe passt nicht** | Gelcoat vergilbt/bleicht über Zeit | Größere Fläche reparieren + Polieren; Farbmischservice nutzen |
| **Epoxid läuft an Vertikale** | Zu dünn; keine Füller | 406 oder 407 als Verdicker → thixotrop machen |
| **Epoxid wird heiß (exotherm)** | Zu große Masse im Becher | Sofort dünn auf große Fläche verteilen! Nie in Becher stehen lassen |
| **Polyester-Geruch zu stark** | Styrol-Emission | ATEMSCHUTZ! Belüftung! Styrolreduziertes Harz verwenden |
| **MEKP-Überdosierung** | Zu viel Härter (>3%) | Riss-Risiko, Verfärbung → Material entfernen, neu ansetzen |
| **Epoxid auf Kleidung** | Versehentlicher Kontakt | Aushärten lassen, dann mechanisch entfernen. NICHT mit Aceton! |
| **Dosierpumpen-Fehler** | Unvollständige Hübe | Volle Hübe! Bei Unsicherheit: Waage verwenden |

### 22.3 Diagnose-Flussdiagramm

```
Reparatur schlägt fehl?
├── Epoxid härtet nicht?
│   ├── Falsches Mischverhältnis → Abkratzen, neu mischen (Waage!)
│   ├── Zu kalt (<15°C) → Heizstrahler, Fast Hardener
│   └── Überlagert (Härter kristallisiert) → Neues Material
├── Haftung schlecht?
│   ├── Oberfläche nicht geschliffen → 80er Schliff, Aceton
│   ├── Amine-Blush nicht entfernt → Warmwasser + Scotch-Brite
│   ├── Feuchtigkeit im Substrat → Feuchtemessung, Trocknung
│   └── CSM mit Epoxid → Woven/Biaxial verwenden!
├── Blasen/Lufteinschlüsse?
│   ├── Zu schnell gearbeitet → Langsamer, Entlüfterrolle
│   ├── Zu viele Lagen auf einmal → Max. 3 Lagen Wet-on-Wet
│   └── Vakuumbagging erwägen → Bessere Kompaktierung
└── Gelcoat-Problem?
    ├── Klebrig → Mylar-Folie oder wachshaltiges Harz
    ├── Risse → Dünnere Schichten (max. 0,5 mm)
    ├── Farbe falsch → Professioneller Farbmischservice
    └── Vergilbt → UV-stabilisiertes ISO-NPG Gelcoat verwenden
```

### 25.10 Bewertungskriterien für Forum-Empfehlungen

Bei der Auswertung von Forum-Empfehlungen muss AYDI folgende Faktoren berücksichtigen:

**Glaubwürdigkeitsfilter:**
- Beitragsanzahl des Autors: >500 Beiträge = erfahren, <50 = Anfänger
- Nachgewiesene eigene Reparaturerfahrung (Fotos, detaillierte Berichte)
- Bestätigung durch andere erfahrene Mitglieder
- Zeitraum der Erfahrung: "Seit 20 Jahren" vs "Mein erstes Boot"

**Regionale Verfügbarkeit:**
- US-Forum-Empfehlungen gelten nicht automatisch für EU-Markt (Versand, Zoll)
- UK-Empfehlungen nach Brexit: Zoll + VAT bei Import in EU
- Deutsche Forum-Empfehlungen bevorzugen R&G/Yachtcare → Verfügbarkeitsvorteil

**Bias-Erkennung:**
- "Ich benutze nur [Marke X]" ohne Vergleichserfahrung → eingeschränkte Aussagekraft
- Verkäufer/Händler-Beiträge → möglicher Interessenkonflikt
- Rezente Empfehlungen (< 2 Jahre) bevorzugen → Produktformulierungen ändern sich

## 31. Lagerung & Haltbarkeit

### 31.1 Haltbarkeit nach Produkt

| Produkt | Ungeöffnet | Geöffnet | Temperatur | Hinweis |
|---------|-----------|---------|-----------|--------|
| **West 105 Resin** | 3+ Jahre | 12–18 Monate | 5–30°C | Kristallisiert bei <10°C (Wasserbad löst) |
| **West 205/206 Hardener** | 3+ Jahre | 12 Monate | 5–30°C | Vergilbt = noch OK |
| **G/flex 650/655** | 2+ Jahre | 12 Monate | 5–30°C | Vorverdickt, kühl lagern |
| **Marine-Tex** | 2+ Jahre | 6–12 Monate | 10–30°C | Tube verschließen! |
| **Polyester-Harz** | 6–12 Monate | 3–6 Monate | 10–25°C, dunkel | Styrol verflüchtigt |
| **MEKP-Härter** | 12 Monate | 6 Monate | Kühl, dunkel | Brandgefahr! Separat lagern |
| **Gelcoat** | 6–12 Monate | 3–6 Monate | 10–25°C | Haut entfernen, umrühren |
| **Glasfaser** | Unbegrenzt | Unbegrenzt | Trocken | Vor Feuchtigkeit schützen |

### 31.2 Lagerung an Bord

| Hinweis | Begründung |
|---------|-----------|
| **Hitze vermeiden** | Bilge oder Kabine unter Deck, NICHT in der Sonne (Harz wird dünn, Haltbarkeit sinkt) |
| **Trocken halten** | Glasfaser ist hygroskopisch; Epoxid-Härter reagiert mit Feuchtigkeit → Kristallisation |
| **Separat lagern** | MEKP-Härter separat (Brandgefahr!), Epoxid getrennt von Polyester |
| **Verschlossen** | Alle Behälter sofort nach Gebrauch verschließen (Härter oxidiert) |
| **Kontrollieren** | Alle 6 Monate: Konsistenz prüfen, Verfallsdaten checken, Pumpen reinigen |
| **Zip-Lock-Beutel** | Jedes Produkt in eigenen Beutel → Auslaufschutz |
| **Temperaturbereich** | Ideal: 15–25°C. Unter 10°C: Harz kristallisiert. Über 35°C: Haltbarkeit halbiert |

### 31.3 Kristallisation von Epoxid-Harz

| Frage | Antwort |
|-------|---------|
| **Was passiert?** | Harz wird trüb, dickflüssig, oder bildet Kristalle am Boden |
| **Ist es noch verwendbar?** | JA — Kristallisation ist reversibel |
| **Wie lösen?** | Flasche in warmes Wasserbad (50°C) stellen, regelmäßig schütteln, 30–60 min |
| **Mikrowelle?** | NIEMALS — Gefahr der Überhitzung und Polymerisation |
| **West System Empfehlung** | Flasche seitlich in Warmwasser legen, alle 10 min drehen |
| **Prävention** | Über 15°C lagern; Flasche nicht langfristig bei Kälte stehen lassen |

### 31.4 Haltbarkeit verlängern

| Maßnahme | Wirkung |
|----------|--------|
| **Stickstoff-Überlagerung** | Oxidation des Härters verhindern (bei größeren Gebinden) |
| **Originalbehälter** | NIEMALS in andere Behälter umfüllen (Kontamination) |
| **Aufrecht lagern** | Pumpen nach Gebrauch sauber wischen, Harz tropft nicht in Gewinde |
| **Kleinere Gebinde** | Harz lieber in 2× 1 kg als 1× 2 kg → zweites Gebinde bleibt frisch |
| **FIFO-Prinzip** | First In, First Out — ältere Materialien zuerst verbrauchen |

## 32. FAQ — Häufige Fragen

### F1: Welches Reparatur-Set soll ich kaufen?
**West System 105-K** als Basis. Plus: **G/flex 655** für Notfälle und **Marine-Tex** als Epoxid-Kitt. Dazu **Gelcoat** (farblich passend) separat kaufen.

### F2: Epoxid oder Polyester für Reparaturen?
**Epoxid** für ALLES Strukturelle. Polyester NUR für Gelcoat (Farbkompatibilität) und Budget-Kosmetik. Forum-Konsens: "Im Zweifel: Epoxid."

### F3: Kann ich Epoxid auf Polyester-Laminat auftragen?
**JA** — Epoxid haftet exzellent auf geschliffenem Polyester. Umgekehrt (Polyester auf Epoxid) ist problematisch.

### F4: Was ist G/flex und wann brauche ich es?
G/flex ist ein flexibles Epoxid (30% Dehnung vs. 3–5% bei Standard-Epoxid). Ideal für: nasses/feuchtes GFK, Holz-GFK-Verbindungen, Metall-GFK-Verbindungen, Reparaturen unter schwierigen Bedingungen.

### F5: Wie viele Lagen Glasfaser brauche ich?
Faustregel: Original-Laminatdicke messen, dann in Glasfaser-Lagen umrechnen. 200 g/m² Gewebe = ~0,25 mm pro Lage bei 55% FVA. Zuschlag: +10–20% über Original.

### F6: Was ist Amine-Blush und wie entferne ich es?
Wachsartiger Film, der bei Epoxid-Aushärtung unter 23°C entsteht. **Entfernung:** Warmwasser + Scotch-Brite VOR dem Schleifen. NIEMALS mit Lösemittel.

### F7: Wie lange muss ich eine Osmose-Reparatur trocknen lassen?
**Minimum 4–6 Wochen**, ideal **8–12 Wochen**. Feuchtemessgerät verwenden: <0,5% Restfeuchte. Praxisregel: "So lang wie möglich."

### F8: Kann ich GFK unter Wasser reparieren?
**Provisorisch ja:** West G/flex 650 oder Epoxid-Stick (Marine-Tex, TotalBoat MixStix). **Permanent:** Boot muss raus; unter Wasser = Notfall.

### F9: Was kostet eine professionelle GFK-Reparatur?
Arbeitsstunde: **€50–150**. Gelcoat-Chip: €50–100. Strukturelle Reparatur (30 cm Riss): €300–800. Osmose-Sanierung (10 m Boot): **€3.000–8.000+**.

### F10: Ist West System das einzig Wahre?
West System ist der Referenzstandard, aber NICHT die einzige Option. TotalBoat, System Three, MAS, R&G sind gleichwertig. West hat den besten Support und die weltweite Verfügbarkeit.

### F11: Wie erkenne ich ob mein Boot Osmose hat?
**Symptome:** Blasen (2–20 mm Durchmesser) unter dem Gelcoat, besonders am Unterwasserschiff. Blasenflüssigkeit riecht nach Essig (Acetylgeruch). **Diagnose:** Feuchtemessung (Sovereign, Tramex).

### F12: Kann ich Chopped Strand Mat (CSM) mit Epoxid verwenden?
**NEIN.** CSM hat einen Polyester-Binder, der sich nur in Styrol (Polyester/VE) löst. Mit Epoxid bleibt CSM steif und haftet nicht. Für Epoxid: Woven Cloth, Biaxial oder Stitched Mat verwenden.

### F13: Wie repariere ich einen Gelcoat-Kratzer so dass man ihn nicht sieht?
Fast unmöglich bei älteren Booten (Gelcoat vergilbt). Beste Methode: Farblich passenden Gelcoat verwenden (Spectrum/International), Mylar-Folie für glattes Finish, Stufenschliff (400→800→1000), Polieren (Rubbing Compound → Finish-Politur).

### F14: Was brauche ich für eine Reparatur am Kiel?
**West System 105 + 206 + 404 (High Density Filler)** für strukturelle Verklebung. **Biaxial ±45°** für Verstärkung. NIEMALS Polyester am Kiel — sicherheitskritisch!

### F15: Wie transportiere ich Reparaturmaterialien im Flugzeug?
Epoxid-Harz + Härter: **nur als Frachtgut** (IATA-Gefahrgut Klasse 8/9). Im Handgepäck: Marine-Tex erlaubt (paste, <100 g). Glasfaser: unproblematisch. Aceton: verboten (Flüssigkeit, entzündlich).

### F16: Welches Gelcoat-Weiß passt zu meinem Boot?
Kein "Standard-Weiß" passt! Boote vergilben unterschiedlich. Am besten: Probe aus unauffälligem Bereich nehmen → Spectrum (US) oder International (EU) Farbmischservice nutzen.

### F17: Wie repariere ich einen Riss im Deck richtig?
V-Nut schleifen (Fächerwinkel), mit Epoxid füllen, Glasfaser-Tape darüber laminieren, aushärten, schleifen, Gelcoat auftragen. Für Decks mit Kern: Prüfen ob Wasser eingedrungen ist → Kern trocknen/ersetzen BEVOR repariert wird.

### F18: Kann ich UV-härtende Produkte (Solarez) für Bootsreparaturen nutzen?
Für KLEINE Gelcoat-Chips JA — schnell, einfach. Für strukturelle Reparaturen NEIN — zu dünn, keine Glasfaser-Integration, nicht wasserfest genug.

### F19: Was ist der häufigste Fehler bei GFK-Reparaturen?
**Unzureichende Vorbereitung** (nicht geschliffen, nicht gereinigt, feucht). Zweithäufigster: **Falsches Mischverhältnis** bei Epoxid. Dritthäufigster: **Zu wenige Lagen** (Reparatur schwächer als Original).

### F20: Welches Reparaturmaterial gibt es in der Karibik?
**West System** ist in den meisten Karibik-Ländern über Budget Marine verfügbar. Marine-Tex ebenfalls häufig. Polyester/Glasfaser in jedem Baumarkt. TotalBoat/System Three: Import nötig.

### F21: Brauche ich eine Waage zum Epoxid-Mischen?
Bei West System mit Dosierpumpen (300 Mini Pump Set): NEIN — Pumpen dosieren automatisch 5:1. Bei anderen Systemen oder präziser Arbeit: JA — Digitalwaage mit 0,1 g Auflösung.

### F22: Was ist besser: Epoxid-Spachtel oder Glasfaser-Reparatur?
**Für kosmetische Reparaturen** (Dellen, kleine Kratzer): Epoxid-Spachtel (105 + 407 oder TotalFair). **Für strukturelle Reparaturen** (Risse, Löcher, Delamination): Glasfaser-Laminat mit Epoxid — IMMER.

### F23: Wie repariere ich einen Unterwasser-Blister provisorisch?
Blase öffnen, Flüssigkeit ablaufen lassen, mit Frischwasser spülen, so gut wie möglich trocknen, Marine-Tex oder G/flex 655 einstreichen, Antifouling darüber. KEINE permanente Reparatur — Boot muss für vollständige Osmose-Sanierung aus dem Wasser.

### F24: Wie lagere ich West System Epoxid an Bord eines Bootes?
Kühl, trocken, dunkel. Ideal: Bilge (Temperaturstabil). Harz und Härter getrennt lagern. Alle Behälter fest verschließen. Alle 6 Monate Konsistenz prüfen. Bei Kristallisation (Harz wird trüb/fest bei Kälte): Flasche in warmes Wasserbad stellen — löst sich wieder.

### F25: Macht Vakuumbagging bei einer Reparatur Sinn?
Bei kleinen Reparaturen (<0,5 m²): NEIN — zu viel Aufwand. Bei großflächigen strukturellen Reparaturen (Osmose-Sanierung, Deck-Reparatur): JA — bessere Kompaktierung, höherer FVA, weniger Blasen. Cross-Reference: 04.15 Vakuuminfusion.

### A.1 Harzverbrauch für Reparatur

```python
def harzverbrauch_reparatur(
    flaeche_m2: float,
    glasfaser_gsm: float,
    lagen: int,
    harzsystem: str = "epoxid"
) -> dict:
    """Berechnet Harzverbrauch für eine GFK-Reparatur."""
    # Harz-Glas-Verhältnisse
    verhaeltnis = {
        "epoxid_gewebe": 1.0,      # 1:1 Harz:Glas bei Gewebe
        "epoxid_biaxial": 0.8,     # Weniger Harz bei Biax
        "polyester_csm": 2.5,      # Viel Harz bei CSM
        "polyester_gewebe": 1.5,   # Mehr als Epoxid
    }

    if harzsystem == "epoxid":
        ratio = verhaeltnis["epoxid_gewebe"]
    else:
        ratio = verhaeltnis["polyester_gewebe"]

    glasfaser_kg = flaeche_m2 * glasfaser_gsm * lagen / 1000
    harz_kg = glasfaser_kg * ratio
    # Zuschläge
    wet_out = harz_kg * 0.10       # 10% für Oberflächen-Benetzung
    verschnitt = harz_kg * 0.15    # 15% Verschnitt
    harz_gesamt = harz_kg + wet_out + verschnitt

    return {
        "flaeche_m2": flaeche_m2,
        "glasfaser_kg": round(glasfaser_kg, 2),
        "harz_netto_kg": round(harz_kg, 2),
        "harz_gesamt_kg": round(harz_gesamt, 2),
        "harzsystem": harzsystem,
    }

# Beispiel: 0,5 m² Reparatur, 3 Lagen 200 g/m² Gewebe, Epoxid
print(harzverbrauch_reparatur(0.5, 200, 3, "epoxid"))
# → glasfaser_kg: 0.3, harz_netto_kg: 0.3, harz_gesamt_kg: 0.38
```

### A.4 Lagenanzahl für Reparatur berechnen

```python
def lagen_berechnung(
    original_dicke_mm: float,
    glasfaser_gsm: float = 200,
    fva: float = 0.55,
    zuschlag: float = 0.15,
) -> dict:
    """Berechnet nötige Lagenanzahl für äquivalente Reparatur."""
    # Glasdichte: ~2540 kg/m³
    # Laminat-Dicke pro Lage = Glasgewicht / (Glasdichte * FVA)
    dicke_pro_lage_mm = glasfaser_gsm / (2540 * fva) * 1000
    lagen_exakt = original_dicke_mm / dicke_pro_lage_mm
    lagen_mit_zuschlag = lagen_exakt * (1 + zuschlag)
    import math
    lagen_gerundet = math.ceil(lagen_mit_zuschlag)

    return {
        "original_dicke_mm": original_dicke_mm,
        "glasfaser_gsm": glasfaser_gsm,
        "fva": fva,
        "dicke_pro_lage_mm": round(dicke_pro_lage_mm, 3),
        "lagen_exakt": round(lagen_exakt, 1),
        "lagen_mit_zuschlag": round(lagen_mit_zuschlag, 1),
        "lagen_empfohlen": lagen_gerundet,
        "resultierende_dicke_mm": round(lagen_gerundet * dicke_pro_lage_mm, 1),
    }

# Beispiele:
print(lagen_berechnung(5, 200))     # 5 mm, 200 gsm → 4 Lagen
print(lagen_berechnung(8, 450))     # 8 mm, 450 gsm → 3 Lagen Biaxial
print(lagen_berechnung(12, 200))    # 12 mm → 10 Lagen (dickes Laminat)
```

### A.5 Topfzeit-Korrektur nach Temperatur

```python
def topfzeit_temperatur(
    topfzeit_25c_min: float,
    temperatur_c: float,
) -> dict:
    """Korrigiert Topfzeit basierend auf Temperatur (Arrhenius-Näherung)."""
    # Faustregel: Topfzeit halbiert sich pro 10°C über 25°C
    # Faustregel: Topfzeit verdoppelt sich pro 10°C unter 25°C
    delta = temperatur_c - 25
    faktor = 2 ** (-delta / 10)
    topfzeit_korrigiert = topfzeit_25c_min * faktor

    empfehlung = ""
    if temperatur_c < 15:
        empfehlung = "Fast Hardener (205) verwenden + Heizstrahler"
    elif temperatur_c < 20:
        empfehlung = "Fast Hardener (205) empfohlen"
    elif temperatur_c <= 30:
        empfehlung = "Standard Hardener (206) OK"
    elif temperatur_c <= 35:
        empfehlung = "Slow Hardener (207/209) empfohlen"
    else:
        empfehlung = "Extra Slow Hardener (209) + im Schatten arbeiten!"

    return {
        "temperatur_c": temperatur_c,
        "topfzeit_25c_min": topfzeit_25c_min,
        "topfzeit_korrigiert_min": round(topfzeit_korrigiert, 0),
        "faktor": round(faktor, 2),
        "haerter_empfehlung": empfehlung,
    }

# Beispiele:
print(topfzeit_temperatur(25, 15))   # Kalt: ~50 min
print(topfzeit_temperatur(25, 25))   # Normal: 25 min
print(topfzeit_temperatur(25, 35))   # Tropisch: ~12 min!
```

### A.6 Reparatur-Gewichtsberechnung

```python
def reparatur_gewicht(
    flaeche_m2: float,
    glasfaser_gsm: float,
    lagen: int,
    harzsystem: str = "epoxid",
) -> dict:
    """Berechnet Gewicht der Reparatur (für Trimm-Beurteilung)."""
    harz_faktor = 1.0 if harzsystem == "epoxid" else 1.5
    glas_kg = flaeche_m2 * glasfaser_gsm * lagen / 1000
    harz_kg = glas_kg * harz_faktor
    gesamt_kg = glas_kg + harz_kg
    # Gelcoat: ~0.6 kg/m² bei 0.5 mm
    gelcoat_kg = flaeche_m2 * 0.6

    return {
        "flaeche_m2": flaeche_m2,
        "glas_kg": round(glas_kg, 2),
        "harz_kg": round(harz_kg, 2),
        "laminat_kg": round(gesamt_kg, 2),
        "gelcoat_kg": round(gelcoat_kg, 2),
        "gesamt_kg": round(gesamt_kg + gelcoat_kg, 2),
        "trimm_relevant": gesamt_kg > 5.0,  # >5 kg = Trimm prüfen
    }

# Beispiel: 1 m² Reparatur, 4 Lagen 200 gsm
print(reparatur_gewicht(1.0, 200, 4))
# → laminat_kg: 1.6, gelcoat_kg: 0.6, gesamt_kg: 2.2
```

### A.7 Kosten-ROI-Berechnung: DIY vs. Werft

```python
def diy_vs_werft_roi(
    material_kosten_eur: float,
    arbeitszeit_h: float,
    werft_stundensatz_eur: float = 80,
    werkzeug_kosten_eur: float = 50,
) -> dict:
    """Vergleicht DIY- vs. Werftkosten."""
    diy_kosten = material_kosten_eur + werkzeug_kosten_eur
    werft_kosten = material_kosten_eur * 1.3 + arbeitszeit_h * werft_stundensatz_eur
    ersparnis = werft_kosten - diy_kosten
    ersparnis_prozent = (ersparnis / werft_kosten) * 100

    return {
        "diy_kosten_eur": round(diy_kosten, 0),
        "werft_kosten_eur": round(werft_kosten, 0),
        "ersparnis_eur": round(ersparnis, 0),
        "ersparnis_prozent": round(ersparnis_prozent, 0),
        "empfehlung": "DIY lohnt sich" if ersparnis_prozent > 50 else "Werft erwägen",
    }

# Beispiele:
print(diy_vs_werft_roi(50, 4))       # Kleiner Riss: DIY spart ~70%
print(diy_vs_werft_roi(800, 120))    # Osmose: DIY spart ~85%
```

```python
from pydantic import BaseModel
from enum import Enum
from typing import Optional, List

class DamageType(str, Enum):
    GELCOAT_SCRATCH = "gelcoat_scratch"
    GELCOAT_CHIP = "gelcoat_chip"
    STRUCTURAL_CRACK = "structural_crack"
    OSMOSIS = "osmosis"
    DELAMINATION = "delamination"
    CORE_ROT = "core_rot"
    KEEL_DAMAGE = "keel_damage"
    GROUNDING = "grounding"
    IMPACT = "impact"
    TRANSOM_ROT = "transom_rot"

class RepairUrgency(str, Enum):
    EMERGENCY = "emergency"        # Unterwegs, sofort
    SOON = "soon"                  # Nächster Hafen
    PLANNED = "planned"            # Winterlager

class BoatClass(str, Enum):
    PRODUCTION_SAIL = "production_sail"
    SEMI_CUSTOM = "semi_custom"
    CUSTOM_SUPER = "custom_super"
    RIB = "rib"
    MOTOR = "motor"

class RepairRecommendation(BaseModel):
    model_config = {"from_attributes": True}

    damage_type: DamageType
    urgency: RepairUrgency
    boat_class: BoatClass
    resin_system: str
    hardener: str
    filler: Optional[str]
    reinforcement: str
    finish: str
    estimated_material_cost_eur: float
    estimated_work_hours: float
    difficulty: str  # "einfach", "mittel", "fortgeschritten", "professionell"
    diy_possible: bool
    confidence: str
    warnings: List[str]


def recommend_repair(
    damage: DamageType,
    urgency: RepairUrgency,
    boat_class: BoatClass,
    area_m2: float = 0.1,
) -> RepairRecommendation:
    """Empfiehlt Reparaturmaterialien basierend auf Schadenstyp."""

    # Defaults
    resin = "West System 105"
    hardener = "206 Slow"
    filler = None
    reinforcement = "E-Glas Köper 200 g/m², 3 Lagen"
    finish = "Gelcoat + Polieren"
    cost = 50.0
    hours = 4.0
    difficulty = "mittel"
    diy = True
    warnings = []

    if urgency == RepairUrgency.EMERGENCY:
        resin = "West G/flex 655"
        hardener = "(integriert)"
        reinforcement = "Glasfaser-Tape 75 mm"
        finish = "Provisorisch"
        cost = 40.0
        hours = 2.0
        difficulty = "einfach"
        warnings.append("Provisorische Reparatur — permanente Lösung im nächsten Hafen")

    elif damage == DamageType.GELCOAT_SCRATCH:
        resin = "Polyester-Gelcoat"
        hardener = "MEKP 1,5%"
        filler = None
        reinforcement = "—"
        finish = "Mylar-Folie → Schleifen → Polieren"
        cost = 15.0
        hours = 1.0
        difficulty = "einfach"

    elif damage == DamageType.OSMOSIS:
        hardener = "206 Slow"
        filler = "407 Low Density (Fairing) + 422 Barrier Coat"
        reinforcement = "—"
        finish = "5–6 Schichten Barrier Coat + Antifouling"
        cost = 200.0 + area_m2 * 40
        hours = 60.0 + area_m2 * 3
        difficulty = "fortgeschritten"
        warnings.append("Trocknungszeit: 4–12 Wochen!")
        warnings.append("Feuchtemessung vor Reparatur: <0,5%")

    elif damage in [DamageType.KEEL_DAMAGE, DamageType.STRUCTURAL_CRACK]:
        filler = "404 High Density"
        reinforcement = "Biaxial ±45° 450 g/m², 3–5 Lagen"
        cost = 80.0 + area_m2 * 60
        hours = 8.0
        difficulty = "fortgeschritten"
        if damage == DamageType.KEEL_DAMAGE:
            warnings.append("SICHERHEITSKRITISCH — Gutachter-Abnahme empfohlen")
            diy = False

    elif damage == DamageType.CORE_ROT:
        filler = "406 Colloidal Silica"
        reinforcement = "Biaxial ±45° 450 g/m² + Marine-Sperrholz BS 1088"
        cost = 150.0 + area_m2 * 80
        hours = 20.0
        difficulty = "professionell"
        warnings.append("Kern muss KOMPLETT trocken sein vor Reparatur")

    confidence = "estimated"

    return RepairRecommendation(
        damage_type=damage,
        urgency=urgency,
        boat_class=boat_class,
        resin_system=resin,
        hardener=hardener,
        filler=filler,
        reinforcement=reinforcement,
        finish=finish,
        estimated_material_cost_eur=round(cost, 0),
        estimated_work_hours=round(hours, 1),
        difficulty=difficulty,
        diy_possible=diy,
        confidence=confidence,
        warnings=warnings,
    )
```

```python
from fastapi import APIRouter, HTTPException
from pydantic import BaseModel, Field
from typing import Optional, List

router = APIRouter(prefix="/api/v1/repair", tags=["repair"])


class RepairRequest(BaseModel):
    model_config = {"from_attributes": True}

    damage_type: str = Field(..., description="Schadenstyp (gelcoat_scratch, osmosis, etc.)")
    urgency: str = Field(default="planned", description="Dringlichkeit (emergency, soon, planned)")
    boat_class: str = Field(default="production_sail", description="Bootsklasse")
    area_m2: float = Field(default=0.1, description="Schadensfläche in m²")
    temperature_c: Optional[float] = Field(default=20.0, description="Arbeitstemperatur")
    humidity_percent: Optional[float] = Field(default=50.0, description="Relative Feuchte")


class RepairResponse(BaseModel):
    model_config = {"from_attributes": True}

    recommendation: dict
    material_list: List[dict]
    estimated_cost_eur: float
    estimated_hours: float
    difficulty: str
    diy_possible: bool
    warnings: List[str]
    confidence: str
    hardener_recommendation: str
    topfzeit_adjusted_min: float


@router.post("/recommend", response_model=RepairResponse)
async def recommend_repair_endpoint(request: RepairRequest):
    """Empfiehlt Reparaturmaterialien basierend auf Schadensparametern."""
    # Import recommendation engine
    from app.services.knowledge import recommend_repair, topfzeit_temperatur

    try:
        rec = recommend_repair(
            damage=request.damage_type,
            urgency=request.urgency,
            boat_class=request.boat_class,
            area_m2=request.area_m2,
        )

        topfzeit = topfzeit_temperatur(25, request.temperature_c)

        return RepairResponse(
            recommendation=rec.model_dump(),
            material_list=_build_material_list(rec),
            estimated_cost_eur=rec.estimated_material_cost_eur,
            estimated_hours=rec.estimated_work_hours,
            difficulty=rec.difficulty,
            diy_possible=rec.diy_possible,
            warnings=rec.warnings,
            confidence=rec.confidence,
            hardener_recommendation=topfzeit["haerter_empfehlung"],
            topfzeit_adjusted_min=topfzeit["topfzeit_korrigiert_min"],
        )
    except Exception as e:
        raise HTTPException(status_code=400, detail=str(e))


def _build_material_list(rec) -> List[dict]:
    """Erstellt Einkaufsliste aus Empfehlung."""
    materials = [
        {"name": rec.resin_system, "category": "Harz", "essential": True},
        {"name": rec.hardener, "category": "Härter", "essential": True},
    ]
    if rec.filler:
        materials.append({"name": rec.filler, "category": "Füller", "essential": True})
    if rec.reinforcement != "—":
        materials.append({"name": rec.reinforcement, "category": "Glasfaser", "essential": True})
    # Standard-Zubehör
    materials.extend([
        {"name": "Nitrilhandschuhe", "category": "Schutz", "essential": True},
        {"name": "Mischbecher", "category": "Werkzeug", "essential": True},
        {"name": "Schleifpapier-Satz", "category": "Finish", "essential": True},
    ])
    return materials
```
